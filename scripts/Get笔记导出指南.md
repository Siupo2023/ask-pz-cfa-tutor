# Get笔记 导出指南

## 🎯 目标
将 Get笔记 中所有 CFA 相关笔记导出到 ask-pz references 目录

## 📋 方法一：Get笔记 Web 界面导出

### 步骤：
1. **打开 Get笔记 Web 版**
   - 访问：https://www.biji.com
   - 登录你的账号

2. **搜索 CFA 相关笔记**
   - 使用搜索框搜索 "CFA"
   - 或搜索关键词：企业合并、权益法、FRA、Equity、Fixed Income 等

3. **批量导出笔记**
   - 选择所有 CFA 相关笔记
   - 使用导出功能（如果有）
   - 选择 Markdown 格式

4. **保存到指定位置**
   - 保存到：`~/Desktop/GetNotes导出/` 或其他临时目录

## 📋 方法二：Get笔记 客户端导出

### 桌面客户端：
1. 打开 Get笔记 桌面应用
2. 使用搜索功能查找 CFA 相关笔记
3. 逐个或批量导出为 Markdown
4. 保存到临时目录

## 📋 方法三：手动复制粘贴

如果无法直接导出：
1. 打开每个 CFA 相关笔记
2. 复制全部内容
3. 创建对应的 Markdown 文件
4. 我会帮你整理到 ask-pz references
