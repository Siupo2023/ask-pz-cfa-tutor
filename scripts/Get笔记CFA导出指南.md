# Get笔记 CFA 学习笔记导出指南

## 🎯 目标
将 Get笔记 中所有 CFA 相关笔记导出到 ask-pz references 目录

## 📋 推荐方法：Get笔记 客户端导出

### 方法一：Get笔记 Web 版

1. **打开 Get笔记 Web 版**
   - 访问：https://www.biji.com
   - 登录你的账号

2. **搜索 CFA 相关笔记**
   - 在搜索框中输入：`CFA`
   - 或搜索其他关键词：企业合并、权益法、Acquisition、FRA、Equity 等

3. **导出笔记**
   - 点击每个笔记，打开详情页
   - 复制笔记内容（Ctrl/Cmd + A 全选，Ctrl/Cmd + C 复制）
   - 创建对应的文本文件保存

### 方法二：Get笔记 移动端/客户端

1. **打开 Get笔记 APP**
2. **搜索 "CFA"**
3. **批量选择**相关笔记（如果支持）
4. **分享或导出**为文本格式
