#### 知识精讲

本节课核心讲解AR模型的核心假设与平稳性概念，重点阐述了协方差平稳的三个条件、均值回归特性及其在金融市场的应用。

**AR模型的引入与核心假设**
* **AR模型的提出背景**：趋势模型在实际应用中常出现残差序列相关问题，因时间序列数据具有惯性（如“昨天影响今天”），故引入AR模型捕捉此规律。
* **AR模型的选择起点**：从AR(1)开始，因其假设“最近的过去对现在影响最大”，若AR(1)能完全捕捉序列特点，则无需更高阶模型。
   - 🚩重点：AR模型成立需满足三个关键假设，是本章核心考点。
* **AR模型的三个假设**：
   - **假设1（平稳性）**：时间序列数据本身需满足协方差平稳（弱平稳）。
   - **假设2（无自相关）**：模型残差项ε需无自相关（no auto correlation）。
   - **假设3（同方差性）**：模型残差项ε需满足同方差（no heteroscedasticity）。
   - *补充说明*：假设1针对原始序列，假设2和3针对模型残差项。

**协方差平稳（Covariance Stationary）**
* **定义**：时间序列数据在性质上未发生重大改变，具体需满足三个条件。
   - 🚩重点：协方差平稳的三个条件是本章最重要考点。
   - *核心条件*：
     1. 均值存在且稳定不变（E[Xt] = μ，μ为常数）
     2. 方差存在且稳定不变（Var(Xt) = σ²，σ²为常数）
     3. 协方差存在且仅与时间间隔有关（Cov(Xt, Xt+k) = γk，γk仅与k有关）
* **非平稳序列案例**：
   - 中国1990-2020年进出口额数据（2001年加入WTO导致市场结构突变）
   - 万科股票1995-2020年收益率数据（2006年股权分置改革导致市场流动性突变）
   - *补充说明*：非平稳序列前后分段的均值、方差、协方差存在显著差异。

**均值回归（Mean Reversion）**
* **定义**：平稳序列围绕固定均值波动，当偏离均值时会向均值方向调整的特性。
   - *公式推导*：对于AR(1)模型 Xt = β0 + β1Xt-1 + εt，若序列平稳，则均值 μ = β0 / (1 - β1)。
   - *核心特征*：
     - 若Xt > μ，则Xt+1会向μ靠近（Xt+1 < Xt），但不一定小于μ
     - 若Xt < μ，则Xt+1会向μ靠近（Xt+1 > Xt），但不一定大于μ
   - 🚩重点：均值回归≠“超调”（并非从均值一侧跳到另一侧），而是“向均值靠近”。
* **生活与金融应用**：
   - 考试成绩波动（围绕平均名次波动）
   - 股票价格受冲击后回归长期均值（若市场满足平稳性）
   - *补充说明*：趋势性市场（如股价持续上涨/下跌）不满足均值回归，属于非平稳序列。

**平稳性与均值回归的关系**
* **核心结论**：若序列协方差平稳，则其均值 μ = β0/(1-β1)，且具有均值回归特性；非平稳序列无固定均值，不表现均值回归，常呈现趋势性。

开始正式讲解AR模型核心。时间序列建模的目标是预测未来，需捕捉数据全部特点。趋势模型因残差序列相关问题（时间序列惯性导致“昨天影响今天”）不成立，故引入AR模型。

[00:01:23](https://getnotes.seek:83) **AR模型的阶数选择**
应从AR(1)开始，因其假设“最近的过去影响最大”。若AR(1)能完全捕捉序列特点，则无需高阶模型；否则需修正或使用其他模型。

[00:02:35](https://getnotes.seek:155) **AR模型的三大假设**
判断AR模型是否成立需满足三个假设，是本章核心考点。若违反假设需修正模型以更好捕捉序列特点。

[00:03:44](https://getnotes.seek:224) **假设1：协方差平稳性**
要求原始时间序列数据满足协方差平稳（covariance stationary，弱平稳）。平稳性指数据性质未发生重大改变，与多元回归中的平稳概念一致。

[00:05:10](https://getnotes.seek:310) **假设2：残差无自相关**
要求AR模型残差项ε无自相关（no auto correlation），与普通回归模型对残差的要求一致，确保残差无规律性（“真正的垃圾桶”）。

[00:06:10](https://getnotes.seek:370) **假设3：残差同方差**
要求AR模型残差项ε满足同方差（no heteroscedasticity），即残差波动稳定，不随自变量改变，确保残差无规律性。

[00:07:29](https://getnotes.seek:449) **协方差平稳的重要性**
三大假设中最重要的是协方差平稳性，若本章只考一道题，必为此考点。

[00:07:47](https://getnotes.seek:467) **平稳性的直观理解**
平稳序列指中间未发生性质重大改变的数据。举例：1990-2020年中国进出口额（2001年加入WTO导致市场结构突变，非平稳）；1995-2020年万科股票收益率（2006年股权分置改革导致流动性突变，非平稳）。

[00:11:46](https://getnotes.seek:706) **协方差平稳的三大条件**
弱平稳需满足：①均值存在且不变；②方差存在且稳定；③协方差存在且仅与时间间隔有关。

[00:12:54](https://getnotes.seek:774) **非平稳序列的统计特征**
以进出口额数据为例，加入WTO前后分段均值、方差、协方差均不相等，故非平稳。

[00:16:59](https://getnotes.seek:1019) **AR(1)模型的均值推导**
AR(1)模型：Xt = β0 + β1Xt-1 + εt。若序列平稳，则E[Xt] = E[Xt-1] = μ，推导得 μ = β0/(1 - β1)。

[00:18:51](https://getnotes.seek:1131) **均值回归的定义**
平稳序列围绕均值μ波动，当Xt > μ时，Xt+1会向μ靠近（Xt+1 < Xt）；当Xt < μ时，Xt+1会向μ靠近（Xt+1 > Xt）。

[00:19:50](https://getnotes.seek:1190) **均值回归的常见误解**
均值回归≠“必然超越均值”，而是“向均值靠近”。例如Xt > μ时，Xt+1只需小于Xt即可，不一定小于μ。

[00:21:13](https://getnotes.seek:1273) **均值回归的生活案例**
考试成绩围绕平均名次波动（考第2名后倾向退步，考第15名后倾向进步）；恋爱中追求期与稳定期行为差异（追求期表现高于均值，后回归正常水平）；“乐极生悲”“人无千日好”等哲学思想。

[00:23:35](https://getnotes.seek:1415) **均值回归的金融应用**
若股票价格满足均值回归，受利好冲击冲高后会回归原均值；若冲击后形成新均值并波动，则为趋势性（非平稳）。金融市场多数数据呈趋势性，属非平稳序列。

[00:26:00](https://getnotes.seek:1560) **平稳性与均值回归的关系总结**
协方差平稳序列具有固定均值μ=β0/(1-β1)和均值回归特性；非平稳序列无固定均值，不表现均值回归，常呈趋势性。

#### 💡 核心概念
- **协方差平稳（弱平稳）**：时间序列的均值、方差恒定，协方差仅与时间间隔有关。
- **均值回归**：平稳序列围绕固定均值波动，偏离均值时向均值方向调整的特性。
- **AR(1)模型均值公式**：μ = β0 / (1 - β1) （仅当序列平稳时成立）
