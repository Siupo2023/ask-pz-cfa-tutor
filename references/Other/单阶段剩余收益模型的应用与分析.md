#### 录音总结
本次课程讲解了无限期企业的单阶段剩余收益模型，包括模型构成、计算示例、隐含增长率推导及与其他估值模型的比较。

**剩余收益模型基本构成**
* **模型核心公式**：企业价值被定义为两个部分，即当前账面价值（B0）加上所有未来剩余收益的现值。对于无限期且剩余收益按固定增长率增长的企业，未来剩余收益的现值可通过戈登增长模型（永续增长模型）计算，公式为第一年剩余收益（RI1）除以（必要收益率R - 增长率G）。
* **第一年剩余收益计算**：RI1等于第一年的预期收益（E1）减去权益费用（R×B0）。其中，E1可由期初账面价值（B0）乘以净资产收益率（ROE）得出，即E1 = ROE × B0。

**单阶段剩余收益模型应用示例**
* **已知条件与增长率计算**：给定B0=23，ROE=14%，必要收益率R=12%，股息支付率60%。由于未直接给出增长率G，需通过可持续增长率公式计算，即G = 留存率 × ROE。留存率为1 - 股息支付率 = 40%，因此G = 40% × 14% = 5.6%。
* **剩余收益与企业价值计算**：首先计算E1 = 14% × 23 = 3.22。权益费用 = 12% × 23 = 2.76。RI1 = E1 - 权益费用 = 3.22 - 2.76 = 0.46。未来剩余收益现值 = RI1 / (R - G) = 0.46 / (12% - 5.6%) = 0.46 / 6.4% = 7.19。企业价值 = B0 + 未来剩余收益现值 = 23 + 7.19 = 30.19。

**隐含增长率的推导**
* **已知市场价格反推增长率**：若已知市场价格，假设其与单阶段剩余收益模型估值一致，可通过公式反推隐含增长率G。方法是将市场价格代入企业价值公式，得到关于G的方程，通过试错法（如代入选项中的中间值）求解。例如，若市场价格为35，高于当前计算的30.19，则需提高G，经计算隐含增长率为8.17%。

**剩余收益模型与其他模型的比较**
* **与市净率的关系**：剩余收益模型与市净率（P/B）关系密切，公式为P/B = 1 + (ROE - G)/(R - G)。当剩余收益为正时（即ROE > R），P/B应大于1。
* **与其他DCF模型的共性与差异**：剩余收益模型、股息贴现模型（DDM）和自由现金流模型均属于贴现未来收益的估值方法，核心差异在于对“收益”的定义不同：DDM使用股息，自由现金流模型使用自由现金流，剩余收益模型使用剩余收益。

### 📅 章节概要
[00:00:01](https://getnotes.seek:1) **无限期企业剩余收益模型引入**

从有限期清算股利的剩余收益模型过渡到无限期企业的一般情况，将企业价值定义为当前账面价值与未来所有剩余收益现值之和。

[00:00:33](https://getnotes.seek:33) **第一年剩余收益的分子构成**

明确剩余收益现值公式中分子为第一年的剩余收益（RI1），因为ROE乘以期初账面价值（B0）得到第一年收益（E1），即E1 = ROE × B0。

[00:00:46](https://getnotes.seek:46) **第一年剩余收益的计算逻辑**

使用期初账面价值计算的是该年度末的收益（E1），因此第一年剩余收益RI1 = E1 - 权益费用，其中权益费用为必要收益率（R）乘以期初账面价值（B0）。

[00:01:05](https://getnotes.seek:65) **固定增长率下的剩余收益现值计算**

若剩余收益按固定增长率G增长，其现值可通过戈登增长模型计算，即RI1 / (R - G)，本质是增长型永续年金的现值。

[00:01:28](https://getnotes.seek:88) **单阶段模型应用示例：已知条件**

给出示例条件：账面价值B0=23，ROE=14%，必要收益率R=12%，股息支付率60%，需计算企业价值。

[00:01:49](https://getnotes.seek:109) **股息支付率的作用**

股息支付率本身不直接用于剩余收益计算，但因未直接给出增长率G，需通过股息支付率计算留存率以推导G。

[00:02:00](https://getnotes.seek:120) **增长率的计算**

留存率 = 1 - 股息支付率 = 40%，根据可持续增长率公式G = 留存率 × ROE，计算得G = 40% × 14% = 5.6%。

[00:02:33](https://getnotes.seek:153) **ROE与R的正差额对估值的影响**

由于ROE（14%）大于必要收益率R（12%），存在正差额，因此企业价值将高于当前账面价值，即未来剩余收益现值为正。

[00:02:55](https://getnotes.seek:175) **剩余收益与企业价值的初步计算**

剩余收益的正差额部分为（ROE - R）× B0 = (14% - 12%) × 23 = 0.46（即RI1）。未来剩余收益现值 = 0.46 / (12% - 5.6%) = 7.19。

[00:03:12](https://getnotes.seek:192) **企业价值的最终计算**

企业价值 = 当前账面价值 + 未来剩余收益现值 = 23 + 7.19 = 30.19。

[00:03:39](https://getnotes.seek:219) **隐含增长率的推导方法**

若已知市场价格，可假设其符合单阶段剩余收益模型，通过代入不同G值（如选项中的5%、6%、8%）试错，根据估值与价格的比较调整G，最终得到隐含增长率（示例中为8.17%）。

[00:04:00](https://getnotes.seek:240) **试错法计算隐含增长率的步骤**

以市场价格35为例，先代入中间值6%计算估值，若估值低于35，则需提高G，最终确定隐含增长率为8.17%。

[00:04:41](https://getnotes.seek:281) **剩余收益模型与市净率的关系**

剩余收益模型与市净率（P/B）密切相关，公式为P/B = 1 + (ROE - G)/(R - G)，当剩余收益为正时，P/B > 1。

[00:05:05](https://getnotes.seek:305) **剩余收益模型与其他DCF模型的比较**

剩余收益模型、DDM和自由现金流模型均通过贴现未来收益估值，区别在于“收益”定义不同：DDM用股息，自由现金流模型用自由现金流，剩余收益模型用剩余收益。

[00:05:23](https://getnotes.seek:323) **不同DCF模型的共性**

所有DCF模型本质上都是对未来收益的贴现，只是具体的收益指标（股息、自由现金流、剩余收益）有所不同。
