#### 录音总结
本次课程讲解围绕股息贴现模型展开，重点介绍了恒定增长模型（戈登增长模型）的公式、假设、应用场景、优缺点，以及多阶段股息贴现模型的基本概念和终端价值的估算方法。

**恒定增长模型（戈登增长模型）基础**
* **模型公式与核心假设**：恒定增长模型的价值计算公式为\( P_0 = D_1 / (R - G) \)，其中\( D_1 \)是预期第一年股息，\( R \)是要求回报率，\( G \)是恒定增长率。核心假设包括增长率\( G \)恒定且永续，且\( R > G \)以保证分母为正，否则需重新检查\( R \)或\( G \)的估算。
* **\( D_1 \)的含义与计算**：\( D_1 \)是投资者购买股票后获得的第一笔现金流，即预期第一年的股息，由已支付的股息\( D_0 \)乘以\( (1 + G) \)得出，投资者无法获得购买前已支付的股息。

**恒定增长模型的应用场景与适用性**
* **适用对象**：该模型适用于增长率恒定的情况，对于成熟公司，其增长已过高速期，未来可能以稳定速度增长；对于广泛的股票指数（如标普500），由于包含多家公司，整体平均增长率可能接近恒定。
* **在复杂模型中的作用**：在两阶段、三阶段或H模型等复杂模型中，恒定增长模型可用于估算终端价值，因为任何公司无论当前增长多快，最终都会进入稳定增长阶段，其增长率接近整体经济增长率。

**恒定增长模型的实例与推导**
* **股票估值实例**：已知某股票昨日支付股息\( D_0 = 1.80 \)澳元，增长率\( G = 3.5\% \)，通过资本资产定价模型（\( R = 无风险利率 + \beta \times (市场回报率 - 无风险利率) \)，其中无风险利率4%，\( \beta = 1.5 \)，市场回报率8%）计算得\( R = 10\% \)，则\( D_1 = 1.80 \times (1 + 3.5\%) \)，股票价值\( P_0 = D_1 / (10\% - 3.5\%) = 28.66 \)澳元。
* **隐含增长率的推导**：若已知股票市场价格\( P_0 \)、\( D_1 \)和\( R \)，可通过公式\( G = R - (D_1 / P_0) \)推导隐含增长率，其中\( D_1 / P_0 \)为股息收益率，\( G \)为资本利得收益率，二者之和为总回报率\( R \)。

**增长机会现值（PVGO）与市盈率（PE）**
* **PVGO的概念与计算**：PVGO是市场赋予公司增长机会的价值，等于股票市场价格减去无增长公司的价值（\( E_1 / R \)，\( E_1 \)为预期第一年 earnings）。例如，股票价格60瑞士法郎，\( E_1 = 5 \)法郎，\( R = 10\% \)，则无增长价值为\( 5 / 10\% = 50 \)法郎，PVGO = 60 - 50 = 10法郎。
* **市盈率与PVGO的关系**：领先市盈率（\( P_0 / E_1 \)）可表示为\( (1/R) + (PVGO / E_1) \)。无增长公司的领先市盈率为\( 1/R \)，若实际市盈率高于此值，则高出部分源于PVGO。例如，\( R = 10\% \)时无增长市盈率为10，若实际市盈率为12，则PVGO贡献的额外市盈率为2。

**恒定增长模型的优劣势**
* **优势**：模型简单易用，可用于估算广泛市场指数的价值，能分解模型以估算增长率、要求回报率、PVGO等未知变量，也可用于复杂模型中终端价值的估算。
* **劣势**：估值结果对\( R \)和\( G \)的估算高度敏感，即使小幅误差也可能导致估值显著偏离，进而引发错误交易决策；无法用于非分红股票，因其缺乏具体的股息数据支持，估算误差会很大。

**多阶段股息贴现模型**
* **模型背景与类型**：单阶段模型假设增长率永久恒定，对多数公司不现实，因此需多阶段模型。常见类型包括两阶段模型（高速增长后突然转为恒定增长）、H模型（增长率线性衰减至恒定）、三阶段模型（增长率逐步阶梯式下降至恒定）。
* **各增长阶段特征**：高速增长阶段，公司盈利快速增长，自由现金流为负（因大量资本支出），ROE > R；过渡阶段，增长放缓，股息增加，ROE接近R，自由现金流转正；成熟阶段，增长率与经济增长一致，自由现金流为正，ROE接近R，可应用戈登增长模型。

**终端价值的估算方法**
* **戈登增长模型法**：在未来某一时点\( T \)的终端价值\( TV_T \)，可通过\( TV_{T-1} = D_T / (R - G) \)计算，需注意时间下标，\( D_T \)是第\( T \)期的股息，对应\( T-1 \)期的终端价值。
* **市盈率倍数法**：若已知未来某时点的行业平均市盈率，可通过该市盈率乘以预期每股收益（EPS）估算终端价值，即\( TV = 预期EPS \times 行业平均PE \)。

[00:01:21](https://getnotes.seek:81) **强调\( R - G \)为负的不可能性**

[00:01:40](https://getnotes.seek:100) **恒定增长模型的假设条件**

假设条件包括：\( D_1 \)是投资的首个预期现金流（第一年股息），增长率恒定且永续，增长率必须小于要求回报率。

[00:01:59](https://getnotes.seek:119) **恒定增长模型的适用情境**

当恒定增长的假设与现实差距不大时模型有用，关键在于判断增长率是否可能恒定。

成熟公司的高增长期已过，未来可能以稳定速度增长；广泛的股票指数（如标普500）包含多家公司，整体平均增长率可能接近恒定，因此模型适用。

以某股票为例，昨日支付股息1.80澳元（\( D_0 \)），增长率3.5%，需计算\( D_1 \)，并通过给定的\( \beta \)、无风险利率和市场回报率计算要求回报率。

[00:04:05](https://getnotes.seek:245) **计算要求回报率**

已知\( \beta = 1.5 \)，无风险利率4%，市场回报率8%，通过公式\( R = 4\% + 1.5 \times (8\% - 4\%) \)计算得\( R = 10\% \)，\( R - G = 10\% - 3.5\% = 6.5\% > 0 \)，符合模型条件。

[00:04:24](https://getnotes.seek:264) **计算股票价值**

\( D_1 = D_0 \times (1 + G) = 1.80 \times (1 + 3.5\%) \)，代入公式\( P_0 = D_1 / (R - G) \)，计算得股票价值为28.66澳元。

[00:04:51](https://getnotes.seek:291) **隐含增长率的概念**

若假设恒定增长模型有效，已知股票市场价格、要求回报率和预期第一年股息，可通过模型反推隐含增长率。

[00:05:05](https://getnotes.seek:305) **推导隐含增长率公式**

由\( P_0 = D_1 / (R - G) \)变形得\( R - G = D_1 / P_0 \)，进而\( G = R - (D_1 / P_0) \)。

[00:05:43](https://getnotes.seek:343) **总回报率的构成**

要求回报率\( R \)是股东的总回报率，包括股息收益率（\( D_1 / P_0 \)）和资本利得收益率（\( G \)），二者之和等于总回报率。

[00:06:00](https://getnotes.seek:360) **引入增长机会现值（PVGO）**

开始讲解PVGO，即市场赋予公司增长机会的价值。

[00:06:34](https://getnotes.seek:394) **PVGO的定义与构成**

PVGO是股票市场价格中源于公司增长机会的部分，股票价格由两部分构成：无增长公司价值（\( E_1 / R \)，\( E_1 \)为预期第一年 earnings）和PVGO。

[00:06:49](https://getnotes.seek:409) **PVGO的计算公式**

PVGO = 股票市场价格 - 无增长公司价值（\( E_1 / R \)）。

[00:07:24](https://getnotes.seek:444) **市盈率与PVGO的关系**

讨论领先市盈率（基于预期 earnings）与PVGO的关系，领先市盈率可表示为无增长市盈率（\( 1/R \)）与PVGO贡献部分（\( PVGO / E_1 \)）之和。

[00:07:47](https://getnotes.seek:467) **无增长公司价值的计算**

无增长公司价值为\( E_1 / R \)，即永续年金的现值。

[00:08:00](https://getnotes.seek:480) **PVGO计算实例**

已知股票市场价格60瑞士法郎，\( E_1 = 5 \)法郎，\( R = 10\% \)，无增长价值为\( 5 / 10\% = 50 \)法郎，PVGO = 60 - 50 = 10法郎，该公司84%价值来自现有盈利，16%来自增长机会，属成熟低增长公司。

[00:08:37](https://getnotes.seek:517) **领先市盈率与无增长市盈率**

该股票领先市盈率为\( 60 / 5 = 12 \)，无增长市盈率为\( 1 / 10\% = 10 \)，PVGO贡献的额外市盈率为12 - 10 = 2。

[00:09:00](https://getnotes.seek:540) **理论领先市盈率公式**

由\( P_0 = E_1 / R + PVGO \)两边除以\( E_1 \)，得领先市盈率 = \( (1/R) + (PVGO / E_1) \)，无增长时市盈率为\( 1/R \)。

[00:09:32](https://getnotes.seek:572) **引入合理市盈率倍数**

讲解使用戈登增长模型计算理论上合理的市盈率倍数（基于基本面，非市场价格），可与市场倍数比较，后续在权益投资中会详细学习。

[00:10:00](https://getnotes.seek:600) **推导合理领先市盈率公式**

由\( P_0 = D_1 / (R - G) \)两边除以\( E_1 \)，得领先市盈率 = \( (D_1 / E_1) / (R - G) \)，其中\( D_1 / E_1 \)为股息支付率（1 - 留存率\( B \)）。

[00:10:18](https://getnotes.seek:618) **恒定增长模型的适用性总结**

适用条件包括：有历史数据指导股息情况，董事会有稳定的股息政策，留存率与增长率存在关联；适用于成熟公司和广泛市场指数。

[00:10:40](https://getnotes.seek:640) **恒定增长模型的优势**

优势包括：可用于估算广泛市场指数价值，能分解模型估算增长率、要求回报率、PVGO等未知变量，可用于复杂模型的终端价值估算。

[00:11:02](https://getnotes.seek:662) **恒定增长模型的劣势**

劣势包括：估值对\( R \)和\( G \)的估算高度敏感，小幅误差可能导致显著估值偏差和错误交易决策；无法用于非分红股票，因缺乏具体股息数据，估算误差大。

[00:11:45](https://getnotes.seek:705) **引入多阶段股息贴现模型**

单阶段模型假设增长率永久恒定不现实，多数公司需多阶段模型，包括两阶段、H模型、三阶段等。

[00:12:00](https://getnotes.seek:720) **两阶段模型的假设**

两阶段模型假设股息在初始阶段以某一增长率增长，之后突然转为恒定增长率。

[00:12:17](https://getnotes.seek:737) **H模型的假设**

H模型假设增长率从当前高水平线性衰减至最终恒定增长率，非突然下降。

[00:12:33](https://getnotes.seek:753) **三阶段模型的假设**

三阶段模型假设增长率分阶段阶梯式下降，最终达到恒定增长率。

[00:13:06](https://getnotes.seek:786) **多阶段模型的本质**

多阶段模型本质仍是计算现金流现值，需估算终端价值以结束无限现金流序列，主要依赖计算器进行计算。

[00:13:22](https://getnotes.seek:802) **高速增长阶段特征**

高速增长阶段：盈利快速增长，自由现金流为负（因大量资本支出），ROE > 要求回报率，公司应留存收益用于再投资。

[00:13:42](https://getnotes.seek:822) **过渡阶段特征**

过渡阶段：增长放缓，股息增加，ROE接近要求回报率，自由现金流转正（因资本支出减少）。

[00:14:00](https://getnotes.seek:840) **成熟阶段特征**

成熟阶段：增长率与经济增长一致，自由现金流为正，ROE接近要求回报率，可应用戈登增长模型， residual income 接近零。

[00:14:37](https://getnotes.seek:877) **终端价值的定义**

终端价值是未来某一时点的预测价值，可视为该时点的股票出售价格，是股息贴现模型中未来现金流的一部分。

[00:14:58](https://getnotes.seek:898) **戈登增长模型估算终端价值**

使用戈登增长模型估算终端价值时，需注意时间下标，若计算第\( T \)期的终端价值\( TV_T \)，需用第\( T+1 \)期的股息\( D_{T+1} \)，公式为\( TV_T = D_{T+1} / (R - G) \)。

[00:15:31](https://getnotes.seek:931) **市盈率倍数法估算终端价值**

若已知未来某时点的行业平均市盈率，可通过该市盈率乘以预期每股收益（EPS）估算终端价值，即\( TV = 预期EPS \times 行业平均PE \)。

[00:15:54](https://getnotes.seek:954) **多阶段模型的计算工具**

多阶段模型的计算主要依赖计算器完成，是对计算器使用能力的练习。

[00:16:19](https://getnotes.seek:979) **不同增长阶段的ROE与分红策略**

当ROE > 要求回报率时，公司应留存收益再投资；当ROE < 要求回报率时，应将收益以股息形式返还股东。

[00:16:42](https://getnotes.seek:1002) **终端价值估算的时间下标注意事项**

计算终端价值时需注意时间一致性，例如用第\( n \)期股息计算的是第\( n-1 \)期的终端价值，避免因时间下标错误失分。

[00:16:57](https://getnotes.seek:1017) **行业平均PE在终端价值估算中的应用**

若已知未来某年后的行业平均PE，可将其应用于该年的预期EPS，计算得到终端价值。

[00:17:12](https://getnotes.seek:1032) **多阶段模型的数学本质**

所有多阶段模型均为计算现值的数学方法，通过设定不同阶段的增长率，最终以终端价值结束无限现金流。

[00:17:33](https://getnotes.seek:1053) **高速增长阶段的自由现金流**

高速增长阶段因大量资本支出用于扩张，自由现金流通常为负。

[00:18:25](https://getnotes.seek:1105) **成熟阶段与戈登增长模型的对应**

成熟阶段的公司特征符合戈登增长模型的假设，因此可直接应用该模型估算其价值。

[00:18:50](https://getnotes.seek:1130) **过渡阶段的ROE变化**

过渡阶段ROE逐渐接近要求回报率，与高速增长阶段的高ROE形成对比。

[00:19:11](https://getnotes.seek:1151) **终端价值作为未来出售价格**

终端价值可理解为投资者在未来某时点出售股票的价格，需将其折现至当前时点计入总价值。

[00:19:27](https://getnotes.seek:1167) **不同终端价值估算方法的应用**

戈登增长模型和市盈率倍数法是估算终端价值的两种主要方法，可根据已知条件选择使用。

[00:19:45](https://getnotes.seek:1185) **多阶段模型的阶段划分依据**

多阶段模型的阶段划分基于公司增长的不同生命周期，通常分为高速增长、过渡和成熟阶段。

[00:20:09](https://getnotes.seek:1209) **股息贴现模型的核心是现值计算**

无论是单阶段还是多阶段股息贴现模型，核心均为将未来股息和终端价值折现至当前时点求和。

[00:20:37](https://getnotes.seek:1237) **终端价值计算的股息时间匹配**

使用戈登增长模型时，终端价值\( TV_t \)对应的股息是\( D_{t+1} \)，需确保股息与终端价值的时间点匹配。

[00:21:00](https://getnotes.seek:1260) **市盈率倍数法的计算逻辑**

市盈率倍数法估算终端价值的逻辑是：价格 = 市盈率 × 每股收益，适用于已知未来市盈率的情况。
