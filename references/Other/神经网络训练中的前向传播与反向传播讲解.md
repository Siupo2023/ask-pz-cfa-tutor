#### 知识精讲
本次课程讲解了神经网络训练的完整流程，拆分介绍了前向传播与反向传播算法的核心作用、计算步骤，明确了神经网络训练本质是迭代优化权重参数以降低预测误差的过程。

**神经网络训练的整体框架**
* **训练核心目标**：训练神经网络的最终目的是估计出所有连接权重$w$的最优值，使得模型预测输出与真实值之间的误差最小。
* **训练核心流程**：整个训练过程分为前向传播（Forward）和反向传播（Backward）两个核心步骤，通过不断迭代两个步骤完成权重优化。
   - 🚩重点：整个神经网络训练本质是**重复前向传播+反向传播的迭代过程**，是期末考试/面试的核心考点。

**前向传播算法**
* **核心定义**：给定输入样本和初始权重，从输入层逐层计算得到最终预测输出的过程。
* **计算步骤**：
1. 初始化：指定神经网络结构、激活函数，随机初始化所有权重参数
2. 输入层：将输入特征$x_1, x_2,...x_n$传入网络
3. 隐藏层计算：每个隐藏层神经元完成两步计算：
   - 第一步：加权加总得到净输入（total net input）：
   $$z_j = \sum_{i=1}^n w_{ij} x_i$$
   - 第二步：将净输入传入激活函数，得到当前神经元的输出：
   $$a_j = f(z_j)$$
4. 输出层计算：整合所有隐藏层神经元的输出，得到最终模型预测输出$y$：
   $$y = \sum_{j=1}^k w_{j} a_j$$
* **激活函数的作用**：
   - 🚩重点：激活函数本质是**调节输入强度的“灯光调节器”**，为网络引入非线性，常用激活函数有两种：
     1. ReLU（修正线性单元）：$f(z) = \max(0, z)$，当前应用更广泛
     2. Sigmoid（S型函数）：输入$z<0$时输出接近0（降低强度/不激活），输入$z>0$时输出接近1（提高强度/激活）
* **核心前提**：前向传播需要先初始化权重，通常初始权重是随机赋值（例如全部设为1），不需要提前知道最优值。

**反向传播算法**
* **核心定义**：根据前向传播得到的预测误差，反向逐层更新调整权重参数，降低预测误差的调参过程。
* **计算步骤**：
1. 计算误差：用前向传播得到的预测输出$\hat{y}$与真实标签$y$计算误差，常用均方误差（MSE）：
   $$MSE = \frac{1}{N}\sum_{i=1}^N (\hat{y}_i - y_i)^2$$
2. 反向更新权重：根据误差的大小，从输出层反向传播到输入层，逐层更新所有权重，目标是降低总误差
3. 迭代：更新完权重后，再次执行前向传播得到新的预测输出，重新计算误差，若误差仍不满足要求则重复反向传播调参

**神经元的工作原理**
* **双函数结构**：每个隐藏层神经元都包含两个功能部分：
1. 加总函数：计算所有输入的加权和，得到净输入$z$，所有神经网络的加总函数形式完全一致，仅权重不同。
2. 激活函数：对净输入做非线性变换，得到当前神经元的输出，传递给下一层。

本节课讲解正统深度学习教材中神经网络训练的具体过程，教材内容较晦涩，将拆分步骤拆解讲解。神经网络训练分为前向传播算法和反向传播算法两个核心步骤。

[00:01:16](https://getnotes.seek:76) **前向传播算法的核心定义**
前向传播算法指：给定一组输入，逐层计算得到最终输出的过程：输入传入神经元得到净输入$z_1$，经过激活函数得到激活后输出$f(z_1)$，再传递到下一层得到$z_2$，重复该过程最终整合得到输出$y$，整个计算过程就是前向传播。

[00:02:27](https://getnotes.seek:147) **前向传播计算过程（以4输入3隐藏1输出结构为例）**
输入为$x_1$到$x_4$，隐藏层共3个神经元：
第一个神经元净输入为$z_1 = w_1 x_1 + w_2 x_2 + w_3 x_3 + w_4 x_4$，$z_1$被称为总净输入（total net input），再将$z_1$传入激活函数，以ReLU为例，$f(z_1) = \max(0, z_1)$。
第二个神经元同理，净输入为$z_2 = w_5 x_1 + w_6 x_2 + w_7 x_3 + w_8 x_4$，权重不同，同样经过激活函数得到$f(z_2)$，第三个神经元得到$f(z_3)$。
最终输出层整合三个隐藏层输出，得到最终预测：$y = w_{13} f(z_1) + w_{14} f(z_2) + w_{15} f(z_3)$，只要输入和权重已知，就能计算出输出$y$。

[00:04:27](https://getnotes.seek:267) **前向传播的前提条件**
要完成前向传播，首先需要：1. 指定神经网络结构（本例为4-3-1结构，即4输入、3个隐藏神经元、1个输出）；2. 指定激活函数，目前常用ReLU和Sigmoid两种激活函数；3. 已知所有$w_1$到$w_{15}$的取值。
训练神经网络的本质就是估计得到误差最小的权重参数，初始状态下我们不知道最优权重，因此需要随机初始化权重，通常可以随机赋值，比如将所有权重初始化为1。

[00:06:08](https://getnotes.seek:368) **前向传播的完整流程总结**
随机初始化权重后，从输入层经过隐藏层计算得到输出，整个走完一遍网络的过程就是前向传播。

[00:07:32](https://getnotes.seek:452) **隐藏层神经元的工作逻辑**
每个隐藏层神经元都有两个功能部分：第一个是加总函数，第二个是激活函数。加总函数就是计算输入的加权和得到总净输入，本质是线性计算，所有神经网络的加总函数形式完全一致，仅权重取值不同。
得到总净输入后，再将其传入激活函数得到该神经元的输出，这个输出只是当前神经元的输出，不是整个网络的最终输出。

[00:08:55](https://getnotes.seek:535) **激活函数的“灯光调节器”比喻**
激活函数类似灯光调节器，可以降低或提高输入信号的强度，是**非线性函数**，常用的两种为Sigmoid（S型函数）和ReLU（修正线性单元）。
以Sigmoid为例，如果总净输入$z<0$，经过Sigmoid后输出接近0，相当于把信号调暗，该神经元处于不激活状态；如果$z>0$，输出接近1，相当于把信号调亮，该神经元处于激活状态。

[00:10:12](https://getnotes.seek:612) **前向传播的问题**
前向传播的初始权重是随机指定的，因此得到的输出是初始预测，几乎一定会和真实值存在误差，不可能所有样本都恰好预测正确。我们计算得到的是预测值，和真实值的差就是模型误差，为了消除误差正负的影响，通常计算误差的平方，多个样本则把所有样本的误差加总得到总误差。

[00:11:52](https://getnotes.seek:712) **反向传播算法的核心作用**
计算得到预测误差后，就进入反向传播（也可翻译为后向传播）步骤，反向传播本质就是调整权重参数的过程。

[00:12:05](https://getnotes.seek:725) **反向传播的调参流程**
1. 根据随机初始权重做前向传播，得到预测输出，计算预测输出和真实值的误差；
2. 根据误差大小反向更新所有权重，目标是降低总误差；
3. 更新完权重后，再次做前向传播，得到新的预测输出，重新计算误差；
4. 如果误差仍然较大，重复反向传播调参，不断迭代。

[00:13:00](https://getnotes.seek:780) **神经网络训练的本质总结**
整个神经网络的训练过程，就是不断重复前向传播和反向传播的迭代过程：随机初始化权重→前向传播得预测→计算误差→反向传播更新权重→再次前向传播，循环直到误差降低到可接受范围。训练的最终目标就是得到误差最小的一组权重参数。

#### 💡 核心概念
- **前向传播算法**：给定输入和权重，从输入层逐层计算得到最终预测输出的过程。
- **反向传播算法**：根据预测误差，反向逐层更新权重以降低误差的调参过程。
- **总净输入（total net input）**：神经元对所有输入做加权加总得到的线性结果，公式：
  $$z_j = \sum w_{ij} x_i$$
- **激活函数**：对总净输入做非线性变换的函数，用于调节信号强度，为网络引入非线性，常见ReLU：
  $$f(z) = \max(0, z)$$
- **均方误差（MSE）**：神经网络常用的误差衡量指标，公式：
  $$MSE = \frac{1}{N}\sum_{i=1}^N (\hat{y}_i - y_i)^2$$
