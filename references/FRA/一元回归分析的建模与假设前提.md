#### 知识精讲

本课程是线性回归分析的复习与进阶，重点回顾一元线性回归的建模步骤、最小二乘法原理及经典假设，并为多元回归做铺垫。

**回归分析的基本框架**
* **四步流程**：建模 → ANOVA表分析（拟合优度）→ 模型检验 → 预测。其中前两步是所有回归分析的基础。
   - 🚩重点：模型检验的本质是检验真实系数（如 $\beta_1$）是否为0，而非估计系数。

**一元线性回归模型**
* **变量定义**：
   - 因变量（Dependent Variable / Explained Variable）：$Y$（被解释变量、预测对象）
   - 自变量（Independent Variable / Explanatory Variable）：$X$（解释变量、影响因素）
* **模型表达式**：
   - 估计的回归方程：$\hat{Y} = \hat{\beta}_0 + \hat{\beta}_1 X$ （根据课堂图片）
   - 真实模型：$Y = \beta_0 + \beta_1 X + \varepsilon$ （根据课堂图片）
   - 残差项：$e_i = Y_i - \hat{Y}_i = Y_i - (\hat{\beta}_0 + \hat{\beta}_1 X_i)$ （根据课堂图片）
* **系数解释**：
   - $\hat{\beta}_0$：当 $X=0$ 时，$Y$ 的估计值（截距项）。其期望值 $E(Y) = \beta_0$，因为 $E(\varepsilon) = 0$。
   - $\hat{\beta}_1$：$X$ 每变动1个单位，$Y$ 平均变动 $\hat{\beta}_1$ 个单位（斜率项）。

**最小二乘法 (OLS, Ordinary Least Squares)**
* **核心思想**：通过最小化残差平方和来估计系数 $\hat{\beta}_0$ 和 $\hat{\beta}_1$。
   - *公式*：$\min \sum_{i=1}^{n} e_i^2 = \min \sum_{i=1}^{n} (Y_i - \hat{Y}_i)^2$ （根据课堂图片）
* **几何意义**：找到一条直线，使所有样本点到该直线的垂直距离（残差）的平方和最小。

**一元线性回归的经典假设 (Assumptions)**
* **线性性 (Linearity)**：$Y$ 与 $X$ 之间存在线性关系。
   - 推论1：$X$ 是非随机的（所有随机性体现在残差项 $\varepsilon$ 中）。
   - 推论2：$\varepsilon$ 与 $X$ 不相关（$Cov(X, \varepsilon) = 0$）。
* **同方差性 (Homoscedasticity)**：残差项 $\varepsilon$ 的方差为常数，即 $Var(\varepsilon_i) = \sigma^2$。
   - 违反情况：异方差性 (Heteroscedasticity)，表现为残差波动随 $X$ 变化。
* **独立性 (Independence)**：不同观测值的残差项之间相互独立，即 $Cov(\varepsilon_i, \varepsilon_j) = 0$ for $i \neq j$。
   - 违反情况：自相关性 (Autocorrelation)，常见于时间序列数据。
* **正态性 (Normality)**：残差项 $\varepsilon$ 服从正态分布，即 $\varepsilon \sim N(0, \sigma^2)$。
   - 🚩重点：这四个假设的核心是残差项 $\varepsilon$ 应满足：均值为0、方差恒定、无自相关、正态分布，即“白噪声”序列特征。

#### 📖 课堂实录

[00:01:12](https://getnotes.seek:72) **回归分析框架回顾**

回归分析主要分为四步：建模、ANOVA表分析（拟合优度）、模型检验、预测。前两步是所有回归分析的基础。模型检验的本质是检验真实系数（如 $\beta_1$）是否为0，因为若真实 $\beta_1=0$，模型则无意义。这与一级假设检验的内容相通。

[00:03:04](https://getnotes.seek:184) **一元回归模型定义**

一元回归的特点是只有一个自变量 $X$。$Y$ 称为因变量（Dependent Variable）或被解释变量（Explained Variable），依赖于 $X$；$X$ 称为自变量（Independent Variable）或解释变量（Explanatory Variable），其取值独立于方程。**在做题时，看到 "dependent variable" 应记为 $Y$，"independent variable" 记为 $X$，以清晰梳理题目信息。**

[00:04:13](https://getnotes.seek:253) **最小二乘法原理**

建模的核心是估计系数 $\beta_0$ 和 $\beta_1$，采用最小二乘法（OLS）。基本理念是：对于散点图中的 $n$ 个样本点 $(X_i, Y_i)$，找到一条直线 $\hat{Y} = \hat{\beta}_0 + \hat{\beta}_1 X$，使得所有点到该直线的残差（$e_i = Y_i - \hat{Y}_i$）的平方和最小。即 $\min \sum_{i=1}^{n} e_i^2$。这里的 $\hat{Y}$ 读作 "Y-hat" 或 "Y-cap"，表示估计值。

[00:07:56](https://getnotes.seek:476) **真实模型与残差**

估计的回归方程 $\hat{Y} = \hat{\beta}_0 + \hat{\beta}_1 X$ 只是对真实关系的近似。真实模型应为 $Y = \beta_0 + \beta_1 X + \varepsilon$，其中 $\varepsilon$ 是残差项（扰动项），代表所有未被 $X$ 解释的随机因素。残差项 $\varepsilon$ 可视为一个“垃圾桶”，包含了除 $X$ 之外所有对 $Y$ 的次要影响因素。

[00:10:38](https://getnotes.seek:638) **一元回归的经典假设（Assumptions）**

使用OLS方法必须满足以下前提假设，这些假设在多元回归中大部分依然适用：

[00:10:59](https://getnotes.seek:659) **假设1：线性性（Linearity）**

$Y$ 与 $X$ 之间存在线性关系。推论包括：(1) $X$ 是非随机的，所有随机性都体现在 $\varepsilon$ 中；(2) $\varepsilon$ 与 $X$ 不相关。这意味着 $X$ 是给定的“原因”，$\varepsilon$ 是纯粹的随机“噪音”。

[00:15:31](https://getnotes.seek:931) **假设2：同方差性（Homoscedasticity）**

残差项 $\varepsilon$ 的方差为常数 $Var(\varepsilon_i) = \sigma^2$。若残差方差随 $X$ 变化（如 $X$ 增大时残差波动也增大），则称为异方差性（Heteroscedasticity），违反此假设。

[00:18:25](https://getnotes.seek:1105) **假设3：独立性（Independence）**

不同观测值的残差项之间相互独立，即 $Cov(\varepsilon_i, \varepsilon_j) = 0$（$i \neq j$）。若残差序列呈现规律性波动（如今天涨明天也涨），则存在自相关性（Autocorrelation），常见于时间序列数据（如季度销售额的季节性波动），违反此假设。🚩重点：此假设是二级考试的重点。

[00:24:28](https://getnotes.seek:1468) **假设4：正态性（Normality）**

残差项 $\varepsilon$ 服从正态分布 $\varepsilon \sim N(0, \sigma^2)$。可通过绘制残差的频率分布图进行检验。

[00:25:00](https://getnotes.seek:1500) **假设总结**

四个假设的核心是残差项 $\varepsilon$ 应满足：均值为0、方差恒定、无自相关、正态分布，即构成“白噪声”序列。多元回归仅比一元回归多一个假设，其他完全相同。

#### 💡 核心概念
- **被解释变量**：$Y$（Dependent Variable），模型试图解释或预测的变量。
- **解释变量**：$X$（Independent Variable），用于解释 $Y$ 变化的变量。
- **OLS估计**：通过 $\min \sum (Y_i - (\hat{\beta}_0 + \hat{\beta}_1 X_i))^2$ 估计系数。
- **残差**：$e_i = Y_i - \hat{Y}_i$，衡量模型拟合误差。
- **自相关**：残差序列 $\{e_i\}$ 自身存在相关性，违反独立性假设。

### 🎯 拍图总结
#### 📊 线性回归基础
- **核心模型**：  
  - 真实模型：$Y = \beta_0 + \beta_1 X + \varepsilon$，其中$\beta_0$为截距项，$\beta_1$为斜率系数，$\varepsilon$为随机误差项。  
  - 估计模型：$\hat{Y} = \hat{\beta}_0 + \hat{\beta}_1 X$，通过样本数据估计参数$\hat{\beta}_0$和$\hat{\beta}_1$。

- **最小二乘法（OLS）**：  
  - **原理**：通过最小化残差平方和估计参数，即$\min \sum_{i=1}^n e_i^2 = \min \sum_{i=1}^n (Y_i - \hat{Y}_i)^2$，其中$e_i = Y_i - \hat{Y}_i$为残差。  
  - **几何意义**：在散点图中，回归线是使所有观测点到直线的垂直距离平方和最小的直线（如图示：X为自变量，Y为因变量，残差为观测点与回归线的纵向距离）。

#### 🔄 模型形式对比
| **模型类型**       | **函数形式**                | **特点**                     |
|--------------------|-----------------------------|------------------------------|
| **线性回归**       | $Y = \beta_0 + \beta_1 X + \varepsilon$ | 变量间呈线性关系             |
| **指数模型（示例）** | $Y = \beta_0 e^{\beta_1 X} + \varepsilon$ | 变量间呈指数关系，需转换后线性化 |

#### 📈 模型应用与假设
- **核心假设**：线性回归要求误差项$\varepsilon$满足零均值、同方差、无自相关等条件（图示中“假设”部分提及，具体内容需结合后续课程）。  
- **系数解释**：$\hat{\beta}_1$表示X每变动1单位时，Y的平均变动量（如$\hat{\beta}_1=0.5$意味着X增加1，Y平均增加0.5）。  
- **残差分析**：通过残差平方和判断模型拟合优度，残差越小，模型对数据的解释能力越强。
