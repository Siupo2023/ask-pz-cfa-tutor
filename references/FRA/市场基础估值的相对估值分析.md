#### 录音总结
本次课程讲解聚焦市场基础估值（相对估值），涵盖企业价值比率、股息收益率、跨境估值差异、动量指标及中心趋势测量等核心内容，并通过实例演示计算方法与应用逻辑。

**企业价值比率（EV/EBITDA）**
* **EV/EBITDA的必要性**：传统价格类比率（如市盈率）分子为股权价值，分母若为企业层面指标（如EBITDA），会因企业资本结构不同导致结果失真。EV/EBITDA以企业价值（EV）为分子，EBITDA为分母，实现分子分母均为企业层面指标的匹配，适用于比较不同资本结构的企业。
* **EV的构成与计算**：EV = 普通股市场价值（股价×股数） + 债务市场价值 + 优先股市场价值（如有） - 现金及投资。其中，现金及投资需扣除是因为EBITDA反映的是生产性资产产生的现金流，非生产性资产（如现金、土地）未在EBITDA中体现，需单独处理。
* **EBITDA的计算**：EBITDA = 净利润 + 利息 + 税费 + 折旧与摊销。通过实例演示，某公司股价22.50美元，4000万股流通股，债务市值1.37亿美元，现金及证券6200万，投资3.27亿美元，经计算EV为64.77亿美元；净利润1.37亿美元，利息690万，折旧摊销1040万，税费9500万，计算EBITDA为2.5亿美元，EV/EBITDA比率为2.6。
* **EV/EBITDA的应用与优缺点**：通过与行业基准比较判断企业估值（如行业基准为4则2.6被视为低估）。优点是控制了折旧摊销差异，即使盈利为负EBITDA可能仍为正；缺点是忽略营运资本投资和资本支出，理论上EV/自由现金流更优，但实务中EBITDA倍数应用广泛。

**股息收益率（Dividend Yield）**
* **股息收益率的定义与计算**：股息收益率是总回报的组成部分（总回报=股息收益率+资本回报），计算公式为股息/股价，可使用 trailing（最近四次季度股息或最近一次季度股息×4）或 leading（预期年度股息）股息计算。
* **股息收益率与价值策略的关系**：高股息收益率通常与价值策略相关，因股息相对稳定，股价下跌会推高股息收益率，此类股票可能被市场低估，存在均值回归机会。
* **合理股息收益率（Justified Dividend Yield）**：公式为（必要回报率-增长率）/（1+增长率）。实际股息收益率高于合理值时，企业可能被低估。其影响因素包括：股息增加、必要回报率上升会使合理股息收益率上升；增长率上升会使合理股息收益率下降。

**跨境估值差异**
* **差异产生的原因**：包括会计处理差异（如美国与其他国家，甚至IFRS使用者之间）、文化差异（如日本企业倾向持有大量土地作为贷款抵押）、经济差异（经济周期管理、增长前景、风险等），导致同一行业不同国家企业的市盈率等比率差异显著且长期存在。

**动量指标（Momentum Indicators）**
* **基于价格的动量指标**：如相对强度指标，属于技术分析范畴。
* **基于盈利的动量指标**：主要指盈利惊喜（Earnings Surprise），即实际EPS与预期EPS的差额，可标准化为（盈利惊喜/盈利惊喜标准差）。研究发现盈利惊喜具有持续性，市场调整存在滞后，但长期来看动量效应难以持续。

**中心趋势测量（Measures of Central Tendency）**
* **算术平均值（Arithmetic Mean）**：易受极端值（尤其是高值）影响。
* **调和平均值（Harmonic Mean）**：相比算术平均值，受高极端值影响较小，但受低极端值影响较大，适用于无市场权重时，会给出比算术平均值更低的结果（存在变异时）。
* **加权调和平均值（Weighted Harmonic Mean）**：按市场价值加权，更接近市场价值加权组合的倍数评估。
* **中位数（Median）**：取中间值，最不受极端值影响。

### 📅 章节概要
[00:00:02](https://getnotes.seek:2) **课程引入：相对估值剩余内容概述**

本次课程继续讲解市场基础估值（相对估值），剩余内容包括企业价值比率、股息收益率及总结概念，此前已完成主要比率的学习。

[00:00:30](https://getnotes.seek:30) **回顾价格类比率及其问题**

已学习市盈率、市净率、市销率、市现率等价格类比率，指出使用EBITDA比率（如价格/EBITDA）的问题：分子为股权价值，分母为企业层面的EBITDA，当企业资本结构不同时结果会失真。

[00:01:18](https://getnotes.seek:78) **企业价值比率（EV/EBITDA）的提出**

解决价格/EBITDA比率问题的方法是将分子替换为企业价值（EV），形成EV/EBITDA比率。EV是企业价值，EBITDA是企业层面现金流的粗略 proxy，二者匹配可消除资本结构差异的影响。

[00:02:13](https://getnotes.seek:133) **EV/EBITDA的逻辑与应用注意事项**

EV/EBITDA的分子（EV）与分母（EBITDA）均为企业层面指标，EBITDA近似代表债务和股权的收益流。使用该比率得到的是企业价值，若需股权价值需减去债务价值。

[00:02:43](https://getnotes.seek:163) **EV的构成要素**

计算EV需明确分子分母的匹配，包括普通股市场价值（股价×股数）、债务市场价值、优先股市场价值（如有），并减去现金及投资。

[00:03:30](https://getnotes.seek:210) **扣除现金及投资的原因**

EBITDA反映生产性资产产生的现金流，现金及投资等非生产性资产不贡献EBITDA，因此计算EV时需扣除，若存在未反映在EBITDA中的其他资产（如土地），后续需加回。

[00:04:52](https://getnotes.seek:292) **EV/EBITDA比率的应用示例**

假设可比公司EV/EBITDA比率为5，目标公司EBITDA为10，则目标公司EV约为50，此价值基于生产性资产的现金流能力，不包含非生产性资产。

[00:06:08](https://getnotes.seek:368) **EV计算实例引入**

给出实例数据：股价22.50美元，4000万股流通股，债务市值1.37亿美元，现金及证券6200万，投资3.27亿美元，需计算EV。

[00:06:39](https://getnotes.seek:399) **EV计算逻辑**

EV = 普通股市场价值 + 债务市场价值 - 现金及投资，目的是建立EV与EBITDA的良好关系。

[00:06:59](https://getnotes.seek:419) **EBITDA计算逻辑**

需从净利润倒推EBITDA：净利润 + 税费 + 利息 + 折旧与摊销。实例中给出净利润，需加回利息（690万）、折旧摊销（1040万）、税费（9500万）。

[00:07:21](https://getnotes.seek:441) **EV计算结果**

普通股市场价值 = 22.50×4000万 = 9亿美元，EV = 9亿 + 1.37亿 - 6200万 - 3.27亿 = 64.77亿美元。

[00:07:46](https://getnotes.seek:466) **EBITDA计算与EV/EBITDA比率**

EBITDA = 净利润（1.37亿） + 利息（690万） + 折旧摊销（1040万） + 税费（9500万） = 2.5亿美元，EV/EBITDA = 64.77亿 / 2.5亿 = 2.6。

[00:08:16](https://getnotes.seek:496) **EV/EBITDA比率的估值应用**

通过与行业基准比较判断估值，如行业基准为4则2.6表明低估，基准为0.4则表明高估，该比率反映企业整体价值（债务+股权）。

[00:08:56](https://getnotes.seek:536) **EV/EBITDA的优势**

相比价格/EBITDA，EV/EBITDA因分子分母均为企业层面指标，可控制资本结构差异；EBITDA加回折旧摊销，可控制折旧政策差异（如5年与50年折旧）；EBITDA在盈利为负时可能仍为正。

[00:09:32](https://getnotes.seek:572) **EV/EBITDA的局限性**

忽略营运资本投资和资本支出，理论上EV/自由现金流更优，但实务中EBITDA倍数应用广泛，课程按此覆盖。

[00:10:00](https://getnotes.seek:600) **基于EV/EBITDA的估值判断**

当EV/EBITDA低于基准时企业被低估（如2.6 < 4），高于基准时被高估（如26 > 15），与市盈率等比率的估值逻辑一致。

[00:10:51](https://getnotes.seek:651) **股息收益率的引入**

股息收益率是潜在考点，属于总回报的一部分，有人认为股息部分比总回报风险更低（“在手之鸟”理论），课程按教材观点讲解。

[00:11:59](https://getnotes.seek:719) **股息收益率的计算方式**

Trailing股息收益率 = 最近一次季度股息×4 / 当前股价；Leading股息收益率 = 预期年度股息 / 当前股价。

[00:12:36](https://getnotes.seek:756) **股息收益率与价值策略**

股息收益率 = 股息/股价，因股价为正而股息可能为零，故采用此形式。高股息收益率通常对应价值策略，即股息相对股价较高，可能因股价被市场低估。

[00:13:00](https://getnotes.seek:780) **合理股息收益率公式**

合理股息收益率 = （必要回报率 - 增长率）/（1 + 增长率）。实际股息收益率高于合理值时，企业可能被低估，与市盈率等比率不同（市盈率低为低估，股息收益率高为低估）。

[00:14:00](https://getnotes.seek:840) **影响合理股息收益率的因素**

股息增加、必要回报率上升会使合理股息收益率上升；增长率上升会使合理股息收益率下降，因高增长企业可能更少分红。

[00:14:32](https://getnotes.seek:872) **高股息收益率与价值策略的关联**

股息相对稳定，若股价下跌（企业被市场看空），股息收益率上升，此时买入高股息收益率股票属于价值策略，预期股价均值回归。

[00:15:30](https://getnotes.seek:930) **跨境估值差异的原因**

包括会计差异（美国与其他国家，IFRS使用者之间）、文化差异（如日本企业持有大量土地作为贷款抵押）、经济差异（经济周期、增长前景、风险等），导致同一行业不同国家企业的市盈率差异显著且持续。

[00:17:00](https://getnotes.seek:1020) **动量指标的概念**

动量指标是另一种买卖决策依据，包括基于价格的指标（如相对强度，技术分析）和基于盈利的指标（如盈利惊喜），曾被视为重要市场异象。

[00:17:38](https://getnotes.seek:1058) **盈利惊喜（Earnings Surprise）**

盈利惊喜 = 实际EPS - 预期EPS，预期EPS无官方数据，依赖分析师估计等。研究发现盈利惊喜具有持续性，股价在惊喜公布后会持续反应，且惊喜可能连续出现。

[00:19:10](https://getnotes.seek:1150) **标准化盈利惊喜**

标准化盈利惊喜 = 盈利惊喜 / 盈利惊喜的标准差，使不同企业的惊喜程度具有可比性，但长期市场会调整，动量效应难以永久持续。

[00:19:33](https://getnotes.seek:1173) **中心趋势测量方法**

比较样本的多个比率（如5个市盈率）需通过中心趋势测量整合，包括算术平均值、调和平均值、加权调和平均值、中位数。

[00:19:57](https://getnotes.seek:1197) **算术平均值的特点**

算术平均值 = 总和/个数，易受极端高值影响（如10,10,10,10,25的算术平均值受25影响较大）。

[00:20:10](https://getnotes.seek:1210) **调和平均值的特点**

调和平均值 = n / (Σ(1/xi))，受极端高值影响较小，受极端低值影响较大，无市场权重时使用，结果低于算术平均值（存在变异时）。

[00:20:28](https://getnotes.seek:1228) **加权调和平均值与中位数**

加权调和平均值按市场价值加权，更接近市场组合的倍数评估；中位数取中间值，最不受极端值影响。

[00:21:38](https://getnotes.seek:1298) **算术平均值与调和平均值的比较**

当所有比率相同时（如10,10,10），二者相等；存在变异时（如10,11,9），调和平均值低于算术平均值。

[00:22:51](https://getnotes.seek:1371) **相对估值章节总结**

相对估值是重要且易考内容，难度不高，涉及公式、影响因素、可比法与基本面预测法、PEG比率、EV/EBITDA、市现率、股息收益率、中心趋势测量等，需重点掌握以提高考试分数。
