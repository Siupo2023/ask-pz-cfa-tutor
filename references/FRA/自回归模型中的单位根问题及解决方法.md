#### 录音总结
本次课程继续讲解自回归模型，核心议题为单位根问题，包括随机游走定义、单位根的影响、迪基-富勒检验及一阶差分法修正。

**随机游走与单位根的定义**
* **随机游走的定义**：自回归模型（AR模型）中存在单位根（即自回归系数等于1）时，该模型被称为随机游走。其关键特征是自回归系数为1，而非截距项（漂移项）的存在与否。
* **随机游走的两种形式**：随机游走模型可分为无漂移和有漂移两种。无漂移模型中截距项B0为零，模型表示为xt = xt-1 + 误差项；有漂移模型中截距项B0非零，模型表示为xt = B0 + xt-1 + 误差项。无论是否有漂移，单位根（B1=1）是其核心定义特征。

**单位根的影响**
* **破坏协方差平稳性**：在一阶自回归模型（AR(1)）中，当自回归系数B1等于1（存在单位根）时，模型不再是协方差平稳的。协方差平稳要求B1的绝对值小于1，此时模型存在均值回复水平；而B1=1会导致均值回复水平公式中出现除以零的情况，因此不存在均值回复水平。
* **常见于趋势性时间序列**：单位根常见于持续上升或下降的时间序列，例如90年代多数资产类别持续上涨的股票市场，此类序列因缺乏均值回复特性，更可能存在单位根。

**单位根的检验：迪基-富勒检验**
* **检验原理与模型转换**：迪基-富勒检验用于检测单位根的存在。其通过对原AR模型进行转换，将xt = B0 + B1*xt-1 + 误差项转换为Δxt = B0 + (B1 - 1)*xt-1 + 误差项，定义新的斜率系数G1 = B1 - 1。
* **假设设定与判断标准**：检验的原假设为G1 = 0（即B1 = 1，存在单位根），备择假设为G1 < 0（即B1 < 1，不存在单位根）。若未能拒绝原假设，则表明存在单位根；若拒绝原假设，则表明不存在单位根。该检验使用的t统计量需与修正后的临界t值进行比较，而非标准的学生t分布。

**单位根问题的修正：一阶差分法**
* **一阶差分法的定义**：一阶差分法是通过对时间序列进行差分变换以消除单位根的数学过程。具体而言，定义新变量yt = xt - xt-1（即变量的变化量），将原模型转换为对变化量的建模。
* **修正效果**：经过一阶差分后，新模型的自回归系数B1变为0，满足协方差平稳的条件（B1 < 1）。从图形上看，原本持续上升或下降的序列（无均值回复）经差分后，新序列围绕均值（通常为零）波动，具备均值回复特性，从而成为协方差平稳序列。

[00:10:45](https://getnotes.seek:645) **一阶差分法的操作**

以AR(1)模型为例，说明一阶差分法是将原模型中对变量绝对值的建模，改为对变量变化量的建模，即建模Δxt（xt - xt-1）。
