#### 录音总结
本次课程讲解聚焦企业合并中的商誉问题，涵盖商誉的定义、计算（部分商誉与完全商誉）、减值测试及负商誉处理，并涉及企业合并相关法定定义。

**企业合并相关法定定义**
* **合并（Merger）**：两家公司（如公司A和公司B）合并，仅公司A存续，公司B的净资产转移至公司A后停止存在。
* **收购（Acquisition）**：两家公司合并后，一家成为母公司，另一家成为子公司，两者均继续存在。
* **新设合并（Consolidation）**：公司A和公司B合并形成全新的公司C，原公司A和B均停止存在。
* **可变利益实体（Variable Interest Entities, VIE）**：控制权并非基于股权 ownership，常被称为特殊目的实体（SPV），US GAAP常使用此术语，其控制通过可变利益而非股本实现，将在未来模块详细探讨。

**商誉的基本概念与计算基础**
* **商誉的性质**：商誉是不可辨认资产，因其无法与公司分离单独出售，在编制合并报表时，子公司自身报表中的商誉需被排除，仅考虑可辨认净资产的公允价值。
* **商誉计算示例**：Wood公司以6亿美元收购Pine公司100%股权。Pine公司报表显示流动资产8000万、净PP&E（厂房、 property和设备净值）7.6亿、负债4亿，子公司自身商誉不计入。其净资产账面价值为8000万 + 7.6亿 - 4亿 = 4.4亿。PP&E公允价值高于账面价值1.2亿，故可辨认净资产公允价值为4.4亿 + 1.2亿 = 5.6亿。商誉 = 收购价6亿 - 可辨认净资产公允价值5.6亿 = 4000万。

**部分商誉与完全商誉**
* **适用场景差异**：当收购100%股权时，部分商誉与完全商誉计算结果无差异；仅在收购少于100%股权时存在区别。
* **会计准则要求**：US GAAP要求使用完全商誉；IFRS允许选择完全商誉或部分商誉。部分商誉的计算方法与联营企业的权益法核算类似。
* **部分商誉计算**：公式为收购价格 - 收购方所占可辨认净资产公允价值份额。例如，Wood公司以4.5亿美元收购Pine公司75%股权，可辨认净资产公允价值5.6亿，收购方份额为5.6亿 × 75% = 4.2亿，部分商誉 = 4.5亿 - 4.2亿 = 3000万。
* **完全商誉计算**：公式为子公司整体公允价值 - 可辨认净资产公允价值。通过收购价推算子公司整体公允价值，如4.5亿 / 75% = 6亿，完全商誉 = 6亿 - 5.6亿 = 4000万。两者可相互转换，部分商誉除以持股比例可得完全商誉，反之亦然。
* **对财务报表影响**：完全商誉比部分商誉高1000万（4000万 - 3000万），导致资产负债表资产增加1000万，为保持平衡，非控制性权益（少数股东权益）也需相应增加1000万。完全商誉下非控制性权益 = 子公司整体公允价值 × 少数股权比例 = 6亿 × 25% = 1.5亿；部分商誉下非控制性权益 = 可辨认净资产公允价值 × 少数股权比例 = 5.6亿 × 25% = 1.4亿。

**商誉减值**
* **减值测试要求**：商誉不进行摊销，需每年强制进行减值测试，无论是否存在减值迹象。
* **减值判断标准**：当报告单元（或现金生成单元）的账面价值（商誉 + 净资产）大于其公允价值时，发生减值。
* **减值处理影响**：减值损失计入利润表，作为非经常性或非频繁项目，同时减少资产负债表中的商誉价值，且商誉减值损失不可转回（US GAAP不允许任何资产减值转回，IFRS通常允许但商誉除外，因子公司公允价值主观性高，易被用于盈余管理）。
* **IFRS减值测试步骤**：采用一步法，比较现金生成单元的可收回金额（处置净价与使用价值两者较高者）与账面价值，若可收回金额低于账面价值，差额即为减值损失。
* **US GAAP减值测试步骤**：采用两步法。第一步比较报告单元的公允价值与账面价值，若公允价值低于账面价值则需进行第二步；第二步计算减值损失，即原商誉账面价值与当前隐含商誉（报告单元公允价值 - 可辨认净资产公允价值）的差额。
* **减值损失超过商誉账面价值的处理**：IFRS下，先将商誉减记至零，剩余损失再冲减其他非现金资产（如无形资产、PP&E）；US GAAP下，仅将商誉减记至零，超额部分不予处理。
* **减值示例**：母公司以100万收购子公司，收购日可辨认净资产公允价值80万，初始商誉20万。年末子公司账面价值（含商誉）98万，公允价值95万，可辨认净资产公允价值77.5万。IFRS减值损失 = 98万 - 95万 = 3万，商誉减记至17万；US GAAP先确认减值迹象（95万 < 98万），再计算隐含商誉 = 95万 - 77.5万 = 17.5万，减值损失 = 20万 - 17.5万 = 2.5万，商誉减记至17.5万。

**负商誉**
* **定义与罕见性**：负商誉指收购价格低于子公司可辨认净资产公允价值，这种情况非常罕见，因为出售方通常可通过拆分出售资产获得更高价值。
* **处理流程**：首先需重新检查收购日可辨认净资产的公允价值评估是否准确；若确认无误，则负商誉立即作为利得计入收购当期的利润表。

商誉是不可辨认资产，这在一级课程已学过，因其无法与公司分离单独出售，所以在合并时需排除子公司自身的商誉。

[00:03:14](https://getnotes.seek:194) **合并报表中可辨认净资产的处理原则**

编制合并报表时，需纳入子公司可辨认净资产的公允价值，因此要忽略子公司自身的商誉，接下来开始商誉计算。

[00:03:42](https://getnotes.seek:222) **PP&E的公允价值调整**

[00:04:13](https://getnotes.seek:253) **商誉计算步骤：确定可辨认净资产公允价值**

由于 PP&E 公允价值高于账面价值1.2亿，所以可辨认净资产公允价值 = 4.4亿 + 1.2亿 = 5.6亿。

[00:05:16](https://getnotes.seek:316) **商誉计算步骤：计算商誉金额**

因收购100%股权，需减去100%的可辨认净资产公允价值。收购价6亿 - 可辨认净资产公允价值5.6亿 = 4000万，此为合并报表中的商誉金额。

收购100%股权时，部分商誉与完全商誉无差异；仅在收购少于100%股权时存在区别。

[00:06:25](https://getnotes.seek:385) **部分商誉与完全商誉的会计准则要求**

US GAAP 要求使用完全商誉；IFRS 允许选择完全商誉或部分商誉，部分商誉的计算方法与联营企业的权益法核算类似。

[00:06:45](https://getnotes.seek:405) **部分商誉的计算方法**

部分商誉 = 收购价格 - 收购方所占可辨认净资产公允价值份额，需注意排除子公司自身商誉。

[00:06:59](https://getnotes.seek:419) **非控制性权益在部分商誉下的计算**

非控制性权益（少数股东权益） = 可辨认净资产公允价值 × 少数股东持股比例。

[00:07:20](https://getnotes.seek:440) **完全商誉的计算方法**

完全商誉 = 子公司整体公允价值 - 可辨认净资产公允价值，US GAAP 要求使用，IFRS 可选。

[00:07:49](https://getnotes.seek:469) **部分商誉与完全商誉计算术语差异**

部分商誉使用“收购价格”和“收购方所占可辨认净资产份额”；完全商誉使用“子公司整体公允价值”和“100%可辨认净资产公允价值”。

[00:08:15](https://getnotes.seek:495) **100%股权下两种商誉计算的一致性**

收购100%股权时，收购价格等于子公司整体公允价值，收购方所占份额等于100%可辨认净资产公允价值，故两种方法结果一致。

[00:08:50](https://getnotes.seek:530) **非控制性权益在完全商誉下的计算**

完全商誉下，非控制性权益 = 子公司整体公允价值 × 少数股东持股比例，因商誉计算不同，非控制性权益金额与部分商誉下不同，以保持资产负债表平衡。

[00:09:10](https://getnotes.seek:550) **部分商誉与完全商誉计算示例引入**

延续前例，假设 Wood 公司以4.5亿美元收购 Pine 公司75%股权，分别用两种方法计算商誉。

[00:09:27](https://getnotes.seek:567) **完全商誉计算：确定子公司整体公允价值**

通过收购价推算子公司整体公允价值，4.5亿（75%股权价格） / 75% = 6亿，即可辨认净资产公允价值5.6亿，完全商誉 = 6亿 - 5.6亿 = 4000万。

[00:10:28](https://getnotes.seek:628) **部分商誉计算：确定收购方所占可辨认净资产份额**

可辨认净资产公允价值5.6亿，收购方75%份额为5.6亿 × 75% = 4.2亿，部分商誉 = 4.5亿 - 4.2亿 = 3000万。

[00:10:50](https://getnotes.seek:650) **部分商誉与完全商誉的转换方法**

部分商誉除以持股比例可得完全商誉（3000万 / 75% = 4000万），完全商誉乘以持股比例可得部分商誉，考试中可灵活转换。

[00:11:41](https://getnotes.seek:701) **两种商誉计算结果对比及对资产的影响**

完全商誉4000万比部分商誉3000万高1000万，导致资产负债表资产增加1000万。

[00:12:11](https://getnotes.seek:731) **两种商誉下非控制性权益的差异及资产负债表平衡**

完全商誉下非控制性权益 = 6亿 × 25% = 1.5亿，部分商誉下非控制性权益 = 5.6亿 × 25% = 1.4亿，差异1000万，与商誉差异一致，保持资产负债表平衡。

[00:12:45](https://getnotes.seek:765) **商誉减值的基本要求**

商誉不摊销，需每年强制进行减值测试，无论是否存在减值迹象，一级课程已提及。

[00:13:57](https://getnotes.seek:837) **商誉减值的判断标准**

当报告单元（或现金生成单元）的账面价值（商誉 + 净资产）大于其公允价值时，发生减值。

[00:14:20](https://getnotes.seek:860) **商誉的不可分离性与减值测试单元**

商誉无法与整体业务分离，减值测试需针对包含商誉的业务单元进行。

[00:14:32](https://getnotes.seek:872) **报告单元账面价值的构成**

报告单元的账面价值包括合并报表中该子公司的商誉及净资产（资产与负债）。

[00:14:44](https://getnotes.seek:884) **减值损失的会计处理**

减值损失计入利润表，作为非经常性或非频繁项目，同时减少资产负债表中的商誉价值。

[00:15:06](https://getnotes.seek:906) **商誉减值损失的不可转回性**

US GAAP 不允许任何资产减值转回（除持有待售资产外），IFRS 通常允许但商誉减值不可转回，因子公司公允价值主观性高，易被用于盈余管理。

[00:15:33](https://getnotes.seek:933) **IFRS下商誉减值的分摊**

IFRS 要求商誉减值损失分摊至所有从收购中受益的现金生成单元（可能涉及分部报告），影响分部报告中的利润，但不影响合并利润表总额。

[00:16:00](https://getnotes.seek:960) **US GAAP下商誉减值的分摊**

US GAAP 要求将减值损失分摊至所有报告单元（可能是经营分部或组成部分），与 IFRS 理念相似。

[00:16:36](https://getnotes.seek:996) **IFRS商誉减值测试步骤（一步法）**

比较现金生成单元的可收回金额（处置净价与使用价值两者较高者）与账面价值，若可收回金额低于账面价值，差额即为减值损失。

[00:17:21](https://getnotes.seek:1041) **可收回金额的构成**

处置净价 =  selling price - 处置成本；使用价值 = 未来现金流量的现值，两者取较高者，计算主观性强。

[00:17:36](https://getnotes.seek:1056) **US GAAP商誉减值测试步骤（两步法）**

第一步：比较报告单元的公允价值（通常为 selling price，不扣除处置成本）与账面价值，若公允价值低于账面价值则需进行第二步；第二步：计算减值损失 = 原商誉账面价值 - 当前隐含商誉（报告单元公允价值 - 可辨认净资产公允价值）。

[00:18:45](https://getnotes.seek:1125) **减值损失超过商誉账面价值的处理（IFRS）**

IFRS 下，先将商誉减记至零，剩余损失冲减其他非现金资产（如无形资产、PP&E）。

[00:19:19](https://getnotes.seek:1159) **减值损失超过商誉账面价值的处理（US GAAP）**

US GAAP 下，仅将商誉减记至零，超额部分不予处理。

[00:19:32](https://getnotes.seek:1172) **商誉减值示例引入**

母公司去年以100万收购子公司，收购日可辨认净资产公允价值80万，年末子公司账面价值（含商誉）98万，公允价值95万，可辨认净资产公允价值77.5万，分别计算 IFRS 和 US GAAP 下的减值损失。

[00:20:00](https://getnotes.seek:1200) **初始商誉计算**

初始商誉 = 收购价100万 - 可辨认净资产公允价值80万 = 20万。

[00:20:15](https://getnotes.seek:1215) **IFRS减值损失计算**

子公司账面价值98万 > 公允价值95万，减值损失 = 98万 - 95万 = 3万，商誉减记至20万 - 3万 = 17万，损失计入利润表。

[00:20:36](https://getnotes.seek:1236) **US GAAP减值测试第一步：确认减值迹象**

子公司公允价值95万 < 账面价值98万，存在减值迹象，需进行第二步。

[00:21:09](https://getnotes.seek:1269) **US GAAP减值测试第二步：计算隐含商誉及减值损失**

当前隐含商誉 = 子公司公允价值95万 - 可辨认净资产公允价值77.5万 = 17.5万，减值损失 = 原商誉20万 - 隐含商誉17.5万 = 2.5万，商誉减记至17.5万，损失计入利润表。

[00:21:36](https://getnotes.seek:1296) **负商誉的定义**

负商誉指收购价格低于子公司可辨认净资产公允价值，这种情况非常罕见。

[00:21:53](https://getnotes.seek:1313) **负商誉罕见的原因**

理性的出售方通常会选择拆分出售资产以获得可辨认净资产公允价值，而非低价整体出售。

[00:22:18](https://getnotes.seek:1338) **负商誉的处理流程（第一步）**

首先需重新检查收购日可辨认净资产的公允价值评估是否准确。

[00:22:42](https://getnotes.seek:1362) **负商誉的处理流程（第二步）**

若确认公允价值评估准确，负商誉立即作为利得计入收购当期的利润表。

[00:23:00](https://getnotes.seek:1380) **负商誉的会计处理结果**

负商誉不能作为负无形资产列示，需直接计入当期损益，影响收购当期的 earnings。
