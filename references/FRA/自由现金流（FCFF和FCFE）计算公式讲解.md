#### 录音总结
本次课程讲解围绕自由现金流（FCFF和FCFE）的剩余组成部分“净借款”展开，并详细推导了不同起点（如净利润、CFO、EBIT、EBITDA）下FCFF和FCFE的计算公式，强调不同公式应得出一致结果。

**自由现金流组成部分与公式基础**
* **净借款的定义与影响**：净借款是FCFE的组成部分，指当年新增借款减去旧债偿还，体现为资产负债表中长短期债务及应付票据的变化额。它是债务资本提供者的现金流，仅与FCFE相关，FCFF不考虑此类内部现金流。
* **FCFF与FCFE的共同与独特组成**：两者均包含净利润、非现金费用、营运资本投资和固定资本投资四个共同部分。FCFF的独特部分是加回税后利息，FCFE的独特部分是加回净借款，这两个独特部分均与债务相关现金流有关。

**FCFF与FCFE公式推导（基于净利润）**
* **FCFF公式（净利润起点）**：FCFF = 净利润 + 非现金费用 - 营运资本投资 - 固定资本投资 + 税后利息。加回税后利息是因为FCFF是流向公司的现金流，需将支付给债权人的利息（已从净利润中扣除）加回。
* **FCFE公式（净利润起点）**：FCFE = 净利润 + 非现金费用 - 营运资本投资 - 固定资本投资 + 净借款。加回净借款是因为FCFE是流向股权投资者的现金流，净借款是股权投资者可获得的资金来源。

**基于CFO的FCFF与FCFE公式**
* **FCFF（CFO起点）**：CFO = 净利润 + 非现金费用 - 营运资本投资，故FCFF = CFO + 税后利息 - 固定资本投资。
* **FCFE（CFO起点）**：FCFE = CFO - 固定资本投资 + 净借款。两者均以CFO为基础，分别添加各自的独特债务相关项并考虑固定资本投资。

**基于EBIT的FCFF公式推导**
* **从EBIT到FCFF**：净利润 = EBIT×(1-税率) - 利息×(1-税率)，代入FCFF公式后利息×(1-税率)项正负抵消，得到FCFF = EBIT×(1-税率) + 折旧（非现金费用） - 营运资本投资 - 固定资本投资。

**基于EBITDA的FCFF公式推导**
* **EBITDA的分解与调整**：EBITDA×(1-税率) = EBIT×(1-税率) + 折旧×税率（折旧税盾），因此FCFF = EBITDA×(1-税率) + 折旧×税率 - 营运资本投资 - 固定资本投资，需注意添加折旧税盾以还原完整的非现金费用影响。

**公式应用与一致性原则**
* **公式选择依据**：应根据题目或考试提供的数据选择最简便的公式。
* **结果一致性**：无论从净利润、CFO、EBIT还是EBITDA出发，正确应用公式计算出的FCFF和FCFE值应唯一且一致。

[00:00:21](https://getnotes.seek:21) **介绍净借款及其对FCFE的影响**

剩余组成部分是净借款，净借款只影响FCFE，它是来自债务资本提供者的现金流。

[00:00:34](https://getnotes.seek:34) **解释FCFF不考虑内部现金流，FCFE考虑**

将公司比作家庭，家庭内部成员间的资金往来在FCFF计算中不相关，但在FCFE中相关。净借款等于当年新增借款减去旧债偿还。

[00:01:00](https://getnotes.seek:60) **说明净借款在资产负债表中的体现**

净借款体现在资产负债表中长短期债务和应付票据的变化，负债账户的增加已反映借款与偿还的净结果。

[00:01:26](https://getnotes.seek:86) **强调掌握五部分基本公式的重要性**

开始讲解FCFF和FCFE的公式，强调需先掌握五部分基本公式，后续公式均为其会计上的排列组合与相互关联。

[00:02:07](https://getnotes.seek:127) **指出FCFF与FCFE的共同组成部分**

FCFF和FCFE有四个共同组成部分：净利润、非现金费用、营运资本投资和固定资本投资。

[00:02:26](https://getnotes.seek:146) **说明FCFF与FCFE的独特组成部分**

FCFF的第五部分是加回税后利息，FCFE的第五部分是加回净借款，这两个独特部分均与债务相关现金流有关。

[00:02:42](https://getnotes.seek:162) **解释FCFF加回税后利息的原因**

FCFF是流向公司的现金流，加回税后利息是因为利息已从净利润中扣除，而公司内部成员间的现金流需加回。

[00:02:55](https://getnotes.seek:175) **解释FCFE加回净借款的原因**

FCFE加回净借款是因为股权投资者只需减去其在固定资本和营运资本投资中的份额，净借款是股权投资者的现金来源。

[00:03:32](https://getnotes.seek:212) **重新表述FCFE公式**

FCFE公式可表述为：净利润 + 非现金费用 - 股权投资者在固定资本和营运资本投资中的份额。

[00:04:02](https://getnotes.seek:242) **推导FCFE与FCFF的关系及基于CFO的公式起点**

FCFF减去税后利息并加上净借款可得到FCFE。指出CFO = 净利润 + 非现金费用 - 营运资本投资，可作为FCFF和FCFE公式的起点。

[00:04:27](https://getnotes.seek:267) **推导基于CFO的FCFF公式**

以CFO为起点，FCFF = CFO + 税后利息 - 固定资本投资，其中税后利息是第五部分，固定资本投资是第四部分。

[00:04:57](https://getnotes.seek:297) **推导基于CFO的FCFE公式及第五部分符号规则**

以CFO为起点，FCFE = CFO - 固定资本投资 + 净借款。强调FCFF和FCFE公式中的第五部分均为正值。

[00:05:30](https://getnotes.seek:330) **推导基于EBIT的FCFF公式**

若起点为EBIT，净利润 = EBIT×(1-税率) - 利息×(1-税率)，代入FCFF公式后利息项抵消，得到FCFF = EBIT×(1-税率) + 折旧 - 营运资本投资 - 固定资本投资。

[00:06:37](https://getnotes.seek:397) **解释EBITDA的定义及与FCFF的关联**

EBITDA是息税折旧摊销前利润，若乘以(1-税率)，在无摊销时可分解为EBIT×(1-税率) - 折旧×(1-税率)，需注意调整以还原非现金费用影响。

[00:07:05](https://getnotes.seek:425) **说明从EBITDA推导FCFF需添加折旧税盾**

从EBITDA出发推导FCFF时，需添加折旧税盾（折旧×税率）以弥补仅用EBITDA×(1-税率)时缺失的部分，从而与基于EBIT的FCFF公式一致。

[00:08:09](https://getnotes.seek:489) **重申FCFF公式的不同表达及NCC与折旧的关系**

FCFF可表示为EBIT×(1-税率) + 折旧 - 营运资本投资 - 固定资本投资，非现金费用（NCC）是比折旧更宽泛的概念，使用时需注意其内涵。

[00:08:58](https://getnotes.seek:538) **强调FCFF与FCFE公式中的关键区别**

FCFF不包含净借款，添加的是税后利息；FCFE则基于CFO（净利润 + NCC - 营运资本投资），减去固定资本投资并加上净借款。

[00:09:21](https://getnotes.seek:561) **阐述FCFF与FCFE的转换及结果唯一性**

FCFF减去税后利息并加上净借款可得到FCFE。强调FCFF和FCFE各有唯一的估计值，无论使用何种公式或起点，计算结果应一致。

[00:09:52](https://getnotes.seek:592) **建议根据数据选择公式**
