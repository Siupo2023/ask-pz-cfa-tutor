## 🎯 核心知识点

### 1️⃣ 基本概念：Transaction with Associate

**定义**：
- 投资公司与联营企业（associate）之间的内部交易
- 需要对投资收益确认进行调整
- 这是权益法下的**第二个调整项**

**为什么需要调整？**
> **核心逻辑**（老师的橡皮比喻）：
> "如果我把一块橡皮卖给李老师，卖了1万块。你说我们家有没有赚钱？如果赚的话，我卖给你、你卖给我，我们家不就变成亿万富翁了吗？"
>
> **结论**: 只要把投资公司和持股比例的联营企业看成一个整体，内部交易就像"自己关起门来自己玩"，没有真正赚钱。

### 2️⃣ 未实现内部交易损益的判定

**已实现损益** ✅
- 货物已经转卖给**第三方**
- 这部分利润可以确认
- **不需要**从投资收益中剔除

**未实现损益** ❌
- 货物**未转卖给第三方**（仍作为存货）
- 这部分利润**没有真正实现**
- **必须**从投资收益中剔除

### 3️⃣ 顺流交易 vs 逆流交易

**推荐记忆方法**（老师的秘诀）：

```
不要记中文"顺流逆流"！
直接看英文方向：

┌─────────┐
│ 投资公司 │ (A在上面，因为是大的)
└────┬────┘
     │
     │  ⬆️ Upstream (逆流) = 联营企业卖货给投资公司
     │  ⬇️ Downstream (顺流) = 投资公司卖货给联营企业
     ▼
┌─────────┐
│ 联营企业 │ (B在下面，因为是小的)
└─────────┘
```

**金句**:
> "顺流逆流不如看方向记，up向上就是逆流，down向下就是顺流，比中文好记太多。"

### 4️⃣ 逆流交易（Upstream）的处理

**方向**: 联营企业 → 投资公司（B卖货给A）

**利润位置**: 记录在**联营企业**的净利润中

**处理逻辑**：
```
假设：
- A投资了B 20%股权
- B公司净利润: 3500万
- 其中包含未实现内部交易损益: 500万

调整思路：
1. 把A和20%的B看成一个整体
2. 500万未实现损益中，20%对应100万是"自己人"之间的
3. 这100万不承认，需要剔除

最终Equity Income = 3500万×20% - 500万×20%
                 = 700万 - 100万
                 = 600万
```

### 5️⃣ 顺流交易（Downstream）的处理

**方向**: 投资公司 → 联营企业（A卖货给B）

**利润位置**: 记录在**投资公司自身**的净利润中（⚠️ 这是难点）

**理解误区**：
❌ 错误理解："要调整联营企业B的净利润"
✅ 正确理解："要调整最终分享的投资收益金额"

**核心逻辑**：
> "我们并不是为了调整B的净利润，我们调整的是分享经营成果的结果。"

```
假设：
- A投资了B 20%股权
- A卖货给B，总利润500万（未实现）
- B公司净利润: 3000万

为什么也要剔除？
因为A和20%的B是一体的，500万中的20% = 100万
是"自己关起门来自己玩"，不承认！

最终Equity Income = 3000万×20% - 500万×20%
                 = 600万 - 100万
                 = 500万
```

### 6️⃣ 未实现损益的计算方法

**方法一：总利润 × 未实现比例**（推荐⭐）

```
例子：A卖货给B
- 销售收入: 16000
- 销售成本: 9600
- 总利润: 16000 - 9600 = 6400
- B对外卖出: 12000
- 未售出: 16000 - 12000 = 4000

未实现比例 = 4000 / 16000 = 25%
未实现损益 = 总利润 6400 × 25% = 1600
```

**方法二：未售出金额 × 利润率**

```
利润率 = 总利润 / 销售收入
       = 6400 / 16000 = 40%

未实现损益 = 未售出金额 4000 × 利润率 40% = 1600
```

**老师建议**: 用第一种方法更直观！

## 📝 核心公式

### 最终调整公式（最重要！）

```
无论顺流还是逆流，最终公式相同：

Equity Income =
    被投资企业净利润 × 持股比例
  - 未实现内部交易损益 × 持股比例
```

### 投资账面价值计算

```
期末投资账面价值 =
    初始投资成本
  + 当年确认的Equity Income
  - 分得的股利
```

### 需要调整的净利润

```
调整后净利润 =
    账面净利润
  - 公允价值增值导致的额外折旧
```

## 📊 例题精讲

### 例题1：逆流交易（Upstream）

**题目条件**：
- 2018年1月1日，W公司出资100万收购F公司**25%**股权
- F公司净资产账面价值: 380万
- 办公楼公允价值被低估: 4万（剩余寿命20年，直线折旧）
- F公司2018年支付股利: 3200
- F公司2018年账面净利润: 2万
- 逆流交易（F卖货给W）：总利润8000，**全部未售出**

**问题1**：计算2018年权益法投资收益（Equity Income）

**解题步骤**：

**第一步：公允价值调整净利润**
```
办公楼增值4万，20年折旧
每年额外折旧 = 40000 / 20 = 2000

调整后净利润 = 20000 - 2000 = 18000
```

**第二步：计算投资收益（剔除未实现损益）**
```
Equity Income = 18000 × 25% - 8000 × 25%
             = 4500 - 2000
             = 2500
```

**答案**: 2018年权益法投资收益为**2500**

---

**问题2**：计算2018年12月31日投资账面价值

**解题步骤**：

```
期末投资账面价值 =
    初始投资成本 1000000
  + 当年投资收益 2500
  - 分得股利 (3200 × 25% = 800)
  = 1001700
```

**答案**: 期末投资账面价值为**1,001,700**

## 📌 考试重点

### 高频考点

1. **⭐⭐⭐⭐⭐ 顺流逆流交易的区分**
   - 给出交易方向，判断是upstream还是downstream
   - 记忆方法：看英文箭头方向

2. **⭐⭐⭐⭐⭐ 未实现损益的计算**
   - 给出销售价格、成本、已售出比例
   - 计算未实现损益金额

3. **⭐⭐⭐⭐⭐ 投资收益的计算**
   - 综合考察：公允价值调整 + 未实现损益剔除
   - 这是最常见的考法

4. **⭐⭐⭐⭐ 期末投资账面价值**
   - 记住公式：成本 + 收益 - 股利

### 常见陷阱

❌ **陷阱1**: 顺流交易中，误以为不需要调整
- 实际上：无论顺流逆流，都要剔除未实现损益

❌ **陷阱2**: 剔除的是总利润，而不是未实现利润
- 实际上：只剔除未实现部分

❌ **陷阱3**: 忘记乘以持股比例
- 实际上：只剔除投资方对应的比例部分

❌ **陷阱4**: 顺流交易误以为要调整联营企业净利润
- 实际上：调整的是最终分享的投资收益
