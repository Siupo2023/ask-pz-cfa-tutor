#### 录音总结
本次课程围绕假设公司“Stings Deli”的资产负债表和利润表，详细讲解了企业自由现金流（FCFF）和股权自由现金流（FCFE）的计算方法，包括各关键组件的获取与验证。

**FCFF与FCFE计算准备**
* **数据来源与关键输入**：明确计算FCFF和FCFE需使用资产负债表（两年数据，含实际与预测）、利润表。关键输入包括折旧、利息费用、税率、净收入、营运资本投资、固定资本投资和净借款。
* **CFO的构成与验证**：CFO（经营活动现金流）计算公式为“净收入 + 非现金费用（此处仅为折旧） - 营运资本投资”。已知CFO为81，营运资本投资为25，后续将通过资产负债表验证该营运资本投资数值。

**固定资本投资计算**
* **计算方法与步骤**：固定资本投资通过“期末净PP&E - 期初净PP&E + 折旧”得出。示例中期末净PP&E为210，期初为160，折旧为50（来自利润表），因无固定资产出售（利润表未提及处置损益），故固定资本投资 = 210 - 160 + 50 = 100。

**营运资本投资计算**
* **界定与计算**：营运资本投资为“期末营运资本 - 期初营运资本”。营运资本 = 经营性流动资产（应收账款 + 存货，不含现金） - 经营性流动负债（此处仅为应付账款）。示例中期末营运资本为50，期初为25，故营运资本投资 = 50 - 25 = 25，与已知数据一致。

**净借款计算**
* **界定与计算**：净借款为有息负债的变化值。示例中，期初有息负债为110，期末为134，故净借款 = 134 - 110 = 24。

**FCFF计算**
* **五部分公式应用**：FCFF = 净收入 + 非现金费用（折旧） + 税后利息 - 营运资本投资 - 固定资本投资。代入数据：净收入56 + 折旧50 + 税后利息（15*(1-30%)=10.5） - 营运资本投资25 - 固定资本投资100 = -8.5。
* **CFO验证法**：FCFF也可通过“CFO + 税后利息 - 固定资本投资”计算，即81 + 10.5 - 100 = -8.5，结果一致。

**FCFE计算**
* **五部分公式应用**：FCFE = 净收入 + 非现金费用（折旧） - 营运资本投资 - 固定资本投资 + 净借款。代入数据：56 + 50 - 25 - 100 + 24 = 5。
* **CFO验证法**：FCFE也可通过“CFO - 固定资本投资 + 净借款”计算，即81 - 100 + 24 = 5，结果一致。
* **FCFF推导法**：FCFE = FCFF - 税后利息 + 净借款，即-8.5 - 10.5 + 24 = 5，结果一致。

**其他起点计算FCFF/FCFE**
* **以EBIT为起点**：FCFF = EBIT*(1-税率) + 折旧 - 营运资本投资 - 固定资本投资。示例中EBIT为95，税率30%，故95*(1-30%) + 50 - 25 - 100 = -8.5，与之前结果一致。
* **以EBITDA为起点**：FCFF = EBITDA*(1-税率) + 折旧*税率 - 营运资本投资 - 固定资本投资。EBITDA为145（EBIT 95 + 折旧50），故145*(1-30%) + 50*30% - 25 - 100 = -8.5，结果一致。

说明FCFE的五部分公式：前四部分与FCFF相同（净收入 + 非现金费用 - 营运资本投资 - 固定资本投资），第五部分为加回净借款。

[00:07:25](https://getnotes.seek:445) **其他起点计算可能性**

指出无论从何种起点（如EBIT、EBITDA）计算，若公式正确，结果应一致。

[00:07:40](https://getnotes.seek:460) **以EBIT为起点计算FCFF**

说明以EBIT为起点计算FCFF的方法：EBIT*(1-税率) + 折旧 - 营运资本投资 - 固定资本投资。示例中EBIT 95，税率30%，代入得95*(1-30%) + 50 - 25 - 100 = -8.5，结果一致。

[00:08:06](https://getnotes.seek:486) **EBITDA的计算**

指出EBITDA = EBIT + 折旧，示例中EBIT 95 + 折旧50 = 145。

[00:08:30](https://getnotes.seek:510) **以EBITDA为起点计算FCFF**

说明以EBITDA为起点计算FCFF需考虑折旧税盾：EBITDA*(1-税率) + 折旧*税率 - 营运资本投资 - 固定资本投资。代入数据145*(1-30%) + 50*30% - 25 - 100 = -8.5，结果一致。

[00:09:00](https://getnotes.seek:540) **EBIT与FCFF公式的联系**

解释EBIT*(1-税率)等价于FCFF公式中的“净收入 + 利息*(1-税率)”，并再次确认以EBITDA为起点计算FCFF的正确性。
