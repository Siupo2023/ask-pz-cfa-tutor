#### 知识精讲

本课程围绕极大似然估计框架下的模型检验方法展开，核心讲解似然比检验（LRT）的原理、步骤及与OLS中F检验的对比，并介绍伪R²作为逻辑回归拟合优度的补充衡量指标。

**似然比检验（LRT）的基本原理**
* **核心目标**：检验多个自变量是否对模型有显著解释力，或评估整体模型的拟合优度。
* **模型设定**：
  - **无限制模型（Unrestricted Model）**：包含所有自变量的完整模型，如 $\ln\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 X_3 + \epsilon$
  - **限制模型（Restricted Model）**：施加系数限制（如 $\beta_2 = \beta_3 = 0$）后的简化模型，如 $\ln\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1 X_1 + \epsilon$
* **假设提出**：
  - 原假设 $H_0$：被限制系数均为0（如 $\beta_2 = \beta_3 = 0$）
  - 备择假设 $H_1$：至少一个被限制系数不为0
* **🚩重点**：似然比检验统计量的计算与分布
  - *公式*：$LR = -2 \times (\text{log likelihood}_{\text{restricted}} - \text{log likelihood}_{\text{unrestricted}})$
  - *补充说明*：由于无限制模型的似然值更大，括号内结果为负，乘以-2后LR为正数；LR越大表明两模型拟合优度差异越大，越倾向于拒绝原假设。
* **分布与决策规则**：
  - LR统计量服从自由度为限制条件数（Q）的卡方分布
  - 当 $LR > \chi^2_{\alpha, Q}$ 时，拒绝原假设，认为被限制自变量显著

**整体模型显著性检验**
* **特殊限制模型**：仅含截距项的模型 $\ln\left(\frac{p}{1-p}\right) = \beta_0 + \epsilon$，其对数似然值记为 $\text{LL}_{\text{null}}$
* **检验统计量**：$LR = -2 \times (\text{LL}_{\text{null}} - \text{LL}_{\text{unrestricted}})$，自由度为自变量个数
* **🚩重点**：该检验用于判断整个模型是否显著优于仅含截距的基准模型，**考试必考**

**OLS的F检验与LRT的对比**
* **共同点**：均通过对比两模型拟合优度判断自变量显著性
* **差异点**：
  - OLS使用残差平方和（SSE）衡量拟合优度，LRT使用对数似然值（log likelihood）
  - OLS的F统计量为 $\frac{(SSE_r - SSE_u)/Q}{SSE_u/(n-k-1)}$，LRT直接计算似然值之差的-2倍
* **🚩重点**：两种方法的理论基础不同（最小二乘vs极大似然），但检验逻辑一致

**伪R²（Pseudo R²）**
* **定义**：模仿OLS的R²构造，用于衡量逻辑回归模型的拟合优度
* **计算公式**：$Pseudo\ R^2 = 1 - \frac{-2 \times \text{log likelihood}_{\text{unrestricted}}}{-2 \times \text{LL}_{\text{null}}} = 1 - \frac{\text{Deviance}_{\text{unrestricted}}}{\text{Deviance}_{\text{null}}}$
* **特点**：取值范围[0,1]，越接近1表明模型拟合越好，但仅适用于相同样本的模型对比

[00:05:35](https://getnotes.seek:335) **对数似然值的性质**

对数似然值恒为负数，其值越接近0表明模型拟合越好。通过推导公式 $\text{log likelihood} = \sum [Y_i \ln p_i + (1-Y_i) \ln (1-p_i)]$ 说明其构成，强调统计软件会直接输出该值，无需手动计算。

[00:09:05](https://getnotes.seek:545) **似然比统计量的构造**

详细解释LR统计量公式 $LR = -2 \times (\text{LL}_r - \text{LL}_u)$ 的由来：因无限制模型似然值更大（$\text{LL}_u > \text{LL}_r$），括号内为负，乘以-2后得到正统计量。LR值越小表明两模型拟合优度差异越小，越支持原假设。

[00:16:08](https://getnotes.seek:968) **LR统计量的分布与决策**

明确LR服从自由度为限制条件数（Q）的卡方分布。决策规则为：若LR > 卡方临界值，则拒绝原假设，认为被限制自变量显著。以检验X2和X3为例，Q=2，需对比LR与 $\chi^2_{\alpha, 2}$。

[00:19:38](https://getnotes.seek:1178) **整体模型的显著性检验**

当限制模型为仅含截距项的基准模型时，LR检验用于判断整个方程的显著性。此时 $\text{LL}_r$ 记为 $\text{LL}_{\text{null}}$，自由度为自变量总个数。**老师强调：该检验是逻辑回归整体显著性的核心方法，考试重点**。

[00:23:18](https://getnotes.seek:1398) **LR与F检验的对比**

对比OLS的F检验与LRT的异同：两者均通过模型拟合优度差异进行检验，但F检验基于残差平方和，LRT基于对数似然值；F统计量为比值形式，LRT为差值形式。**随手记：F检验和LRT都是比较模型拟合优度，但用的指标不同**。

[00:28:42](https://getnotes.seek:1722) **伪R²的概念**

介绍伪R²的构造思想：模仿OLS的R²，定义为 $1 - \frac{\text{Deviance}_u}{\text{Deviance}_{\text{null}}}$，其中偏差（Deviance）为 $-2 \times \text{log likelihood}$。伪R²越接近1表明模型解释力越强，但仅适用于相同样本的模型比较。

#### 💡 核心概念
- **无限制模型（Unrestricted Model）**：包含所有待检验自变量的完整模型，对数似然值较大。
- **限制模型（Restricted Model）**：施加系数为0限制后的简化模型，对数似然值较小。
- **对数似然值（Log Likelihood）**：极大似然估计中衡量模型拟合优度的指标，值越接近0拟合越好。
- **偏差（Deviance）**：定义为 $-2 \times \text{log likelihood}$，类似于OLS中的SSE，值越小拟合越好。
- **似然比检验统计量（LR）**：$LR = -2 \times (\text{LL}_r - \text{LL}_u) \sim \chi^2(Q)$，用于检验模型限制的合理性。

#### ✨ 课堂金句
- “极大似然估计的目标是最大化对数似然值，而OLS的目标是最小化残差平方和，这是两种方法的本质区别。”
- “LR值越小，说明限制模型与无限制模型拟合优度差异越小，越应该接受原假设。”
- “伪R²只是拟合优度的参考指标，不能像OLS的R²那样直接解释为‘解释变异比例’。”

### 🎯 拍图总结
#### 📊 模型设定与假设检验
- **无限制模型（Unrestricted Model）**：包含全部解释变量的回归方程  
  $ \ln\left(\frac{P}{1-P}\right) = b_0 + b_1X_1 + b_2X_2 + b_3X_3 + \varepsilon $  

- **限制模型（Restricted Model）**：施加系数约束后的简化方程  
  $ \ln\left(\frac{P}{1-P}\right) = b_0 + b_1X_1 + \varepsilon $  
  - **约束条件**：$ b_2 = b_3 = 0 $（即排除$ X_2 $和$ X_3 $的影响）

#### 🔍 假设检验方法
1. **F检验（F-test）**  
   - **用途**：检验多个系数是否同时为0，适用于OLS回归模型  
   - **公式**：  
     $$ F = \frac{(SSE_R - SSE_U)/q}{SSE_U/(n - k - 1)} $$  
     - $ SSE_R $：限制模型的残差平方和  
     - $ SSE_U $：无限制模型的残差平方和  
     - $ q $：约束条件个数（自由度）  
     - $ n - k - 1 $：无限制模型的自由度（$ n $为样本量，$ k $为解释变量个数）  
   - **原假设**：所有被约束系数均为0（如$ H_0: b_2 = b_3 = 0 $）  

2. **似然比检验（Likelihood Ratio Test, LR）**  
   - **用途**：基于对数似然函数的模型显著性检验，适用于各类回归模型  
   - **公式**：  
     $ LR = -2\left(\log L_R - \log L_U\right) $  
     - $ \log L_R $：限制模型的对数似然值  
     - $ \log L_U $：无限制模型的对数似然值  
   - **原假设**：约束条件成立（与F检验一致）  

#### ⚠️ 检验规则
- **拒绝原假设**：检验统计量（F值或LR值）大于临界值，或p值小于显著性水平（如0.05）  
- **结论含义**：被检验的解释变量对因变量存在显著影响，不应从模型中排除  

#### 📌 核心结论
- OLS的F检验和似然比检验均用于验证解释变量的整体显著性，但适用场景不同：  
  - F检验适用于线性回归模型，依赖残差平方和计算  
  - LR检验适用于更广泛的模型（如Logistic回归），基于似然函数构建  
- 两者均通过比较限制模型与无限制模型的拟合优度，判断约束条件是否合理。
