#### 知识精讲

本课程聚焦模型设定错误的第二大类——函数形式错误，详细讲解了遗漏变量、变量形式不正确、数据未标准化、不正确数据合并四种类型及其后果，并回顾了模型设定错误的整体框架。

**模型函数形式错误的类型**
* **遗漏变量（Omitted Variables）**：正确模型含多个自变量（如 $Y = B_0 + B_1 X_1 + B_2 X_2 + \epsilon$），错误模型遗漏关键变量（如 $Y = A_0 + A_1 X_1 + \mu$），导致残差项 $\mu$ 包含遗漏变量 $X_2$ 与真实残差 $\epsilon$。
   - **情况一：遗漏变量与剩余变量不相关（$corr(X_2, X_1) = 0$）**
     - 后果1：残差项均值非零（$E(\mu) \neq 0$）
     - 后果2：残差项非独立同分布（依赖于 $X_2$ 的分布）
     - 后果3：剩余变量系数估计无偏（$A_1$ 估计准确）
     - 后果4：截距项估计有偏（$A_0$ 估计错误）
   - **情况二：遗漏变量与剩余变量相关（$corr(X_2, X_1) \neq 0$）**
     - 后果1：所有系数估计错误（$A_0, A_1$ 均有偏且不一致）
     - 后果2：残差项与自变量相关，违反基本假设
     - 后果3：标准误估计不准确（$SE(A_1)$ 不一致）
* **不正确的变量形式**：未将非线性关系转换为线性形式建模。
   - 示例：科布-道格拉斯生产函数 $Y = A X_1^\alpha X_2^\beta$ 应通过对数转换为线性模型 $\ln Y = \ln A + \alpha \ln X_1 + \beta \ln X_2$，而非直接建模 $Y = A_0 + A_1 X_1 + A_2 X_2$。
   - 核心问题：未识别变量间的非线性关系，需依据经济理论进行变量转换。
* **不正确的数据标准化（Incorrect Scaling）**：未对数据进行规模调整导致模型失真。
   - 示例：分析净利润对股票收益率影响时，应使用净利润率（净利润/销售额）而非原始净利润，以消除公司规模差异。
   - 应用场景：财务分析中常用共同比报表（Common Size Statement）进行数据标准化。
* **不正确的数据合并（Incorrect Pooling）**：将存在结构突变（Structural Break）的数据合并建模。
   - 结构突变场景：政策变化、重大事件（如央行成立、次贷危机、新冠疫情）导致数据生成机制改变。
   - 识别方法：通过散点图观察数据是否形成不同簇（Cluster），均值或波动率存在显著差异。
   - 解决策略：选择最能代表预测期条件的子样本建模。

**模型设定错误整体框架回顾**
* **第一大类：假设违反（Violations of Assumptions）**
   - 异方差（Heteroskedasticity）
   - 序列相关（Serial Correlation）
   - 多重共线性（Multicollinearity）
   - 核心考点：后果（Consequence）、检验（Detection）、修正（Correction）
* **第二大类：函数形式错误（Misspecified Functional Form）**
   - 遗漏变量、变量形式错误、数据未标准化、不正确数据合并

[00:02:06](https://getnotes.seek:126) **遗漏变量（Omitted Variables）定义与案例**
正确模型含两个自变量：$Y = B_0 + B_1 X_1 + B_2 X_2 + \epsilon$；错误模型遗漏 $X_2$：$Y = A_0 + A_1 X_1 + \mu$，此时残差项 $\mu = B_2 X_2 + \epsilon$。

[00:03:02](https://getnotes.seek:182) **遗漏变量后果（情况一：$X_2$ 与 $X_1$ 不相关）**
当 $corr(X_2, X_1) = 0$ 时：残差均值非零（因 $X_2$ 均值非零）、残差非独立同分布（依赖 $X_2$ 分布）、$X_1$ 系数估计无偏（因 $\mu$ 与 $X_1$ 不相关）、截距项估计有偏。

[00:07:22](https://getnotes.seek:442) **遗漏变量后果（情况二：$X_2$ 与 $X_1$ 相关）**
当 $corr(X_2, X_1) \neq 0$ 时：残差与 $X_1$ 相关（违反基本假设），导致 $A_0$ 和 $A_1$ 估计均有偏，标准误不准确。需注意：相关不等于多重共线性（后者指高度相关）。

[00:11:16](https://getnotes.seek:676) **不正确的变量形式**
指未将非线性关系转换为线性模型。例如科布-道格拉斯生产函数 $Y = A X_1^\alpha X_2^\beta$ 需通过对数转换为 $\ln Y = \ln A + \alpha \ln X_1 + \beta \ln X_2$，直接建模 $Y = A_0 + A_1 X_1 + A_2 X_2$ 会导致形式错误。

[00:14:26](https://getnotes.seek:866) **不正确的数据标准化（Incorrect Scaling）**
建模前未对数据进行规模调整。例如分析净利润对股票收益率影响时，应使用净利润率（净利润/销售额）而非原始净利润，以消除公司规模差异（如小公司500万与茅台298亿利润不可直接比较）。

[00:16:32](https://getnotes.seek:992) **不正确的数据合并（Incorrect Pooling）**
将存在结构突变的数据合并建模。例如使用某国80年数据（含中央银行成立前后）分析利率与股票收益率关系，因市场结构改变，合并数据会导致系数估计错误。可通过散点图识别数据簇（Cluster）差异。

[00:25:37](https://getnotes.seek:1537) **模型设定错误整体回顾**
第一大类“假设违反”包括异方差（BP检验）、序列相关（BG检验）、多重共线性（VIF指标，VIF>10为严重）；第二大类“函数形式错误”包括遗漏变量、变量形式错误、数据未标准化、不正确数据合并。

#### 💡 核心概念
- **遗漏变量**：模型中未包含对因变量有显著影响的自变量，导致残差项包含遗漏变量信息
- **结构突变（Structural Break）**：因政策、事件等导致数据生成机制发生改变的现象
- **VIF（方差膨胀因子）**：衡量多重共线性的指标，$VIF = \frac{1}{1 - R^2_j}$，其中 $R^2_j$ 为自变量 $X_j$ 对其他自变量的回归 $R^2$
- **科布-道格拉斯生产函数对数转换**：$\ln Y = \ln A + \alpha \ln X_1 + \beta \ln X_2$

#### ✨ 课堂金句
- “遗漏变量与剩余变量相关时，问题更大，因为它违反了残差与自变量不相关的核心假设。”
- “建模前处理数据是经验，用标准化数据（如净利润率）比原始数据更有可比意义。”
- “结构突变后的数据性质不同，合并建模就像把苹果和橘子放在一起称重。”
