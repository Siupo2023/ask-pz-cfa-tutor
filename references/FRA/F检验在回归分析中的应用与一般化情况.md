#### 知识精讲

本课程系统讲解了F检验在回归分析中的两种核心应用：整体方程显著性检验与部分系数联合检验，重点阐述了受限模型与非受限模型的对比分析及F统计量的计算推导。

**F检验的基本概念与分类**
* **联合检验的定义**：F检验用于判断多个自变量系数是否同时为零，包括检验整个方程显著性和部分系数显著性两种类型。
   - 🚩重点：F检验是**联合检验**，可同时检验多个系数是否为零，区别于T检验的单个系数检验。
* **两种F检验类型**：
  1. **整体方程检验**：检验所有自变量系数是否均为零（一级内容回顾）
  2. **部分系数检验**：检验部分自变量系数是否同时为零（二级扩展内容）

**受限模型与非受限模型**
* **非受限模型（Unrestricted Model）**：包含所有自变量的完整模型，如Fama-French五因子模型。
* **受限模型（Restricted Model）**：对部分系数施加限制（通常设为零）的简化模型，如Fama-French三因子模型。
   - 🚩重点：受限模型是**嵌套（nested）** 在非受限模型中的子模型，通过对比两者拟合优度判断限制是否合理。
   - *补充说明*：以Fama-French模型为例，五因子模型（含Market、SMB、HML、RMW、CMA）为非受限模型，三因子模型（仅含前三个因子）为受限模型，限制条件为RMW和CMA的系数为零。

**联合F检验的步骤**
* **第一步：提出假设**
  - 原假设 \( H_0: \beta_m = \beta_{m+1} = ... = \beta_{m+q-1} = 0 \)（q个系数同时为零）
  - 备择假设 \( H_1 \): 至少有一个系数不为零
   - 🚩重点：Q代表**限制条件的数量**，即被检验系数的个数。
* **第二步：计算F统计量**
  - *公式1（基于SSE）*：$$ F = \frac{(SSE_R - SSE_U)/Q}{SSE_U/(n - k - 1)} $$
    - \( SSE_R \)：受限模型残差平方和
    - \( SSE_U \)：非受限模型残差平方和
    - Q：限制条件数量（分子自由度）
    - \( n - k - 1 \)：非受限模型自由度（分母自由度，k为自变量总数）
  - *公式2（基于R²）*：$$ F = \frac{(R_U^2 - R_R^2)/Q}{(1 - R_U^2)/(n - k - 1)} $$
    - \( R_U^2 \)：非受限模型决定系数
    - \( R_R^2 \)：受限模型决定系数
   - 🚩重点：F统计量**必须掌握计算**，考试会提供ANOVA表数据，需正确选择SSE或R²代入公式。
* **第三步：确定拒绝域**
  - F分布为右偏分布，拒绝域在右侧，单尾检验
  - 临界值需根据显著性水平α、分子自由度Q、分母自由度\( n - k - 1 \)查表获取
* **第四步：做出决策**
  - 若计算的F统计量 > 临界值，拒绝原假设，认为被检验系数整体显著不为零
  - 若计算的F统计量 ≤ 临界值，不拒绝原假设，认为被检验系数整体为零

**两种F检验的统一性**
* **整体方程检验**是联合检验的特例（Q=k，检验所有自变量系数为零）
  - 受限模型仅含截距项，其SSE等于总平方和SST
  - F统计量公式可变形为：$$ F = \frac{SSR/k}{SSE/(n - k - 1)} $$（与一级内容一致）
   - 🚩重点：整体方程检验的F统计量本质是联合检验当Q=k（所有自变量）时的特殊情况。

**假设检验方法总结**
* **T检验**：检验单个系数是否为零（\( H_0: \beta_i = 0 \)）
* **联合F检验**：检验多个系数是否同时为零（\( H_0: \beta_m = ... = \beta_{m+q-1} = 0 \)）
* **整体F检验**：检验所有系数是否同时为零（\( H_0: \beta_1 = ... = \beta_k = 0 \)）

非受限模型（Unrestricted Model）是包含所有自变量的完整模型，受限模型（Restricted Model）是对部分系数施加限制（设为零）的简化模型。例如五因子模型为非受限模型，三因子模型为受限模型（限制\( \beta_4 = \beta_5 = 0 \)）。

[00:06:14](https://getnotes.seek:374) **联合检验的假设设定**
一般化假设设定：原假设\( H_0: \beta_m = \beta_{m+1} = ... = \beta_{m+q-1} = 0 \)（Q个系数为零），备择假设为至少有一个系数不为零。Q代表限制条件数量，即被检验的系数个数。

[00:08:08](https://getnotes.seek:488) **F统计量的推导（基于SSE）**
F统计量衡量受限与非受限模型的拟合优度差异，公式为\( F = \frac{(SSE_R - SSE_U)/Q}{SSE_U/(n - k - 1)} \)。分子反映加入限制后残差平方和的增加量，分母为非受限模型的残差均方。

[00:16:15](https://getnotes.seek:975) **F统计量的R²表示形式**
利用\( SSE = (1 - R^2)SST \)，可推导出基于决定系数的F统计量公式：\( F = \frac{(R_U^2 - R_R^2)/Q}{(1 - R_U^2)/(n - k - 1)} \)，分子为R²的增加量，分母为非受限模型未解释的变异比例。

[00:23:23](https://getnotes.seek:1403) **F检验的分布与拒绝域**
F分布为右偏分布，拒绝域在右侧（单尾检验）。当F统计量大于临界值时，拒绝原假设，表明被检验系数整体显著。

[00:28:52](https://getnotes.seek:1732) **两种F检验的统一性**
整体方程检验是联合检验的特例（Q=k），此时受限模型仅含截距项，其SSE=SST，F统计量可变形为\( F = \frac{SSR/k}{SSE/(n - k - 1)} \)，与一级学过的公式一致。

- [00:25:03](https://getnotes.seek:1503) **决策规则**：F统计量越大，表明模型拟合优度差异越显著，越可能拒绝原假设（被检验系数显著）。

#### 💡 核心概念
- **联合检验**：同时检验多个自变量系数是否为零的统计方法，使用F分布。
- **受限模型**：对部分系数施加限制（通常设为零）的简化模型，SSE较大，R²较小。
- **非受限模型**：包含所有自变量的完整模型，SSE较小，R²较大。
- **F统计量（SSE形式）**：$$ F = \frac{(SSE_R - SSE_U)/Q}{SSE_U/(n - k - 1)} $$
- **F统计量（R²形式）**：$$ F = \frac{(R_U^2 - R_R^2)/Q}{(1 - R_U^2)/(n - k - 1)} $$

#### ✨ 课堂金句
