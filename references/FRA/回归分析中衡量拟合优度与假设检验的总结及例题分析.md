#### 知识精讲

本课程主要围绕回归模型的拟合优度衡量指标、假设检验方法（T检验与F检验）及其在模型选择中的应用展开，并通过例题详细演示了F检验的具体步骤和模型比较方法。

**模型拟合优度与假设检验总结**
* **直接衡量拟合优度的指标**：
   - 🚩重点：考试主要考这一页内容，所有提到的指标都要掌握。
   - *指标及判断标准*：Adjusted R-squared（越大越好）、AIC（越小越好）、BIC（越小越好）。
* **假设检验方法**：
   - 🚩重点：T检验和F检验的决策标准及结论意义。
   - *T检验*：检验单个系数显著性。T统计量若落在critical value之外（尾巴区域），则拒绝原假设（$H_0: \beta_j = 0$），说明该系数对应的自变量X显著。
   - *F检验*：联合检验多个系数。F统计量若超过critical value，则拒绝原假设（$H_0: \beta_{j1} = \beta_{j2} = ... = \beta_{jk} = 0$），说明至少有一个系数不等于0，对应的自变量联合显著。
* **模型选择的迭代过程**：找到最优模型是一个迭代过程，需根据拟合优度调整模型、重新估计、检验，循环往复。（了解即可）

**例题：公司ROA影响因素分析**
* **研究背景**：针对26家公司，考察ROA（Y）受资本支出（X1）、广告费用（X2）、研发费用（X3）的影响。提供了三个模型：模型1（仅X1）、模型2（X1,X2）、模型3（X1,X2,X3）的ANOVA表结果。
   - **单美元符号示例**：行内公式$Y = f(X1, X2, X3)$
* **第一问：模型3整体显著性检验（1% significance level）**
   - 🚩重点：整体模型显著性检验使用F检验。
   - *F统计量计算*：$F = \frac{MSR}{MSE} = \frac{136.9864}{2.5180} = 54.4$
   - *决策过程*：在1%显著性水平下，F临界值为4.8166。计算的F统计量54.4 > 4.8166，落在拒绝域，拒绝原假设$H_0: \beta_1 = \beta_2 = \beta_3 = 0$，结论：模型3整体对Y有显著解释能力。
* **第二问：广告费用与研发费用联合显著性检验（B2和B3是否为0）**
   - 🚩重点：F检验中Restricted Model和Unrestricted Model的选择是关键，不要选错。
   - *假设设定*：$H_0: \beta_2 = \beta_3 = 0$；$H_1: \beta_2$和$\beta_3$至少有一个不为0。
   - *F统计量计算*：
     $$F = \frac{\frac{SSE_r - SSE_u}{q}}{\frac{SSE_u}{n - k - 1}} = \frac{\frac{55.9934 - 55.3949}{2}}{\frac{55.3949}{22}} = 0.1118$$
   - *决策过程*：在1%显著性水平下，分子自由度2、分母自由度22的F临界值为5.7190。计算的F统计量0.1118 < 5.7190，落在接受域，不能拒绝原假设，结论：B2和B3等于0，X2和X3对Y的解释无贡献，模型1（更简洁）优于模型3。
* **第三问：选择最优模型**
   - 🚩重点：模型选择主要依据Adjusted R-squared（越大越好）、AIC（越小越好）、BIC（越小越好），R-squared因自变量越多越大故不看。
   - *结论*：综合指标，仅包含X1的模型（模型1）是最好的模型，因为它有最高的Adjusted R-squared和最低的AIC、BIC。

#### 💡 核心概念
- **Adjusted R-squared**：衡量模型拟合优度的指标，考虑了自变量数量的影响，越大表示模型拟合越好。
- **AIC (Akaike Information Criterion)**：模型选择指标，越小越好，兼顾模型拟合优度和简洁性。
- **BIC (Bayesian Information Criterion)**：类似AIC的模型选择指标，对模型复杂度惩罚更重，越小越好。
- **T检验**：用于检验单个回归系数是否显著不为0，$H_0: \beta_j = 0$，若T统计量落在临界值外则拒绝$H_0$。
- **F检验（整体显著性）**：用于检验所有自变量联合起来对因变量是否有显著影响，$H_0: \beta_1 = \beta_2 = ... = \beta_k = 0$，$F = \frac{MSR}{MSE}$。
- **F检验（联合系数检验）**：用于检验多个特定回归系数是否同时为0，$H_0: \beta_{j1} = ... = \beta_{jm} = 0$，$F = \frac{(SSE_r - SSE_u)/m}{SSE_u/(n - k - 1)}$。
