#### 录音总结
本次课程讲解了多元回归的扩展内容，包括影响分析（高杠杆点与异常值）、定性自变量（虚拟变量）、定性因变量（逻辑回归）及嵌套逻辑模型等关键概念与应用。

**多元回归扩展概述**
* **课程引入**：课程开始回顾了多元回归模型，指出其使用多个自变量（X变量）来预测因变量（Y变量），本次将讨论多元回归模型的一些额外方面，首先是影响分析。

**影响分析**
* **高杠杆点**：当数据集中的X变量（自变量）存在极端值（过高或过低）时，称为高杠杆点。高杠杆点可能导致回归参数（如beta系数）估计不准确，进而影响预测质量。其判断需计算杠杆值（标准化 metric，范围0-1），若超过“3*(k+1)/n”（k为自变量数量，n为样本量）的临界值，则判定为高杠杆点。需评估其是有效数据（如市场动荡时期）还是输入错误（如键盘输入错误）。
* **异常值**：指Y变量（因变量）的极端值。检测异常值需计算学生化残差，其服从t分布。计算过程为：删除一个观测值，用剩余n-1个观测值估计回归模型，再用该模型预测被删除观测值的Y值，与实际Y值的差即为残差，经标准化后得到学生化残差。判断标准为：若学生化残差的绝对值超过自由度为n-k-2（因删除一个观测值）的t分布临界值（如5%显著性水平），则为异常值。
* **异常值检测示例**：以预测IPO三日回报率（3个自变量，样本量24）为例，自由度为24-3-2=19，t分布临界值为2.09。学生化残差绝对值超过2.09的观测值（如第三、二十三、十九个观测值）被判定为异常值。

**定性自变量（虚拟变量）**
* **虚拟变量定义**：用于表示定性特征，数值（通常0或1）无实际意义，仅作分类标记（如1表示“是”，0表示“否”）。例如，用虚拟变量区分新兴市场（0）和发达市场（1）。
* **虚拟变量数量规则**：对于有n个类别的定性变量，应使用n-1个虚拟变量，以避免多重共线性。
* **虚拟变量作用**：用于捕捉不同类别数据中Y与X关系的差异，可改变回归模型的截距、斜率或两者。
* **虚拟变量模型类型**：包括截距虚拟变量（仅改变截距）、斜率虚拟变量（通过交互项改变斜率）、同时改变截距和斜率的模型。解释时需将0或1代入模型进行计算。

**定性因变量（逻辑回归）**
* **逻辑回归适用场景**：当因变量为定性变量（如违约与否，1表示违约，0表示不违约）时，无法使用标准线性回归，需采用逻辑回归模型。
* **逻辑回归原理**：估计事件发生的对数几率（log(odds)，其中odds = p/(1-p)，p为事件发生概率），其模型形式与线性回归类似（右侧为线性组合）。通过log(odds)可反推概率：p = 1 / (1 + e^(-y_hat))，其中y_hat为模型预测的log(odds)。
* **系数解释**：截距项为所有X变量为0时的log(odds)；斜率系数因logistic函数的非线性特性，需在X变量均值处通过改变一个X变量单位并观察概率变化来解释其影响。
* **模型评估**：使用伪R平方（Pseudo R Square），其值越高表示模型拟合越好，但仅适用于比较预测同一因变量的不同模型。
* **逻辑回归示例**：以预测股息增加可能性（1表示增加，0表示不增加）为例，自变量包括上次股息增加至今的年数（Age）、自由现金流与市值比（FCFE）、现金与总资产比（Cash）。通过模型计算，在所有X变量为0时，股息增加概率为3.09%；在X变量取均值时，概率为21.42%；FCFE增加1单位（从0.08到0.09）时，概率增加2.15%。P值用于判断系数显著性（如截距项P值0.002，在1%水平显著；FCFE和Cash在5%水平显著，Age不显著）。

**嵌套逻辑模型**
* **模型定义**：类似于嵌套多元回归，通过从无限制模型（含较多自变量）中移除一个或多个自变量得到限制模型。
* **模型比较**：使用对数似然值（Log Likelihood，值越高拟合越好），计算统计量LR = -2*(LL_restricted - LL_unrestricted)，该统计量服从自由度为被移除自变量数量（Q）的卡方分布。若LR统计量超过临界卡方值，则拒绝原假设（被移除变量无解释力），认为限制模型拟合更差。

开始讲解多元回归的扩展内容，回顾多元回归模型使用多个自变量（X变量）预测因变量（Y变量），本次将讨论影响分析等额外方面。

[00:00:42](https://getnotes.seek:42) **影响分析：高杠杆点概念**

指出数据集中X或Y的极端值可能导致回归参数估计不准确，影响预测质量。定义高杠杆点为X变量（自变量）的极端值（过高或过低）。

[00:01:35](https://getnotes.seek:95) **高杠杆点的影响与判断**

说明单个X变量的高杠杆点可能影响所有回归参数，而非仅对应系数。判断高杠杆点需计算杠杆值（标准化 metric，0-1），临界值为“3*(k+1)/n”（k为自变量数，n为样本量）。

[00:02:24](https://getnotes.seek:144) **异常值概念与检测**

异常值指Y变量（因变量）的极端值。检测高杠杆点需计算杠杆值，检测异常值需计算学生化残差，无需掌握具体计算，考试会提供数据。

[00:03:08](https://getnotes.seek:188) **杠杆值特性**

杠杆值是标准化 metric，范围在0到1之间，值越接近1表示杠杆越高。

[00:03:48](https://getnotes.seek:228) **高杠杆点临界值计算示例**

举例说明临界值计算：k=3（自变量数），n=20（样本量），则临界值为3*(3+1)/20=12/20=0.60。需评估高杠杆点是有效数据还是输入错误。

[00:04:49](https://getnotes.seek:289) **异常值检测：学生化残差**

学生化残差计算过程：删除一个观测值，用剩余n-1个观测值估计模型，预测被删除观测值的Y值，与实际Y值的差为残差，经标准化得到学生化残差，考试会提供该值。

[00:07:23](https://getnotes.seek:443) **学生化残差的分布与判断**

学生化残差服从t分布（学生t分布），判断标准为其绝对值是否超过临界t值，采用双尾检验的绝对值形式，自由度为n-k-2（因删除一个观测值）。

[00:08:31](https://getnotes.seek:511) **学生化残差自由度解释**

解释自由度为n-k-2的原因：标准t统计量自由度为n-k-1，因删除一个观测值，故减少1个自由度。

[00:09:00](https://getnotes.seek:540) **异常值检测示例：IPO回报率预测**

以预测IPO三日回报率（3个自变量，样本量24）为例，自由度为24-3-2=19，t分布临界值为2.09。学生化残差绝对值超过2.09的观测值（如第三、二十三、十九个）为异常值。

[00:11:18](https://getnotes.seek:678) **定性自变量：虚拟变量**

介绍定性自变量（X变量）称为虚拟变量，其数值无实际意义，仅作分类标记（如0或1）。

[00:11:56](https://getnotes.seek:716) **虚拟变量示例与规则**

举例：月份（1-12）数值无实际倍数意义。虚拟变量通常用0/1表示，关键规则为使用n-1个虚拟变量（n为类别数），如新兴市场（0）和发达市场（1）用1个虚拟变量。

[00:13:37](https://getnotes.seek:817) **虚拟变量作用**

用于捕捉不同类别数据中Y与X关系的差异，若不同类别（如新兴市场与发达市场）的Y行为不同，需用虚拟变量，可改变模型的截距、斜率或两者，以提高预测准确性。

[00:15:11](https://getnotes.seek:911) **虚拟变量模型解释**

重点在于解释给定虚拟变量模型，而非构建模型。以单一虚拟变量为例，说明如何代入0/1值进行解释。

[00:16:01](https://getnotes.seek:961) **截距虚拟变量**

截距虚拟变量模型中，虚拟变量D=0时对应一类（如新兴市场），D=1时截距项增加beta1（如发达市场），仅改变截距，不改变斜率。

[00:17:25](https://getnotes.seek:1045) **斜率虚拟变量**

斜率虚拟变量通过虚拟变量与X变量的交互项实现，D=0时模型为基础形式，D=1时斜率变为beta1+beta2，改变斜率。

[00:18:10](https://getnotes.seek:1090) **截距与斜率同时改变的虚拟变量模型**

同时包含截距虚拟变量和交互项，可同时改变模型的截距和斜率，代入0/1值即可得到不同类别的模型表达式。

[00:18:57](https://getnotes.seek:1137) **定性因变量与逻辑回归**

当因变量（Y）为定性变量（如违约与否，0/1）时，需使用逻辑回归模型，而非标准线性回归。

[00:19:23](https://getnotes.seek:1163) **逻辑回归原理**

逻辑回归估计事件发生的对数几率（log(odds)），odds = p/(1-p)，p为事件发生概率。模型右侧为X变量的线性组合，通过log(odds)反推p = 1/(1 + e^(-y_hat))。

[00:21:22](https://getnotes.seek:1282) **逻辑回归概率计算**

给定X变量值，代入模型得到y_hat（log(odds)），再用公式p = 1/(1 + e^(-y_hat))计算概率。

[00:22:15](https://getnotes.seek:1335) **逻辑回归系数解释**

截距项为所有X变量为0时的log(odds)；斜率系数因logistic函数非线性，需在X均值处改变一个X单位，观察概率变化来解释。

[00:23:50](https://getnotes.seek:1430) **斜率系数解释方法**

解释斜率系数时，使用X变量的平均值计算y_hat和概率，再改变一个X变量单位，重新计算概率，两者差值即为该X变量对概率的影响。

[00:24:42](https://getnotes.seek:1482) **逻辑回归模型评估：伪R平方**

传统R平方不适用于逻辑回归，使用伪R平方，值越高拟合越好，仅用于比较预测同一Y变量的不同模型。

[00:26:04](https://getnotes.seek:1564) **逻辑回归示例：股息增加可能性预测**

示例模型预测股息增加（1）或不增加（0），自变量为上次股息增加至今年数（Age）、FCFE/市值、现金/总资产。截距项为-3.445，所有X为0时概率为3.09%。

[00:27:30](https://getnotes.seek:1650) **逻辑回归系数显著性**

P值用于判断系数显著性，截距项P=0.002（1%水平显著），FCFE和Cash的P值小于5%（5%水平显著），Age的P=0.412（不显著）。

[00:30:11](https://getnotes.seek:1811) **逻辑回归概率计算示例**

代入X变量平均值计算y_hat=-1.299，对应概率为21.42%；FCFE增加1单位（0.08→0.09）时，概率增加至23.57%，即增加2.15%，说明斜率系数不直接等于概率变化。

[00:33:01](https://getnotes.seek:1981) **嵌套逻辑模型**

嵌套逻辑模型类似嵌套多元回归，无限制模型含较多自变量，限制模型移除Q个自变量。通过对数似然值比较，计算LR = -2*(LL_restricted - LL_unrestricted)，服从自由度为Q的卡方分布。

[00:34:08](https://getnotes.seek:2048) **嵌套逻辑模型比较**

若LR统计量超过临界卡方值，拒绝原假设（被移除变量无解释力），认为限制模型拟合更差。卡方检验为单尾检验。
