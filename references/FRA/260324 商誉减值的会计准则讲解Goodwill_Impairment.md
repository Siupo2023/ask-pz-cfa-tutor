## 📚 核心内容总结

### 一、商誉减值的基础概念

#### 商誉的特点
- **不可辨认资产**：商誉无法脱离企业实体单独存在
- **不能单独产生现金流**：无法独立计算可收回金额
- **减值测试依附实体**：必须依附对应资产组进行测试

#### 减值依附实体（术语差异）
| 准则 | 术语 | 本质 |
|------|------|------|
| **国际准则（IFRS）** | Cash Generating Unit（现金产出单元） | 被并购的子公司整体 |
| **美国准则（US GAAP）** | Reporting Unit（报告单元） | 被并购的子公司整体 |

**核心含义**：两者本质相同，都是将被并购子公司整体作为测试的资产组

---

### 二、权益法 vs 收购法下的商誉减值

#### 权益法下（Equity Method）

**特点**：
- 商誉**不单独列示**，包含在长期股权投资（Investment）项目中
- 减值针对**整个长期股权投资项目**，而非单独针对商誉

**国际准则规则**：
- 将该投资视为一个**资产组**
- 对比：账面价值（Carrying Value）vs 可收回金额（Recoverable Amount）
- 可收回金额 = Max（公允价值 - 处置费用，使用价值/未来现金流现值）
- 若账面价值 > 可收回金额，计提减值

**美国准则规则**：
- 对比：账面价值（Carrying Value）vs 公允价值（Fair Value）
- 若公允价值发生**永久且不可逆的下降**，计提减值

**核心差异**：国际准则对比可收回金额，美国准则对比公允价值

#### 收购法下（Acquisition Method）

**特点**：
- 商誉**单独列示**在合并资产负债表中
- 需要单独针对商誉做减值测试
- 必须依附对应资产组进行测试

---

### 三、国际准则下商誉减值（一步法）

#### 方法特点
- **一步法（Single Step）**：减值测试和计算减值损失使用**同一个公式**
- 减值损失是**整个资产组**的总减值损失
- 需要按照规则进行**分配**

#### 减值测试与计算

**对比对象**：
- 资产组的**账面价值**（包含商誉）
- 资产组的**可收回金额**（Recoverable Amount）

**可收回金额**：
```
可收回金额 = Max（公允价值 - 处置费用，使用价值）
```

**判断逻辑**：
- 若 账面价值 > 可收回金额 → 需要计提减值
- 减值损失 = 账面价值 - 可收回金额

#### 减值损失分配规则

**分配顺序**：
1. **优先分配给商誉**
   - 如果商誉账面价值足够覆盖减值损失 → 分配结束
   - 不需要调整其他资产

2. **剩余损失分配给其他非现金资产**
   - 若商誉全部冲减至零后仍有剩余
   - 剩余部分按照**账面价值比例**分配给资产组内其他非现金资产
   - CFA二级不要求计算具体分配过程

**核心理念**：
> "商誉本身是不可辨认的资产，本来就是最不靠谱的，所以发生减值的时候，它先当炮灰，先冲减商誉。"

#### 实例演示

**基础条件**：
- 现金产出单元账面价值：160 万
- 其中包含商誉：36 万
- 可收回金额：155 万

**计算过程**：
```
总减值损失 = 160 - 155 = 5 万
商誉账面价值 = 36 万 > 5 万
→ 商誉计提 5 万减值
→ 减值后商誉账面价值 = 36 - 5 = 31 万
→ 不需要调整其他资产
```

**第二种情况（可收回金额变为 90 万）**：
```
总减值损失 = 160 - 90 = 70 万
商誉账面价值 = 36 万 < 70 万
→ 商誉全部冲减为零（计提 36 万减值）
→ 剩余 34 万按比例分配给其他非现金资产
→ 具体分配过程CFA二级不要求计算
```

### 四、美国准则下商誉减值（两步法）

#### 方法特点
- **两步法（Two Step）**：减值测试和计算减值损失使用**不同公式**
- 第一步：判断是否需要计提减值
- 第二步：计算商誉本身的减值损失
- **不涉及分配**问题（直接计算商誉减值）

#### 第一步：减值测试（Impairment Test）

**对比对象**：
- 报告单元的**账面价值**（Carrying Value）
- 报告单元的**公允价值**（Fair Value）

**判断逻辑**：
```
若 账面价值 > 公允价值 → 需要计提减值，进入第二步
```

**注意**：
- 两者都是整个报告单元（资产组）的数据
- 美国准则更依赖资本市场，直接使用公允价值

#### 第二步：计算减值损失

**核心公式**：
```
商誉减值损失 = 商誉原账面价值 - 商誉隐含公允价值
```

**商誉隐含公允价值（Implied Fair Value of Goodwill）**：
```
商誉隐含公允价值 = 报告单元整体公允价值 - 子公司净资产公允价值
```

**理解方式**：
- 假设**重新收购一遍**子公司
- 按当前子公司股票的公允价值进行全额收购
- 收购成本超过净资产公允价值的部分 → 就是重新计算的商誉

#### 特殊情况处理

**若商誉隐含公允价值为负（Bargain Acquisition）**：
- 最多仅将商誉账面价值**冲减至零**
- 不需要调整其他资产
- 不需要额外确认收益

#### 实例演示

**基础条件**：
- 报告单元账面价值：160 万
- 其中包含商誉：36 万
- 报告单元整体公允价值：155 万
- 子公司净资产公允价值：147 万

**计算过程**：

**第一步：减值测试**
```
账面价值 160 万 > 公允价值 155 万
→ 需要计提减值
```

**第二步：计算减值损失**
```
商誉隐含公允价值 = 155 - 147 = 8 万
商誉减值损失 = 36 - 8 = 28 万
→ 减值后商誉账面价值 = 8 万
```

**第二种情况（报告单元公允价值变为 90 万）**：
```
商誉隐含公允价值 = 90 - 147 = -57 万
→ 商誉最多计提 36 万减值（冲减至零）
→ 不需要做其他调整
```

### 五、国际准则 vs 美国准则：核心差异

#### 相同点

| 项目 | 国际准则 | 美国准则 |
|------|----------|----------|
| 商誉能否单独减值测试 | ❌ 不能 | ❌ 不能 |
| 减值测试依附实体 | 现金产出单元 | 报告单元 |
| 商誉是否摊销 | ❌ 不摊销 | ❌ 不摊销 |
| 减值测试频率 | 至少每年一次 | 至少每年一次 |
| 减值损失列示 | 合并损益表 | 合并损益表 |

#### 不同点

| 对比维度 | 国际准则（一步法） | 美国准则（两步法） |
|----------|-------------------|-------------------|
| **减值测试对象** | 账面价值 vs 可收回金额 | 账面价值 vs 公允价值 |
| **可收回金额定义** | Max（公允价值-处置费用，使用价值） | 直接使用公允价值 |
| **减值损失范围** | 整个资产组的总损失 | 仅商誉本身的损失 |
| **是否需要分配** | ✅ 需要分配给商誉和其他资产 | ❌ 不需要分配 |
| **最大减值损失** | 可能超过商誉账面价值 | 最多等于商誉账面价值 |

#### 减值损失范围对比

**相同初始条件下**：
- 假设商誉账面价值为 36 万

**国际准则**：
- 可能计提**超过 36 万**的总减值损失
- 超过部分分配给其他非现金资产

**美国准则**：
- 最多计提 **36 万**减值损失
- 仅将商誉冲减至零

**结论**：国际准则可能产生更大的总减值损失

### 六、考试重点总结

#### 必须掌握的内容

1. ✅ **商誉减值的特点**：
   - 不可辨认、不能单独产生现金流
   - 必须依附资产组进行测试

2. ✅ **国际准则（一步法）**：
   - 对比账面价值和可收回金额
   - 减值损失优先分配给商誉
   - 剩余损失分配给其他非现金资产

3. ✅ **美国准则（两步法）**：
   - 第一步：账面价值 vs 公允价值（判断是否减值）
   - 第二步：商誉隐含公允价值计算
   - 不需要分配给其他资产

4. ✅ **准则差异**：
   - 国际准则可能产生更大的总减值损失
   - 美国准则最多将商誉冲减至零

#### 考试不要求掌握的内容

- ❌ 可收回金额的详细计算过程
- ❌ 剩余减值损失在其他非现金资产的具体分配过程
- ❌ 公式的形式（谁减谁）

#### 考试技巧

**关于公式顺序**：
> "你只要能把这个数给它算出来，就可以了，不需要纠结形式上的谁减谁。"

- 减值损失计算的顺序不影响最终结果
- 得到正确的减值金额即可
- CFA二级不考察公式形式

## 🎓 学习建议

### 记忆技巧

1. **国际准则"一步法"**：
   - 想象商誉是"炮灰"
   - 减值时先砍商誉
   - 商誉不够再砍其他资产

2. **美国准则"两步法"**：
   - 第一步：看看要不要减（整体比较）
   - 第二步：算算商誉减多少（假设重新收购）

3. **准则差异记忆**：
   - 国际准则：更保守（可以减更多）
   - 美国准则：更依赖市场（公允价值）

### 理解要点

1. **商誉为什么不能单独减值测试？**
   - 不可辨认资产，无法单独出售
   - 不能单独产生现金流
   - 必须依附实体存在

2. **为什么国际准则会产生更大的减值损失？**
   - 计算的是整个资产组的减值
   - 商誉冲完后还可以调整其他资产
   - 美国准则最多只冲商誉

3. **美国准则为什么要假设重新收购？**
   - 计算商誉当前应该值多少钱
   - 通过"隐含公允价值"概念
   - 避免分配给其他资产
