#### 录音总结
本课程讲解了企业合并中的商誉（部分商誉与完全商誉）、商誉减值及负商誉的会计处理，涵盖定义、计算方法、会计准则差异及实例分析。

**企业合并类型定义**
* **法定合并**：两家公司合并后仅一家公司存续，被合并方资产转移至存续方后停止存在。
* **收购法**：两家公司合并后形成母子公司关系，双方均继续存在。
* **新设合并**：两家公司合并成立全新实体，原公司均停止存在。
* **可变利益实体（VIE）**：通过可变利益而非股权控制的实体，常被称为特殊目的载体，US GAAP对此有专门规定。

**商誉基础概念与计算**
* **商誉的性质**：商誉是不可辨认资产，因其无法与公司分离单独出售，在编制合并报表时，子公司自身报表中的商誉需被排除。
* **商誉计算步骤**：首先确定子公司可辨认净资产的账面价值（排除其自身商誉），然后考虑资产的公允价值调整（如PP&E的增值），最后用收购价格减去调整后的可辨认净资产公允价值得到商誉。
* **示例计算**：Wood公司以6亿美元收购Pine公司100%股权，Pine公司可辨认净资产账面价值4.4亿美元，PP&E公允价值高于账面价值1.2亿美元，调整后可辨认净资产公允价值为5.6亿美元，故商誉为6亿 - 5.6亿 = 0.4亿美元。

**部分商誉与完全商誉**
* **适用场景差异**：当收购100%股权时，两种方法无差异；收购少数股权时才产生区别。
* **会计准则要求**：US GAAP要求使用完全商誉法；IFRS允许选择完全商誉法或部分商誉法。
* **部分商誉计算**：收购价格减去收购方享有的可辨认净资产公允价值份额。例如，以4.5亿美元收购75%股权，可辨认净资产公允价值5.6亿美元，收购方份额为5.6亿×75% = 4.2亿美元，商誉为4.5亿 - 4.2亿 = 0.3亿美元。
* **完全商誉计算**：子公司整体公允价值减去可辨认净资产公允价值。通过收购价格推算子公司整体公允价值（4.5亿 / 75% = 6亿美元），商誉为6亿 - 5.6亿 = 0.4亿美元。
* **转换关系**：完全商誉 = 部分商誉 / 收购股权比例，部分商誉 = 完全商誉 × 收购股权比例。

**商誉减值**
* **减值测试要求**：商誉不摊销，需每年进行减值测试，无论是否存在减值迹象。
* **减值判断标准**：当报告单元（或现金生成单元）的账面价值（含商誉及净资产）大于其公允价值时，需确认减值。
* **IFRS减值测试**：采用一步法，比较现金生成单元的可收回金额（处置净价与使用价值孰高）与账面价值，差额即为减值损失。
* **US GAAP减值测试**：采用两步法，第一步比较报告单元公允价值与账面价值，若前者低则进入第二步；第二步计算隐含商誉（报告单元公允价值 - 可辨认净资产公允价值），原商誉与隐含商誉的差额为减值损失。
* **减值损失处理**：减值损失计入利润表，作为非经常性项目；商誉减值不可转回（US GAAP和IFRS均如此）。
* **超额减值处理**：IFRS下，减值损失超过商誉账面价值时，先冲销商誉至零，剩余损失冲减其他非现金资产；US GAAP下，仅冲销商誉至零，超额部分不处理。

**负商誉**
* **定义**：收购价格低于子公司可辨认净资产公允价值的差额。
* **处理步骤**：首先需重新检查可辨认净资产公允价值的准确性，若确认无误，则将负商誉立即确认为当期收益，计入利润表。

课程开始介绍本模块将学习商誉（部分商誉与完全商誉）及商誉减值，首先简要讲解企业合并、收购、新设合并和可变利益实体的法定定义。

[00:00:21](https://getnotes.seek:21) **法定合并的特点**
指出法定合并中，A公司与B公司合并后仅A公司存续，B公司的净资产转移至A公司后停止存在，认为考试不太可能测试这些定义。

[00:00:57](https://getnotes.seek:57) **收购法与新设合并的区别**
收购法下，两家公司形成母子公司关系；新设合并则是A、B公司合并成立新实体C，原A、B公司均停止存在。

[00:01:17](https://getnotes.seek:77) **可变利益实体的定义**
可变利益实体（VIE）是控制并非基于股权 ownership 的实体，常称为特殊目的载体，US GAAP常提及此概念，将在未来模块详细讨论。

[00:01:50](https://getnotes.seek:110) **商誉计算示例引入**
以Wood公司支付6亿美元收购Pine公司全部流通股为例，开始讲解商誉计算，指出收购100%股权意味着无少数股东权益。

[00:02:10](https://getnotes.seek:130) **子公司资产负债表项目说明**
Pine公司在收购日的资产负债表项目包括流动资产8000万、净PP&E（成本减累计折旧）、自身商誉（因过去投资其他子公司形成）、负债4亿、股东权益4.7亿，强调商誉是不可辨认资产。

[00:02:59](https://getnotes.seek:179) **合并报表中商誉的处理原则**
回顾一级内容，商誉是不可辨认资产因其无法单独分离出售，编制合并报表时需纳入子公司可辨认净资产的公允价值，忽略子公司自身的商誉。

[00:03:42](https://getnotes.seek:222) **公允价值调整对商誉计算的影响**
指出Pine公司PP&E的公允价值高于账面价值1.2亿美元，其他可辨认资产和负债的公允价值等于账面价值，需据此计算商誉。

[00:04:13](https://getnotes.seek:253) **子公司可辨认净资产账面价值计算**
计算Pine公司可辨认净资产账面价值：流动资产8000万 + 净PP&E7.6亿 - 负债4亿 = 4.4亿美元，明确不包含子公司自身商誉（否则为股东权益4.7亿）。

[00:04:32](https://getnotes.seek:272) **可辨认净资产公允价值计算**
考虑PP&E的公允价值调整后，可辨认净资产公允价值为4.4亿 + 1.2亿 = 5.6亿美元，因收购100%股权，需全额纳入。

[00:05:16](https://getnotes.seek:316) **商誉计算结果**
收购价格6亿美元减去可辨认净资产公允价值5.6亿美元，得出合并报表中的商誉金额为0.4亿美元。

[00:06:05](https://getnotes.seek:365) **部分商誉与完全商誉的适用范围**
收购100%股权时两种方法无差异，仅在收购少数股权时存在区别，US GAAP要求完全商誉法，IFRS允许选择。

[00:06:25](https://getnotes.seek:385) **部分商誉的计算方法**
部分商誉等于收购价格减去收购方享有的可辨认净资产公允价值份额，与权益法核算联营企业类似，需忽略子公司自身商誉。

[00:06:59](https://getnotes.seek:419) **少数股东权益的计算（部分商誉法）**
部分商誉法下，少数股东权益等于可辨认净资产公允价值乘以少数股东持股比例。

[00:07:20](https://getnotes.seek:440) **完全商誉的计算方法**
完全商誉等于子公司整体公允价值减去可辨认净资产公允价值，子公司整体公允价值可通过收购价格除以收购股权比例推算。

[00:08:15](https://getnotes.seek:495) **两种商誉方法的核心区别**
部分商誉关注收购价格与收购方份额，完全商誉关注子公司整体公允价值与全部可辨认净资产，100%收购时两者结果一致。

[00:08:50](https://getnotes.seek:530) **少数股东权益的计算（完全商誉法）**
完全商誉法下，少数股东权益等于子公司整体公允价值乘以少数股东持股比例。

[00:09:27](https://getnotes.seek:567) **商誉计算示例（部分与完全）**
假设Wood公司以4.5亿美元收购Pine公司75%股权，可辨认净资产公允价值5.6亿美元。完全商誉法下，子公司整体公允价值为4.5亿 / 75% = 6亿美元，商誉为6亿 - 5.6亿 = 0.4亿美元；部分商誉法下，收购方份额为5.6亿×75% = 4.2亿美元，商誉为4.5亿 - 4.2亿 = 0.3亿美元。

[00:10:28](https://getnotes.seek:628) **两种商誉方法的转换**
部分商誉除以收购股权比例可得到完全商誉（0.3亿 / 75% = 0.4亿），完全商誉乘以收购股权比例可得到部分商誉。

[00:11:41](https://getnotes.seek:701) **商誉差异对财务报表的影响**
完全商誉比部分商誉高0.1亿美元，导致资产负债表资产增加0.1亿美元，为保持平衡，少数股东权益也需增加0.1亿美元（完全商誉法下少数股东权益为6亿×25% = 1.5亿，部分商誉法下为5.6亿×25% = 1.4亿）。

[00:12:45](https://getnotes.seek:765) **商誉减值的基本要求**
商誉不摊销，需每年进行减值测试，无论是否存在减值迹象，减值测试针对包含商誉的报告单元或现金生成单元。

[00:13:57](https://getnotes.seek:837) **减值的判断标准**
当报告单元的账面价值（商誉 + 净资产）大于其公允价值时，需确认商誉减值，减值损失计入利润表，作为非经常性项目。

[00:14:32](https://getnotes.seek:872) **商誉减值的不可逆性**
US GAAP不允许转回任何资产减值（除持有待售资产），IFRS通常允许转回减值，但商誉减值在IFRS下也不可转回，因子公司公允价值难以可靠确定，防止盈余管理。

[00:16:36](https://getnotes.seek:996) **商誉减值的分摊（IFRS）**
IFRS要求将商誉减值分摊至所有从收购中受益的现金生成单元（可能涉及分部报告），影响各单元的分部报表。

[00:17:36](https://getnotes.seek:1056) **商誉减值的分摊（US GAAP）**
US GAAP要求将商誉减值分摊至所有报告单元（经营分部或组成部分），与IFRS理念类似。

[00:18:00](https://getnotes.seek:1080) **IFRS减值测试步骤**
采用一步法，比较现金生成单元的可收回金额（处置净价与使用价值孰高）与账面价值，差额即为减值损失。

[00:18:59](https://getnotes.seek:1139) **可收回金额的构成**
处置净价 = 售价 - 处置费用；使用价值 = 未来现金流量的现值，两者取高者作为可收回金额。

[00:19:32](https://getnotes.seek:1172) **US GAAP减值测试步骤**
采用两步法：第一步比较报告单元公允价值与账面价值，若前者低则进入第二步；第二步计算隐含商誉（报告单元公允价值 - 可辨认净资产公允价值），原商誉与隐含商誉的差额为减值损失。

[00:20:36](https://getnotes.seek:1236) **超额减值损失的处理（IFRS）**
若减值损失超过商誉账面价值，先将商誉冲减至零，剩余损失冲减其他非现金资产（如无形资产、PP&E）。

[00:21:53](https://getnotes.seek:1313) **超额减值损失的处理（US GAAP）**
若减值损失超过商誉账面价值，仅将商誉冲减至零，超额部分不冲减其他资产。

[00:22:18](https://getnotes.seek:1338) **商誉减值示例引入**
母公司去年以100万美元收购子公司，收购日可辨认净资产公允价值80万美元，商誉20万美元。年末子公司公允价值95万美元，净资产公允价值77.5万美元，账面价值（含商誉）98万美元，需分别计算IFRS和US GAAP下的减值损失。

[00:23:13](https://getnotes.seek:1393) **IFRS下减值损失计算**
IFRS下，可收回金额取子公司公允价值95万美元（假设为处置净价与使用价值孰高者），账面价值98万美元，减值损失为98万 - 95万 = 3万美元，商誉从20万减至17万。

[00:24:52](https://getnotes.seek:1492) **US GAAP下减值损失计算**
US GAAP第一步：报告单元公允价值95万低于账面价值98万，存在减值。第二步：隐含商誉 = 95万 - 77.5万 = 17.5万美元，原商誉20万美元，减值损失为20万 - 17.5万 = 2.5万美元，商誉减至17.5万。

[00:25:17](https://getnotes.seek:1517) **负商誉的概念**
负商誉指收购价格低于子公司可辨认净资产公允价值的情况，这种情况非常罕见。

[00:26:36](https://getnotes.seek:1596) **负商誉的处理流程**
首先需重新检查可辨认净资产公允价值的准确性，若确认无误，负商誉立即确认为当期收益，计入利润表。

[00:26:49](https://getnotes.seek:1609) **负商誉的合理性质疑**
正常情况下，出售方应选择清算资产而非低价出售，因此负商誉需谨慎确认，先核实公允价值调整的准确性。

[00:27:08](https://getnotes.seek:1628) **负商誉的会计处理**
US GAAP和IFRS均要求将确认的负商誉作为当期利得计入利润表，因不能在资产负债表中列示负无形资产。
