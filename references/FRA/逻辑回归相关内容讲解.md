#### 录音总结

本次课程围绕逻辑回归（Logistic Regression）展开，详细讲解了其基本原理、函数形式、系数解释、估计方法及边际效应计算，并结合例题进行了应用说明。

**逻辑回归的基本原理与必要性**
* **定性因变量的问题**：当因变量Y为定性变量（如公司是否破产，取值0或1）时，传统线性回归模型无法保证预测结果在0-1之间，因此需采用逻辑回归。
* **逻辑回归的核心**：逻辑回归并非直接回归Y本身，而是回归Y=1的概率P，并通过S型函数（Sigmoid函数）确保P的取值在0-1之间。

**逻辑回归的函数形式与推导**
* **S型函数的引入**：S型函数形式为P = 1 / (1 + e^(-T))，其中T为线性组合（B0 + B1X1 + B2X2 + ... + BnXn），确保P在0-1之间。
* **Logit变换**：通过对P进行Logit变换（ln(P/(1-P))），将其转化为线性形式ln(P/(1-P)) = B0 + B1X1 + B2X2 + ... + BnXn + ε，便于建模和参数估计。

**逻辑回归的系数解释**
* **截距项（B0）**：当所有自变量X均为0时，事件发生的对数几率（log odds），即ln(P/(1-P)) = B0。可通过反推计算此时事件发生的概率P。
* **斜率项（Bi）**：在其他自变量不变的情况下，自变量Xi变动一个单位时，对数几率的变动量。其解释需结合边际效应。

**边际效应（Marginal Effect）**
* **定义**：自变量Xi变动一个单位时，事件发生概率P的变化量，即概率曲线的斜率。
* **计算方法**：通常在所有自变量取均值的情况下，计算Xi变动一个单位前后的概率差值。例如，在公司回购股票的例题中，当现金比率（X1）均值为8.3%，增加1%（变为9.3%）时，回购概率的变化即为X1的边际效应。

**逻辑回归的估计方法**
* **极大似然估计法（MLE）**：由于逻辑回归的残差项服从逻辑分布而非正态分布，无法使用普通最小二乘法（OLS），故采用极大似然估计法估计参数。

**假设检验与拟合优度**
* **单个系数的显著性检验**：与线性回归类似，通过t检验或z检验判断系数是否显著不为0。
* **整体模型拟合优度检验**：重点介绍似然比检验（LRT），需基于极大似然估计原理理解其检验统计量的计算。

**例题应用**
* **公司回购股票概率的估计**：以500家公司的横截面数据为例，构建逻辑回归模型，解释截距项含义（所有X为0时的对数几率），并计算各财务指标（现金比率、负债权益比、净利润率）的边际效应。例如，现金比率增加1%时，回购概率降低0.19%。
