#### 录音总结
本次课程围绕虚构公司“Stings Deli”的财务报表，详细讲解了自由现金流（FCFF和FCFE）的计算方法，包括各关键组件的提取与公式应用，并验证了不同计算起点的一致性。

**自由现金流计算准备**
* **数据来源与关键指标**：明确计算FCFF和FCFE需使用资产负债表（两年数据，含实际与预测）、利润表，关键输入包括折旧、利息费用、税率、净收入、营运资本投资、固定资本投资和净借款。
* **CFO的构成与验证**：CFO（经营活动现金流）计算公式为“净收入 + 非现金费用（此处仅为折旧） - 营运资本投资”。已知CFO为81，营运资本投资为25，后续将通过资产负债表数据验证营运资本投资的正确性。

**固定资本投资计算**
* **计算方法与步骤**：固定资本投资通过“期末净PP&E - 期初净PP&E + 折旧”得出。示例中期末净PP&E为210，期初为160，折旧为50，因此固定资本投资为210 - 160 + 50 = 100。因利润表未提及固定资产出售损益，故仅考虑购买。

**营运资本投资计算**
* **界定与计算**：营运资本投资为“期末营运资本 - 期初营运资本”。营运资本=经营性流动资产（应收账款+存货，不含现金）-经营性流动负债（仅应付账款）。示例中期末营运资本为50，期初为25，故营运资本投资为50 - 25 = 25，与已知数据一致。

**净借款计算**
* **确定方法**：净借款为“期末有息负债 - 期初有息负债”。示例中期末有息负债为134，期初为110，故净借款为134 - 110 = 24。

**FCFF（公司自由现金流）计算**
* **五部分公式应用**：FCFF = 净收入 + 非现金费用（折旧） + 税后利息 - 营运资本投资 - 固定资本投资。代入数据：56（净收入） + 50（折旧） + 10.5（税后利息=15*(1-30%)） - 25（营运资本投资） - 100（固定资本投资） = -8.5。
* **CFO起点验证**：FCFF也可通过“CFO + 税后利息 - 固定资本投资”计算，即81 + 10.5 - 100 = -8.5，结果一致。

**FCFE（股权自由现金流）计算**
* **五部分公式应用**：FCFE = 净收入 + 非现金费用（折旧） - 营运资本投资 - 固定资本投资 + 净借款。代入数据：56 + 50 - 25 - 100 + 24 = 5。
* **CFO起点验证**：FCFE也可通过“CFO - 固定资本投资 + 净借款”计算，即81 - 100 + 24 = 5，结果一致。
* **FCFF转换验证**：FCFE = FCFF - 税后利息 + 净借款，即-8.5 - 10.5 + 24 = 5，结果一致。

**其他起点计算FCFF验证**
* **EBIT起点**：FCFF = EBIT*(1-税率) + 折旧 - 营运资本投资 - 固定资本投资。示例中EBIT为95，代入得95*(1-30%) + 50 - 25 - 100 = -8.5，结果一致。
* **EBITDA起点**：FCFF = EBITDA*(1-税率) + 折旧*税率 - 营运资本投资 - 固定资本投资。示例中EBITDA为145（EBIT 95 + 折旧50），代入得145*(1-30%) + 50*30% - 25 - 100 = -8.5，结果一致。

[00:02:43](https://getnotes.seek:163) **营运资本界定：明确经营性项目**

界定营运资本计算范围：经营性流动资产仅包括应收账款和存货（不含现金），经营性流动负债仅包括应付账款。

[00:02:54](https://getnotes.seek:174) **营运资本投资计算与验证**

计算营运资本投资：期末营运资本（50） - 期初营运资本（25） = 25，与已知数据一致。

[00:03:19](https://getnotes.seek:199) **组件汇总：明确缺失项为净借款**

汇总已确定的组件：营运资本投资（25）、固定资本投资（100）、利息费用、折旧、净收入，指出唯一缺失项为净借款。

[00:03:34](https://getnotes.seek:214) **净借款计算：关注有息负债**

说明净借款需从资产负债表负债端的有息负债计算，即期末有息负债 - 期初有息负债。

[00:03:49](https://getnotes.seek:229) **净借款结果：计算得出24**

计算净借款：期末有息负债（134） - 期初有息负债（110） = 24。

回忆FCFF的五部分基本公式：净收入 + 非现金费用 + 税后利息 - 营运资本投资 - 固定资本投资，准备代入数据计算。

代入数据计算FCFF：56（净收入） + 50（折旧） + 10.5（税后利息=15*(1-30%)） - 25（营运资本投资） - 100（固定资本投资） = -8.5。

[00:05:26](https://getnotes.seek:326) **CFO起点验证FCFF：结果一致**

通过CFO验证FCFF：CFO（81） + 税后利息（10.5） - 固定资本投资（100） = -8.5，结果一致。

[00:05:45](https://getnotes.seek:345) **FCFE计算引入：转向股权自由现金流**

提出计算FCFE的需求，说明FCFE的五部分公式与FCFF前四部分相同，第五部分为净借款。

[00:06:00](https://getnotes.seek:360) **FCFE公式与数据代入**

FCFE公式：净收入 + 非现金费用 - 营运资本投资 - 固定资本投资 + 净借款，代入数据：56 + 50 - 25 - 100 + 24。

[00:06:28](https://getnotes.seek:388) **FCFE结果与CFO起点验证**

计算得出FCFE为5；通过CFO验证：CFO（81） - 固定资本投资（100） + 净借款（24） = 5，结果一致。

[00:07:00](https://getnotes.seek:420) **FCFF转换验证FCFE：结果一致**

通过FCFF转换计算FCFE：FCFF（-8.5） - 税后利息（10.5） + 净借款（24） = 5，结果一致。

[00:07:25](https://getnotes.seek:445) **其他起点计算：EBIT起点**

说明可从EBIT开始计算FCFF，若公式正确结果应一致，EBIT已知为95，税率30%。

[00:07:40](https://getnotes.seek:460) **EBIT起点计算FCFF：结果一致**

应用公式“EBIT*(1-税率) + 折旧 - 营运资本投资 - 固定资本投资”，代入得95*(1-30%) + 50 - 25 - 100 = -8.5，结果一致。

[00:08:06](https://getnotes.seek:486) **EBITDA计算与应用**

指出EBITDA = EBIT + 折旧 = 95 + 50 = 145，可作为FCFF计算的另一起点。

[00:08:30](https://getnotes.seek:510) **EBITDA起点计算FCFF：需加折旧税盾**

说明从EBITDA计算FCFF需“EBITDA*(1-税率) + 折旧*税率 - 营运资本投资 - 固定资本投资”，代入得145*(1-30%) + 50*30% - 25 - 100 = -8.5，结果一致。

[00:09:00](https://getnotes.seek:540) **公式逻辑总结：不同起点的一致性**

总结无论从净收入、CFO、FCFF、EBIT还是EBITDA出发，只要公式正确，计算结果均一致，再次强调EBIT*(1-税率)替代了FCFF公式中的“净收入 + 利息*(1-税率)”。

### ✨ 金句精选

- “资产负债表满足会计恒等式，资产等于负债加所有者权益，会计师已经正确编制。” (方法技巧)
- “固定资本投资 = 期末净PP&E - 期初净PP&E + 折旧，若没有固定资产出售损益，则仅为购买。” (方法技巧)
- “营运资本投资 = 期末营运资本 - 期初营运资本，营运资本=经营性流动资产（不含现金）-经营性流动负债。” (方法技巧)
- “FCFF五部分公式：净收入 + 非现金费用 + 税后利息 - 营运资本投资 - 固定资本投资。” (执行策略)
- “FCFE五部分公式：净收入 + 非现金费用 - 营运资本投资 - 固定资本投资 + 净借款。” (执行策略)
- “CFO = 净收入 + 非现金费用 - 营运资本投资，可作为FCFF和FCFE计算的起点。” (方法技巧)
- “FCFE = FCFF - 税后利息 + 净借款，两种方法计算结果应一致。” (执行策略)
