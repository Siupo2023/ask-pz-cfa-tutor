#### 录音总结

本次课程讲解了多元回归分析的第四步——预测（Forecasting），包括点估计的计算方法、预测时需注意的事项，以及预测标准误的影响因素。

**多元回归分析步骤回顾与预测引入**
* **多元回归前三步回顾**：课程首先回顾了多元回归分析的前三步，分别是建模、衡量拟合优度（涉及Adjusted R squared、AIC、BIC三个指标）和假设检验。假设检验又分为检验单个X（T检验，原假设H0: Bi=0，统计量T=(Bi cap - 0)/SBi cap）、检验多个X（F检验，通过比较无约束模型unrestricted model与约束模型restricted model，F统计量计算公式涉及两者残差平方和之差及自由度调整，自由度为N-K-1）以及检验所有X（F检验的推广，约束模型为仅含截距项，F统计量为回归平方和与残差平方和之比并调整自由度）。
* **预测作为第四步**：明确多元回归的第四步是预测（forecasting），这是回归分析的终点。

**预测的点估计方法与实例**
* **点估计基本公式**：预测的Y等于B0 + B1×X1 + ... + BK×XK，只需将各X值代入估计的回归方程即可求得。
* **具体实例演示**：以Farmer French五因素模型为例，模型估计式为Y = -2.1876 + 1.5992F1 + 0.1923F2 - 0.7126F3 + 3.3376F4 - 2.6832F5。当F1=0.110，F2=0.040，F3=0.080，F4=-0.010，F5=0.001时，代入计算得到预测Y为-2.0971。

**预测时的注意事项**
* **必须包含所有自变量**：即使某个自变量在假设检验中不显著（如B2不显著，不能拒绝原假设），在使用原模型进行预测时，仍需包含所有自变量。因为模型系数是在考虑所有X后统一估计的结果。若要剔除不显著变量，需重新估计模型得到新系数后再进行预测。
* **必须包含截距项**：进行预测时，回归方程中的截距项不可遗漏。

**预测的区间估计与标准误**
* **点估计与区间估计的区别**：点估计是单个预测值，而真实Y值通常在点估计值附近波动，因此需计算置信区间（confidence interval），其宽度与预测标准误（forecast standard error）相关。
* **预测标准误的构成**：预测标准误取决于两个误差来源，一是模型误差（model error），由残差的标准差反映，残差标准差越小，模型拟合越好，模型估计越稳定，预测标准误越小；二是抽样误差（sampling error），与预测的X值（样本外数据）到其均值的距离有关，X越靠近均值，抽样误差越小，预测标准误越小。
* **预测标准误与置信区间的关系**：预测标准误越大，在相同概率下，置信区间范围越宽；反之则越窄。二级考试不要求计算预测标准误的具体公式，但需理解其定性结论。

**预测计算中的数据单位问题**
* **百分数与小数的处理**：在金融数据（如收益率）预测中，代入X值时需注意数据单位。若数据以百分数形式给出（省略百分号），则直接代入数值（如2%代入2）；若以小数形式给出，则代入小数（如2%代入0.02），需根据题目信息判断。

### 📅 章节概要
[00:00:19](https://getnotes.seek:19) **多元回归前三步回顾**

回顾多元回归分析的前三步：第一步建模，第二步衡量拟合优度（涉及Adjusted R squared、AIC、BIC指标），第三步假设检验（可分为检验单个X、多个X及所有X的假设检验）。

[00:00:57](https://getnotes.seek:57) **检验单个X的方法**

检验单个X使用T检验，原假设H0为Bi=0（即单个X对Y无解释力度），T统计量计算公式为Bi cap除以其标准误SBi cap。

[00:01:23](https://getnotes.seek:83) **检验多个X的方法**

检验多个X使用F检验，通过比较无约束模型（unrestricted model）与约束模型（restricted model，如假设B2和B3等于0），F统计量基于两者残差平方和之差并调整自由度（N-K-1）。

[00:01:53](https://getnotes.seek:113) **检验所有X的方法**

检验所有X是检验多个X的推广，约束模型为仅含截距项，F统计量为回归平方和与残差平方和之比并调整自由度，也可直接按公式计算。

[00:02:25](https://getnotes.seek:145) **多元回归第四步：预测（Forecasting）**

明确多元回归的第四步是预测，是回归分析的终点。预测的基本方法是将X值代入回归方程Y = B0 + B1×X1 + ... + BK×XK计算得到预测Y。

[00:03:29](https://getnotes.seek:209) **预测实例：Farmer French五因素模型**

以Farmer French五因素模型为例，给出模型估计式Y = -2.1876 + 1.5992F1 + 0.1923F2 - 0.7126F3 + 3.3376F4 - 2.6832F5，并代入具体F值（F1=0.110，F2=0.040，F3=0.080，F4=-0.010，F5=0.001）计算得到预测Y为-2.0971。

[00:04:13](https://getnotes.seek:253) **预测注意事项一：包含所有自变量**

预测时必须包含模型中所有自变量，即使某个自变量统计上不显著。因为系数是在所有X共同作用下估计的结果，若剔除不显著变量，需重新估计模型系数后再预测。

[00:05:45](https://getnotes.seek:345) **剔除不显著变量的正确做法**

若要剔除不显著变量（如B2），应重新估计不含该变量的模型，得到新的系数估计后再进行预测，而非直接在原模型中剔除该变量的数据。

[00:06:30](https://getnotes.seek:390) **预测注意事项二：包含截距项**

进行预测时，回归方程必须包含截距项，这是基本要求。

[00:07:02](https://getnotes.seek:422) **点估计与区间估计的概念**

一元回归中曾讲过点估计（point estimation）和区间估计。点估计是单个预测值（如预测2021年1月消费为1万元），但真实值通常在其附近，因此需考虑区间估计。

[00:07:46](https://getnotes.seek:466) **点估计的不确定性**

点估计的Y值会因估计模型所用样本数据不同而变化（如用不同时间段数据估计B0和B1，预测同一Y可能得到不同结果），因此需计算预测Y的标准差（标准误）。

[00:08:34](https://getnotes.seek:514) **预测标准误与置信区间**

预测Y的真实值在点估计值左右波动，形成区间，即置信区间。区间宽度取决于预测标准误（Y cap的标准差），其计算与T分布相关（如95%置信度下T统计量约为2.1，区间为点估计±2.1倍标准误）。

[00:09:35](https://getnotes.seek:575) **预测标准误的定义**

预测标准误（forecast standard error）是预测值Y cap的标准差，取决于model error和sampling error。

[00:10:16](https://getnotes.seek:616) **Model error的定义与影响因素**

Model error是模型估计的错误，由残差的标准差反映。残差越紧密围绕回归线（模型拟合好），模型估计越稳定，model error越小，预测标准误越小。

[00:11:43](https://getnotes.seek:703) **Model error与残差标准差的关系**

模型拟合好（残差标准差小）意味着估计稳定，不同样本估计的系数差异小，因此预测标准误（估计量的标准误）也小，两者逻辑一致。

[00:12:28](https://getnotes.seek:748) **Sampling error的定义与影响因素**

Sampling error是预测X时的误差。预测的X（样本外数据，如预测2021年1月收入）越靠近其均值，预测越准确，sampling error越小；反之（如异常值）误差越大。

[00:13:03](https://getnotes.seek:783) **一级中预测标准误的公式（二级不考计算）**

一级中预测标准误公式为残差标准差乘以根号下[1 + 1/N + (预测X与X均值距离的平方)/X离差平方和]，该公式体现了model error（残差标准差）和sampling error（预测X与均值距离）的影响，二级不要求计算此公式。

[00:14:13](https://getnotes.seek:853) **样本外预测的X带来的误差**

预测所用的X是样本外数据（如未来的X），其预测本身存在误差，这会增加Y预测的error，即sampling error。

[00:15:00](https://getnotes.seek:900) **Sampling error与置信区间的关系**

Sampling error越大，预测标准误（SF）越大，置信区间越宽。

[00:15:39](https://getnotes.seek:939) **二级对预测标准误的要求**

二级书上无预测标准误公式，仅定性描述：预测标准误取决于model error（残差标准误反映）和sampling error（X预测错误反映），两者越大，预测标准误越大，置信区间越宽。

[00:16:12](https://getnotes.seek:972) **点估计与区间估计的联系**

根据历史数据得到模型估计后，点估计是单个预测值，真实Y应在其左右，区间范围取决于预测标准误，标准误越大，区间越宽（因预测准确性低）。

[00:16:46](https://getnotes.seek:1006) **Model error对预测标准误的影响**

Model error由残差标准误反映，残差标准差越小，模型拟合越好，估计越稳定，不同样本估计的预测值差异小，预测标准误越小。

[00:17:36](https://getnotes.seek:1056) **二级对预测标准误的定性结论要求**

需知道预测标准误取决于model error和sampling error，标准误越小，置信区间越小；标准误越大，置信区间越宽。

[00:18:19](https://getnotes.seek:1099) **点估计的考试可能性与数据单位注意事项**

点估计可能考计算，只需代入X值即可。金融数据中需注意X的单位（百分数或小数），若数据以百分数形式给出（省略百分号）则代入数值，以小数形式给出则代入小数，需根据题目信息判断。

[00:18:57](https://getnotes.seek:1137) **数据单位实例说明**

例：模型Y=2+0.8X，若X为2%且题目说明数据按百分数代入（in percent），则代入2，Y=2+0.8×2=3.6（即3.6%）；若按小数代入，则代入0.02，Y=2+0.8×0.02，结果为具体数值。

[00:20:24](https://getnotes.seek:1224) **预测计算总结**

点估计计算只需将X代入模型；定性上了解预测标准误的影响因素即可。
