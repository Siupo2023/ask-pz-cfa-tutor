#### 录音总结

本课程讲解了权益法会计的核心概念、应用场景、会计处理方法、影响证据及实例计算，重点包括成本构成、后续计量、收入确认及商誉处理。

**权益法会计的基本概念与应用场景**
* **权益法的定义与别称**：权益法常被称为“单行合并”，体现在母公司资产负债表和利润表中各有一行体现该投资。它适用于对被投资单位具有重大影响的联营企业，以及国际财务报告准则（IFRS）和美国公认会计原则（US GAAP）下的合营企业。
* **适用的所有权比例**：通常以20%至50%的所有权比例作为判断具有重大影响的“明线标准”，但这只是指导原则而非硬性规定。

**权益法下投资的初始计量与成本构成**
* **初始计量基础**：在资产负债表中，权益法投资最初按成本报告。
* **成本的构成要素**：成本等于投资方在被投资方可辨认净资产公允价值中的份额加上商誉。其中，被投资方可辨认净资产需在购买日按公允价值调整其资产和负债后计算，投资方按持股比例确认相应份额。
* **商誉的处理**：权益法下的商誉隐藏在“单行合并”的投资项目中，不单独作为无形资产列示。

**权益法下投资的后续计量**
* **资产负债表投资价值的调整**：投资价值在初始成本基础上，加上投资方应享有的被投资单位净收益份额，减去应享有的被投资单位股利分配份额。这实质上是按持股比例反映被投资单位留存收益（或净资产）的变动。
* **利润表中的体现**：在利润表中，投资方确认其应享有的被投资单位净收益份额。在CFA课程后续阅读中，为了调整杜邦比率，该权益法核算的投资收益被视为在税后列示，尽管实际会计准则对此位置可能没有明确规定，有时也可能在税前列示。

**重大影响的证据**
* **董事会代表权与政策参与**：在被投资单位董事会拥有代表权，参与其财务、经营和投资政策的制定，是重大影响的典型证据，但不拥有最终控制权。
* **重大交易与人员交流**：与被投资单位之间存在重大交易（如销售、技术共享）或关键人员互换，也表明存在重大影响，这可能会影响投资方的收益质量。

**权益法应用实例（基础版）**
* **案例设定**：P公司投资1000美元获得S公司30%股权，S公司在20X6年和20X7年有特定的净收益和股利数据。
* **投资收益与股利处理**：P公司在利润表中确认30%的S公司净收益（20X6年120美元，20X7年180美元），股利（20X6年30美元，20X7年45美元）不进入利润表，而是作为投资账面价值的减项。
* **投资账面价值计算**：20X6年末投资账面价值 = 初始投资1000 + 120 - 30 = 1090美元；20X7年末投资账面价值 = 1090 + 180 - 45 = 1225美元。
* **资产负债表平衡解释**：利润表中120美元收益使留存收益增加120美元，资产方投资增加90美元（120-30），现金因收到股利增加30美元，保持平衡。

**购买价格高于净资产账面价值的处理**
* **案例设定**：红公司以80,000美元购买蓝公司30%股权，蓝公司在购买日的净资产账面价值和公允价值存在差异（主要为PP&E公允价值高于账面价值50,000美元）。
* **商誉计算**：商誉 = 购买价格 - 投资方在被投资方可辨认净资产公允价值中所占份额。蓝公司可辨认净资产公允价值为250,000美元，红公司所占份额为75,000美元（250,000×30%），故商誉为5,000美元（80,000 - 75,000），此为部分商誉。
* **投资收益的调整**：蓝公司当年净收益100,000美元，红公司应享份额30,000美元。但需扣除PP&E公允价值调整部分的折旧影响：PP&E公允价值高于账面价值50,000美元，红公司份额15,000美元，按10年直线法折旧，每年折旧1,500美元，故调整后投资收益为28,500美元（30,000 - 1,500）。
* **年末投资账面价值计算**：初始投资80,000 + 调整后投资收益28,500 - 应享股利18,000（60,000×30%）= 90,500美元。

**关键影响与结论**
* **公允价值调整的后续影响**：购买日对被投资单位资产和负债的公允价值调整（如PP&E），其后续折旧或摊销会影响投资方各期的投资收益和投资账面价值。

讲解了具有重大影响但非控制的投资会计处理，即权益法，适用于能影响被投资单位的投资、融资和经营活动，母公司资产负债表中的投资将采用权益法核算。

[00:00:29](https://getnotes.seek:29) **权益法的别称与表现形式**

权益法常被称为“单行合并”，因为它在母公司的资产负债表和利润表中各体现为一行。

[00:00:46](https://getnotes.seek:46) **权益法的具体适用对象与所有权比例**

权益法适用于具有重大影响的联营企业，通常也适用于IFRS和US GAAP下的合营企业，一般以20%至50%的所有权比例为“明线标准”，但这只是指导原则而非硬性规定。

[00:01:00](https://getnotes.seek:60) **权益法投资的初始计量基础**

在资产负债表中，权益法投资最初按成本报告。

[00:01:11](https://getnotes.seek:71) **成本的构成要素**

成本包括购买价格，而购买价格实质上由两部分构成：投资方在被投资方可辨认净资产公允价值中的份额加上商誉。

[00:01:23](https://getnotes.seek:83) **被投资方可辨认净资产公允价值份额的计算**

需先将被投资单位资产负债表中的资产和负债调整至公允价值，然后计算投资方按持股比例享有的净 assets 份额。

[00:01:49](https://getnotes.seek:109) **公允价值调整的步骤**

在购买日，需对被投资单位资产负债表中的各项资产和负债进行公允价值调整，然后再计算其净 assets，进而确定投资方的份额。

[00:02:06](https://getnotes.seek:126) **持股比例与净 assets 份额的关系**

例如，若持有联营企业30%的股权，则带入30%的净 assets 份额，同时还包含商誉。

[00:02:21](https://getnotes.seek:141) **权益法下商誉的特点**

权益法核算的投资中存在商誉，但它隐藏在“单行合并”的投资项目中，不单独作为无形资产列示。

[00:02:37](https://getnotes.seek:157) **权益法投资后续计量的基本公式**

资产负债表中的投资价值按“成本 + 收益 - 股利”计量。初始按成本计量，年末则加上应享有的净收益份额，减去应享有的股利份额。

[00:02:56](https://getnotes.seek:176) **净收益减股利与留存收益的关系**

净收益减股利等于留存收益，因此投资账面价值的变动实质上反映了投资方按持股比例享有的被投资单位留存收益的变动，这与享有的净 assets 变动基本一致。

[00:03:11](https://getnotes.seek:191) **投资账面价值变动的本质**

每期投资账面价值的变动额等于投资方享有的被投资单位净 assets 增加额，即按持股比例乘以被投资单位的收益再减去按持股比例乘以其股利。

[00:03:26](https://getnotes.seek:206) **收益减股利与留存收益及净 assets 变动的一致性**

收益减股利等于留存收益变动，而留存收益变动大致等于净 assets 变动，因此投资账面价值的调整体现了这一逻辑。

[00:03:56](https://getnotes.seek:236) **年末投资账面价值的构成**

年末投资账面价值由三部分构成：购买日享有的被投资方可辨认净资产公允价值份额、享有的净 assets 变动份额以及商誉。核心是反映享有的被投资单位留存收益的变动。

[00:04:17](https://getnotes.seek:257) **投资账面价值与留存收益变动的关系**

若持有30%股权，则投资账面价值的变动为被投资单位留存收益变动的30%。

[00:04:41](https://getnotes.seek:281) **权益法在利润表中的体现**

利润表中体现为投资方享有的被投资单位净收益份额，即“一行”。

[00:04:57](https://getnotes.seek:297) **权益法投资收益在利润表中的位置**

会计准则对权益法投资收益在利润表中的位置没有大量指导，但在CFA课程后续阅读中，为调整杜邦比率，将其视为在税后列示，尽管有时也可能在税前列示。

[00:05:14](https://getnotes.seek:314) **CFA课程中对权益法投资收益位置的特殊处理**

CFA课程后续阅读要求从杜邦比率中剔除权益法联营企业的影响，此时需将权益法投资收益视为税后项目。

[00:05:38](https://getnotes.seek:338) **权益法投资收益在利润表位置的注意事项**

考试中可能看到权益法投资收益在税前或税后列示，但为调整杜邦比率，需按税后处理。

[00:05:57](https://getnotes.seek:357) **重大影响的证据类型**

判断对另一实体存在重大影响的证据包括：在董事会拥有代表权、参与政策制定（影响经营、投资、融资）、与被投资单位存在重大交易（如销售、技术共享）、关键人员互换等。

[00:06:24](https://getnotes.seek:384) **董事会代表权与政策参与的意义**

在被投资单位董事会拥有席位并参与政策制定，表明能影响其经营、投资和融资活动，但不拥有最终控制权。

[00:06:43](https://getnotes.seek:403) **重大交易与人员交流的影响**

与被投资单位的重大交易和关键人员互换表明双方存在关联关系，投资方具有影响力，这会影响收益质量。

[00:07:18](https://getnotes.seek:438) **实例中投资收益与股利的计算基础**

需按30%的持股比例计算享有的S公司收益和股利。

[00:07:36](https://getnotes.seek:456) **20X6年投资收益与股利的具体计算**

S公司20X6年收益的30%为120美元（计入利润表），股利的30%为30美元（影响资产负债表投资价值）。

股利不进入利润表，否则会与已确认的投资收益（享有的净收益份额）重复计算，因此仅作为投资账面价值的减项。

与20X6年相同，仅将享有的净收益份额计入利润表，股利作为投资账面价值的减项。

[00:08:28](https://getnotes.seek:508) **20X7年投资收益与股利的具体计算**

S公司20X7年收益的30%为180美元，股利的30%为45美元。

[00:08:44](https://getnotes.seek:524) **投资账面价值的初始构成**

初始投资成本（购买价格）包含购买日享有的被投资方可辨认净资产公允价值份额和商誉。

[00:09:07](https://getnotes.seek:547) **20X6年末投资账面价值的计算**

20X6年末投资账面价值 = 初始投资1000美元 + 享有的收益120美元 - 享有的股利30美元 = 1090美元，反映了享有的留存收益变动。

[00:09:35](https://getnotes.seek:575) **投资账面价值变动的逻辑**

投资账面价值随享有的被投资单位净收益增加，随享有的股利减少。

[00:09:52](https://getnotes.seek:592) **20X6年利润表与资产负债表的关系**

利润表中120美元投资收益使留存收益增加120美元，资产负债表中投资增加90美元（120-30），现金因收到股利增加30美元，保持平衡。

尽管投资账面价值增加90美元，留存收益增加120美元，但现金增加30美元（股利），资产方（投资+现金）共增加120美元，与留存收益增加额相等，报表平衡。

20X7年初投资账面价值为20X6年末的1090美元。

年末投资账面价值 = 购买日享有的公允价值净 assets 份额 + 享有的净 assets 变动份额 + 商誉。

开始讨论购买价格高于被投资单位净资产账面价值时，商誉的计算及相关调整。

[00:11:54](https://getnotes.seek:714) **案例设定：红公司购买蓝公司30%股权**

红公司支付80,000美元购买蓝公司30%股权，已知蓝公司在购买日的净资产账面价值和公允价值数据。

[00:12:13](https://getnotes.seek:733) **账面价值与公允价值净 assets 份额的计算**

蓝公司净资产账面价值的30%为60,000美元，公允价值净 assets 的30%为75,000美元，差异源于PP&E公允价值高于账面价值50,000美元。

[00:12:25](https://getnotes.seek:745) **PP&E公允价值与账面价值差异的原因**

US GAAP下PP&E按历史成本减累计折旧计量，可能与当前公允价值存在较大差异，本例中PP&E公允价值高于账面价值50,000美元。

[00:12:58](https://getnotes.seek:778) **PP&E公允价值调整的份额计算**

PP&E公允价值高于账面价值50,000美元，红公司享有的30%份额为15,000美元，这是导致公允价值净 assets 份额高于账面价值份额的原因。

[00:13:23](https://getnotes.seek:803) **权益法下对公允价值的关注**

收购联营企业时，需关注被投资单位净资产的公允价值而非账面价值。

[00:13:52](https://getnotes.seek:832) **案例补充信息与待解决问题**

蓝公司设备按10年直线法折旧，当年净收益100,000美元，支付股利60,000美元。需计算商誉、第一年权益投资收益及年末投资账面价值。

[00:14:14](https://getnotes.seek:854) **商誉的计算方法**

商誉 = 购买价格 - 投资方享有的被投资方可辨认净资产公允价值份额。本例中，购买价格80,000美元，享有的公允价值净 assets 份额75,000美元，故商誉5,000美元。

[00:14:43](https://getnotes.seek:883) **公允价值净 assets 份额的构成**

公允价值净 assets 份额 = 账面价值净 assets 份额 + 公允价值调整份额（如PP&E的15,000美元）。

[00:15:02](https://getnotes.seek:902) **权益法下的商誉类型**

权益法下计算的商誉是“部分商誉”，这与子公司合并时的商誉计算相关。

红公司享有的蓝公司净收益份额为100,000美元×30% = 30,000美元。

该额外折旧需从享有的净收益份额中扣除，以反映公允价值调整的影响。

[00:16:03](https://getnotes.seek:963) **额外折旧的计算**

15,000美元公允价值调整额按10年折旧，每年折旧1,500美元。

[00:16:26](https://getnotes.seek:986) **调整后的投资收益**

调整后投资收益 = 30,000美元 - 1,500美元 = 28,500美元，此金额将在利润表中列示。

[00:16:59](https://getnotes.seek:1019) **年末投资账面价值的构成与计算**

初始投资80,000美元（含75,000美元公允价值净 assets 份额和5,000美元商誉），加上调整后投资收益28,500美元，减去享有的股利18,000美元（60,000×30%），得到年末投资账面价值90,500美元。

[00:17:25](https://getnotes.seek:1045) **投资账面价值变动的具体项目**

投资账面价值 = 初始成本 + 权益法收益（调整后） - 股利。

[00:17:52](https://getnotes.seek:1072) **公允价值调整的后续影响总结**

购买日的公允价值调整（如PP&E）会通过后续折旧或摊销影响各期投资收益和投资账面价值。
