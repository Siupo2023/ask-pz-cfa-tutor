## 📚 核心框架总结

### 一、Acquisition Method（购买法）概述

**定义**：
非同一控制下企业合并编制合并报表的主流方法，核心原则基于**控制权**合并净资产。

**核心理念**：
> "虽然我只投资了 80%，但是我可以控制你所有的财务和经营决策，那就相当于我控制了你 100% 的资产跟负债。"

---

### 二、合并资产负债表的基本原则

#### 原则一：资产与负债的计量规则

**核心公式**：
```
合并资产负债表项目 = 母公司账面价值 + 子公司净资产 100% 公允价值
```

**关键要点**：
- ✅ 无论持股比例多少（如 80%），都并入子公司 **100%** 的资产和负债
- ✅ 因为母公司拥有**控制权**，可以控制所有财务和经营决策
- ✅ 子公司净资产必须按**公允价值**调整

#### 子公司公允价值调整

**三类常见公允价值增值**：
1. **存货（Inventory）**
2. **固定资产（Fixed Asset / PPE）**
3. **无形资产（Intangible Asset）**

**两类特殊调整项目**（⚠️ 易漏考点）：
- **或有负债（Contingent Liability）**
- **可辨认无形资产（Identifiable Intangible Asset）**
  - 许可证、商标、专利权
  - 内部研究开发形成的技术等

**特殊调整的处理**：
```
个别报表：账面价值 = 0（未确认）
合并报表：按公允价值确认
```

**条件**：只要公允价值可以可靠计量，就必须在合并报表中确认

---

#### 原则二：抵消母公司投资项目

**操作内容**：
- 母公司个别报表中的"对子公司投资（Investment）"项目**抵消**
- 展开为：
  1. **100%** 的子公司资产
  2. **100%** 的子公司负债
  3. **少数股东权益（Minority Interest）**

**理解方式**：
```
母公司报表：
  Investment in Subsidiary  ✓

合并报表：
  Asset of Subsidiary (100%)
  Liability of Subsidiary (100%)
  Minority Interest
```

---

#### 原则三：确认商誉（Goodwill）

**商誉的两种计算方法**：

##### 1. Full Goodwill（全额商誉法）

**适用准则**：
- ✅ US GAAP（美国准则）：**必须使用**
- ✅ IFRS（国际准则）：**允许使用**

**计算公式**：
```
Full Goodwill = (收购对价 ÷ 母公司持股比例) - 子公司可辨认净资产公允价值
```

**理解方式**：
- 假设**全额收购**子公司
- 计算需要支付的总对价
- 总对价超过净资产公允价值的部分 → Full Goodwill

**示例**：
```
假设：
- 收购 80% 股权，支付对价 800 万
- 子公司可辨认净资产公允价值 900 万

Full Goodwill = (800 ÷ 80%) - 900
              = 1000 - 900
              = 100 万
```

##### 2. Partial Goodwill（部分商誉法）

**适用准则**：
- ❌ US GAAP：**不允许使用**
- ✅ IFRS：**允许使用**

**计算公式**：
```
Partial Goodwill = 收购对价 - (子公司可辨认净资产公允价值 × 母公司持股比例)
```

**理解方式**：
- 真实的部分收购情况
- 实际支付对价超过收购净资产份额的部分 → Partial Goodwill

**示例**（使用相同数据）：
```
Partial Goodwill = 800 - (900 × 80%)
                  = 800 - 720
                  = 80 万
```

##### 廉价购买（Bargain Acquisition）

**定义**：
```
收购对价 < 子公司可辨认净资产公允价值
```

**会计处理**：
- ✅ 差额直接在**利润表确认收益**
- ❌ **不产生商誉**（商誉金额 = 0）

**记忆口诀**：
> "你买到就是赚到"

##### 商誉的后续计量

**核心规则**：
- ❌ 不摊销（Not Amortized）
- ✅ 至少每年进行一次**减值测试**（Impairment Test）

---

#### 原则四：确认少数股东权益

**少数股东权益的计算**：

##### Full Goodwill 下

**公式**：
```
少数股东权益 = (收购对价 ÷ 母公司持股比例) × 少数股东持股比例
```

**理解方式**：
- 假设全额收购，需要支付的总对价
- 其中少数股东持股比例的部分 = 向少数股东"借入"的资金
- 本质**偏向负债属性**

**示例**：
```
假设：
- 收购 80% 股权，支付对价 800 万
- 少数股东持股 20%

少数股东权益 = (800 ÷ 80%) × 20%
             = 1000 × 20%
             = 200 万
```

**理解**：相当于向少数股东借了 200 万完成全额收购

##### Partial Goodwill 下

**公式**：
```
少数股东权益 = 子公司可辨认净资产公允价值 × 少数股东持股比例
```

**理解方式**：
- 少数股东享有的子公司净资产份额
- 本质**偏向所有者权益属性**

**示例**：
```
假设：
- 子公司可辨认净资产公允价值 900 万
- 少数股东持股 20%

少数股东权益 = 900 × 20% = 180 万
```

**理解**：少数股东就是另一个"股东"，享有 20% 净资产份额

**为什么要调整少数股东权益？**
- 两种方法计算的商誉金额不同
- 会导致合并资产负债表左边资产总额不一致
- 需要通过调整少数股东权益保证**资产负债表平衡**

### 三、合并利润表的基本原则

#### 三步骤规则

**步骤 1：按公允价值调整子公司利润表**

调整项目：
- **存货增值** → 调整**营业成本（COGS）**
- **固定资产增值** → 调整**折旧费用（Depreciation）**
- **无形资产增值** → 调整**摊销费用（Amortization）**

**步骤 2：合并加总**

```
合并项目 = 母公司项目 + 子公司 100% 的利润表项目
```

**步骤 3：扣除少数股东利润**

```
合并净利润 = 母公司净利润 + 子公司净利润份额
```

**最终结果**：
- 合并净利润 = 母公司净利润 + （子公司净利润 × 母公司持股比例）
- 与权益法、比例合并法得到的结果**相同**

### 四、商誉减值的准则差异

#### IFRS（国际准则）规则

**依附实体**：现金产出单元（Cash Generating Unit）

**减值测试**：
```
对比：现金产出单元的账面价值 vs 可收回金额
```

**可收回金额**：
```
可收回金额 = Max（公允价值 - 处置费用，使用价值）
```

**减值损失分配**：
1. 总减值损失 = 账面价值 - 可收回金额
2. **优先分配给商誉**
3. 若商誉抵减至 0 后仍有剩余 → 按账面价值比例分配给其他非现金资产

**特点**：
- 一步法：减值测试和计算用同一公式
- 减值损失可能超过商誉账面价值

#### US GAAP（美国准则）规则

**依附实体**：报告单元（Reporting Unit）

**减值测试（第一步）**：
```
对比：报告单元的账面价值 vs 公允价值
```

**减值损失计算（第二步）**：
```
商誉减值损失 = 商誉账面价值 - 商誉隐含公允价值
```

**商誉隐含公允价值**：
```
商誉隐含公允价值 = 报告单元公允价值 - 子公司可辨认净资产公允价值
```

**理解方式**：
- 假设**重新收购一遍**子公司
- 计算当前应该产生的商誉金额

**特点**：
- 两步法：减值测试和计算用不同公式
- 商誉最多抵减至 0
- 不涉及分配给其他资产

#### 准则差异总结

| 对比维度 | IFRS（一步法） | US GAAP（两步法） |
|----------|----------------|-------------------|
| 减值测试对象 | 账面价值 vs 可收回金额 | 账面价值 vs 公允价值 |
| 减值损失范围 | 整个资产组的损失 | 仅商誉本身的损失 |
| 是否需要分配 | ✅ 需要分配 | ❌ 不需要分配 |
| 最大减值损失 | 可能超过商誉账面价值 | 最多等于商誉账面价值 |

### 五、合并方法对比

**三种合并方法**：
1. **Acquisition Method（购买法）**
2. **Equity Method（权益法）**
3. **Proportionate Consolidation（比例合并法）**

**对比要点**：
- 不同方法对净利润的影响
- 不同方法对资产负债表的影响
- 不同方法的适用条件

## 🎯 考试重点总结

### 🚩 高频考点

#### 1. 合并资产负债表计量规则
- **考点**：无论持股比例多少，都并入子公司 100% 的资产负债
- **类型**：选择题高频考点
- **关键**：基于控制权的概念

#### 2. 合并报表特殊调整项目
- **考点**：或有负债、未确认可辨认无形资产
- **类型**：易漏考点
- **关键**：个别报表账面价值为 0，合并报表按公允价值确认

#### 3. 商誉计量的准则差异
- **考点**：Full Goodwill vs Partial Goodwill
- **类型**：必考计算考点
- **关键**：
  - US GAAP：只能用 Full Goodwill
  - IFRS：两者都允许

#### 4. 商誉后续计量
- **考点**：不摊销、每年减值测试
- **类型**：概念题考点
- **关键**：Not Amortized，但需 Impairment Test

#### 5. 商誉减值的准则差异
- **考点**：IFRS vs US GAAP 的减值计算规则
- **类型**：核心计算考点
- **关键**：
  - IFRS：一步法，需分配
  - US GAAP：两步法，不分配

#### 6. 少数股东权益的计算
- **考点**：Full vs Partial Goodwill 下的不同计算
- **类型**：计算考点
- **关键**：为了资产负债表调平

#### 7. 合并利润表的编制步骤
- **考点**：三步骤规则
- **类型**：流程理解考点
- **关键**：公允价值调整 → 合并加总 → 扣除少数股东利润

### 💡 核心概念

- **Acquisition Method（购买法）**：非同一控制下企业合并的主流方法
- **Consolidated Balance Sheet（合并资产负债表）**：反映合并整体财务状况的报表
- **Full Goodwill（全额商誉）**：假设全额收购，计算 100% 商誉
- **Partial Goodwill（部分商誉）**：仅按收购份额确认商誉
- **Bargain Acquisition（廉价购买）**：收购对价小于净资产公允价值
- **Minority Interest（少数股东权益）**：非控制性股东享有的净资产权益
- **Cash Generating Unit（现金产出单元）**：IFRS 术语，依附实体
- **Reporting Unit（报告单元）**：US GAAP 术语，依附实体

### 📝 记忆技巧

1. **合并资产负债表四句话**：
   - 资产负债：母公司账面价值 + 子公司 100% 公允价值
   - 抵消投资：展开为 100% 资产、100% 负债、少数股东权益
   - 确认商誉
   - 确认少数股东权益

2. **合并利润表三步骤**：
   - 调整公允价值影响
   - 合并加总
   - 扣除少数股东利润

3. **商誉减值**：
   - IFRS：一步法，先减商誉，再减其他
   - US GAAP：两步法，只减商誉
