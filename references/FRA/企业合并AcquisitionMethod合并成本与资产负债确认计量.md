# 企业合并 Acquisition Method：合并成本与资产负债确认计量

- **标签**：企业合并, Acquisition Method, 合并成本, 资产负债确认计量

## 笔记内容

### 📑 核心概念总结

#### 1. 企业合并类型

**法定合并**：
- A公司与B公司合并后仅A公司存续
- B公司净资产转移至A公司后停止存在

**收购法**：
- 两家公司形成母子公司关系
- 双方均继续存在
- **CFA 考试重点**

**新设合并**：
- A、B公司合并成立新实体C
- 原A、B公司均停止存在

**可变利益实体（VIE）**：
- 通过可变利益而非股权控制的实体
- 常称为特殊目的载体
- US GAAP 有专门规定

---

### 💰 合并成本

#### 2.1 合并成本构成

**基本公式**：
```
合并成本 = 对价支付 + 直接费用
```

**对价支付类型**：
- 现金支付
- 股票交换（按公允价值计量）
- 混合对价

**直接费用**：
- 中介费（投行、律师、会计师）
- 尽职调查费用
- 注册登记费用

#### 2.2 合并成本调整

**收购价格调整**：
- 收购子公司时，需将资产负债从账面价值调整至公允价值
- **不包含**子公司账面上的商誉

**商誉计算**：
```
商誉 = 收购价格 - 可辨认净资产公允价值
```

**示例**：
- Wood 公司以 6 亿美元收购 Pine 公司 100% 股权
- Pine 公司可辨认净资产账面价值：4.4 亿美元
- PP&E 公允价值高于账面价值：1.2 亿美元
- 调整后可辨认净资产公允价值：5.6 亿美元
- **商誉 = 6亿 - 5.6亿 = 0.4 亿美元**

---

### 📊 资产负债确认与计量

#### 3.1 可辨认资产和负债的调整

**核心原则**：
1. **必须调整至公允价值**
2. **不包含**子公司自身商誉
3. 调整具有主观性

**调整动机**：
收购方有动机**低估**可辨认净资产公允价值，以**最大化商誉**：
- 商誉除非减值，否则不影响未来损益表
- 高估固定资产等会因折旧增加而减少未来利润

#### 3.2 需特别考虑的项目

**1. 内部产生的无形资产**
- 如品牌名称
- 需估计价值并确认为无形资产

**2. 或有负债**
- **IFRS**：可靠计量时确认
- **US GAAP**：很可能发生时按公允价值确认

**3. 赔偿资产**
- 若卖方就或有负债提供赔偿
- 需在收购日确认资产

**4. 金融资产和负债**
- 可按收购方会计政策重分类
- 如从交易性证券改为持有至到期投资

#### 3.3 企业合并中的特殊项目处理

**研发支出（R&D）**：
- IFRS 和 US GAAP **均要求资本化**
- 作为无形资产，后续摊销或减值
- 与平时研发费用化处理不同

**重组成本**：
- **不能**作为收购成本的一部分
- 需在未来发生时费用化
- 不影响收购日可辨认净资产公允价值

**或有对价**：
- 依赖未来事件（如子公司业绩）的收购价格
- 需在收购日按估计公允价值确认：
  - 资产（可能减少支付）
  - 负债（可能增加支付）
  - 权益（股份交换）
- 负债的公允价值变动计入损益表
- 权益类变动不影响损益表
- **US GAAP**：允许确认或有资产
- **IFRS**：不允许确认或有资产

---

### 📐 部分商誉 vs 完全商誉

#### 4.1 适用场景

- 收购 **100% 股权**：两种方法无差异
- 收购 **少数股权**：才产生区别

#### 4.2 会计准则要求

- **US GAAP**：必须使用**完全商誉法**
- **IFRS**：可选择完全商誉法或部分商誉法

#### 4.3 计算方法

**部分商誉计算**：
```
部分商誉 = 收购价格 - (可辨认净资产公允价值 × 收购股权比例)
```

**示例**：
- 以 4.5 亿美元收购 75% 股权
- 可辨认净资产公允价值：5.6 亿美元
- 收购方份额：5.6亿 × 75% = 4.2 亿美元
- **商誉 = 4.5亿 - 4.2亿 = 0.3 亿美元**

**完全商誉计算**：
```
完全商誉 = 子公司整体公允价值 - 可辨认净资产公允价值
```

**示例**：
- 子公司整体公允价值：4.5亿 / 75% = 6 亿美元
- 可辨认净资产公允价值：5.6 亿美元
- **商誉 = 6亿 - 5.6亿 = 0.4 亿美元**

**转换关系**：
```
完全商誉 = 部分商誉 / 收购股权比例
部分商誉 = 完全商誉 × 收购股权比例
```

#### 4.4 少数股东权益的计算

**部分商誉法**：
```
少数股东权益 = 可辨认净资产公允价值 × 少数股东持股比例
= 5.6亿 × 25% = 1.4 亿美元
```

**完全商誉法**：
```
少数股东权益 = 子公司整体公允价值 × 少数股东持股比例
= 6亿 × 25% = 1.5 亿美元
```

### 📉 不同会计处理方法比较

#### 5.1 权益法 vs 比例合并法 vs 收购法

| 项目 | 权益法 | 比例合并法 | 收购法 |
|------|--------|-----------|--------|
| **销售** | 仅母公司 | 母公司+合资企业份额 | 母公司+子公司**全部** |
| **费用** | 仅母公司 | 母公司+合资企业份额 | 母公司+子公司**全部** |
| **净收入** | 相同 | 相同 | 相同（需扣除非控制性权益） |
| **资产** | 最低（单线合并） | 中等 | **最高**（包含全部） |
| **负债** | 最低（单线合并） | 中等 | **最高**（包含全部） |
| **股东权益** | 较低 | 相同 | **更高**（含少数股东权益） |

#### 5.2 财务比率影响

**净利润率**：
- 权益法：**最高**（销售基数小）
- 收购法：较低（销售基数大）

**资产回报率（ROA）**：
- 权益法：**最高**（资产基数小）
- 收购法：较低（资产基数大）

**股东权益回报率（ROE）**：
- 权益法：**较高**（权益基数小）
- 收购法：**最低**（权益基数大）

**分析师调整**：
通常剔除少数股东权益后计算权益类比率，因少数权益更类似债务

---

### ⚠️ 财务报告质量问题

#### 6.1 企业合并中的操纵手段

**低估可辨认资产，高估商誉**：
- 动机：商誉不影响未来损益表（除非减值）
- 影响：减少未来折旧对收益的影响

**收购使过往期间比较困难**：
- 可掩盖欺诈或业绩下滑
- 财务比率基数变化

#### 6.2 警示信号

- 企业合并后盈利能力突然提升
- 商誉占收购价格比例异常高
- 收购后折旧费用反而下降
- 少数股东权益波动异常

---

### 🎯 关键要点总结

1. **合并成本 = 对价支付 + 直接费用**
2. **资产负债必须调整至公允价值**
3. **商誉 = 收购价格 - 可辨认净资产公允价值**
4. **研发支出必须资本化**（与平时不同）
5. **重组成本不能计入收购成本**
6. **US GAAP 要求完全商誉法，IFRS 可选**
7. **收购法产生最保守的财务比率**
