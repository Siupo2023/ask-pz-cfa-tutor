#### 录音总结

本次课程详细讲解了k近邻（k-NN）算法的基本原理、关键要点、优缺点及应用场景，通过实例帮助理解其在分类和回归问题中的应用。

**k近邻算法基本原理**

-   **核心思想**：基于“物以类聚，人以群分”的原则，通过待分类对象的最近邻样本类别来判断其类别。该算法属于“懒惰学习”（lazy learning），无需训练过程，直接利用已知样本数据进行分类或回归预测。
    
-   **分类应用示例**：以公司违约分类（叉叉代表坏公司，三角代表好公司）和电影类型分类（爱情片与动作片，特征为接吻镜头和打斗镜头数量）为例，说明通过计算待分类对象与样本集中各对象的距离，选取k个最近邻，按“少数服从多数”原则确定类别。
    
-   **回归应用示例**：以房屋租金预测为例，选取k个最相似房屋（特征为位置和面积），计算其租金均值作为预测结果。
    

**k近邻算法关键要点**

-   **k值选择**：k值是超参数（hyperparameter），需人为指定。通常选择奇数以避免分类平局；k值不宜过小（易受异常值影响，错误率高）或过大（稀释近邻概念，降低分类精度），实际应用中多选择相对较小的k值，并可通过交叉验证提升模型精度。
    
-   **距离（相似度）计算**：常用欧几里得距离（直线距离），也可使用相关关系（correlation）衡量相似度。特征数量对距离计算准确性影响较大，k-NN适用于特征数量较少（small features）的数据集，特征过多时需进行特征选择（feature selection）以去除无关特征。
    

**k近邻算法优缺点**

-   **优点**：模型直观（intuitive）、非参数（nonparametric，无需估计参数）、对数据分布无假设、支持多分类（multi class classification）。
    
-   **缺点**：属于懒惰学习，分类时需计算待分类对象与所有样本的距离，当样本量（training set）较大时计算量巨大；对特征数量敏感，特征过多时距离计算不准确。
    

**k近邻算法应用场景**

-   **分类问题**：公司破产估计（判断是否违约）、公司债券评级（如AAA级、AA级）、股票指数构建（将相似股票归类）。
    
-   **回归问题**：股价预测（预测具体数值）。
